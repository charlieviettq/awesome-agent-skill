# Example: SolarVenture 台灣離岸風電投資延遲決策

## Scenario

SolarVenture 是一家台灣中型再生能源開發商，正在評估是否立即進入台灣離岸風電市場。專案需要在 2026 年底前決定：

- **立即投資**：簽署 25 年場址租約，投入 NT$4.2B 開發成本，取得 200MW 容量
- **等待 2 年**：觀察政府 2027 年躉購費率（FIT）公告後再決策

財務長提出疑問：「DCF 顯示 NPV 為 -NT$180M，但策略團隊說這個案子不能放棄。到底哪裡出了問題？」

**已知資訊：**
- 目前專案 PV（現金流現值）：NT$4,020M
- 開發投資成本：NT$4,200M
- 靜態 NPV：-NT$180M
- 市場觀察期：2 年（等待 FIT 公告）
- 無風險利率：1.5%（10 年期公債）
- 估計波動率：28%（參考台灣離岸風電已上市同業 + 電力市場情境分析）

---

## Analysis

### Step 1 — 識別嵌入選擇權

| 選擇權類型 | 描述 | 觸發條件 |
|-----------|------|---------|
| **Defer（延遲）** | 等待 FIT 費率確認後再決定是否進場 | 2027 年能源署公告 |
| Abandon | 若 FIT 不佳，放棄投資回收場址押金部分 | FIT < 盈虧平衡費率 |

核心選擇權為 **Option to Defer**：2 年後，若市場條件有利，SolarVenture 可選擇行使投資；若不利，讓選擇權到期失效，損失僅限於場址預約保證金 NT$50M。

### Step 2 — 對應選擇權參數

| 參數 | 財務選擇權對應 | 本案數值 | 說明 |
|------|------------|---------|------|
| Underlying (S) | 股票現價 | NT$4,020M | 專案現金流的現值 |
| Exercise price (K) | 履約價格 | NT$4,200M | 完整開發投資成本 |
| Time (T) | 到期日 | 2 年 | 等待 FIT 公告的決策窗口 |
| Volatility (σ) | 標的資產波動率 | 28% | 離岸風電同業 + 電力市場情境 |
| Risk-free rate (r) | 無風險利率 | 1.5% | 10 年期公債殖利率 |

**靜態 NPV：**
```
Static NPV = S - K = 4,020 - 4,200 = -NT$180M
```

### Step 3 — 延遲選擇權估值（Black-Scholes 類比）

計算 d₁、d₂：

```
d₁ = [ln(S/K) + (r + σ²/2)·T] / (σ·√T)
   = [ln(4020/4200) + (0.015 + 0.0392)·2] / (0.28·√2)
   = [-0.0444 + 0.1084] / 0.3960
   = 0.064 / 0.3960
   ≈ 0.162

d₂ = d₁ - σ·√T
   = 0.162 - 0.396
   = -0.234
```

查標準常態累積分配：
```
N(d₁) = N(0.162) ≈ 0.564
N(d₂) = N(-0.234) ≈ 0.407
```

Black-Scholes Call 選擇權價值：
```
C = S·N(d₁) - K·e^(-rT)·N(d₂)
  = 4020 × 0.564 - 4200 × e^(-0.03) × 0.407
  = 2267.3 - 4200 × 0.9704 × 0.407
  = 2267.3 - 1659.5
  ≈ NT$608M
```

### Step 4 — 計算 Expanded NPV

```
Expanded NPV = Static NPV + Option Value
             = -180 + 608
             = +NT$428M
```

## Result

## Real Options Analysis: SolarVenture 離岸風電投資

### Static NPV
- NPV = **-NT$180M**（傳統 DCF，立即投資，拒絕）

### Embedded Options Identified
| 選擇權 | 類型 | 價值驅動因素 |
|--------|------|-------------|
| 等待 FIT 公告 | Defer | 2027 年費率不確定性（σ=28%） |
| 放棄預約場址 | Abandon | FIT 公告後市場不佳的保護 |

### Option Valuation
| 參數 | 數值 |
|------|------|
| Underlying (S) | NT$4,020M |
| Exercise price (K) | NT$4,200M |
| Volatility (σ) | 28% |
| Time (T) | 2 年 |
| d₁ / d₂ | 0.162 / -0.234 |
| **延遲選擇權價值** | **NT$608M** |

### Expanded NPV
- Static NPV + Option Value = **-180 + 608 = +NT$428M**
- **決策：保留選擇權（Preserve Option）**

### 管理意涵

1. **不要現在簽長約**：立即承諾 NT$4,200M 消滅了 NT$608M 的選擇權價值
2. **支付保留成本**：場址預約保證金 NT$50M 是購買 2 年資訊等待的合理代價
3. **監控觸發條件**：若 FIT > NT$5.8/kWh（盈虧平衡），2027 年應立即行使選擇權
4. **注意選擇權交互作用**：一旦行使延遲選擇權進場，放棄選擇權（Abandon）的價值會隨建設推進而遞減

> **財務長問題的答案**：DCF 顯示 -NT$180M 是正確的——針對「立即且不可逆投資」。但這個數字忽略了 NT$608M 的管理彈性價值。Expanded NPV 為 +NT$428M，策略團隊的直覺正確，但需要選擇權框架來量化它。
