# IPO 治理升級路徑與常見缺陷筆記

## 為什麼 IPO 需要治理升級

### 從私人公司到公眾公司
- **股東**：從少數熟識 → 眾多不認識
- **資訊**：從封閉 → 公開揭露
- **決策**：從老闆 → 董事會 + 主管機關 + 市場
- **問責**：從家族自評 → 法律、市場雙重
- **時間觀**：從長期 → 季度壓力

### IPO 治理升級的雙重目標
1. **合規**：符合上市 / 上櫃審查標準
2. **市場可信**：投資人願意買單、長期經營

## 典型時間軸（以上櫃為例）

### T-36 至 T-24：啟動期

**治理盤點**
- 董事會能力矩陣
- 三道防線現況
- 關係人交易盤點
- 內控制度評估

**關鍵人選**
- 承銷商（主辦、協辦）
- 簽證會計師（四大事務所常見）
- 外部律師事務所
- IPO 專案顧問

**高層共識建立**
- 家族 / 大股東共識：股權稀釋、治理放權、透明揭露
- 高管準備度：面對董事會、分析師、媒體

### T-24 至 T-18：架構期

**董事會重組**
- 獨立董事預聘 3 位（專長互補）
- 能力矩陣補位
- 章程修訂準備

**委員會設置**
- 審計委員會
- 薪酬委員會
- 提名委員會（建議）
- 永續委員會（建議）

**三道防線部署**
- 第二道：法遵長、風管長
- 第三道：內部稽核室
- 章程明定獨立升級權

### T-18 至 T-12：制度化

**內部控制**
- 九大循環文件化
- 關鍵控制點定義
- 控制自評制度

**財務制度**
- IFRS 公報轉換
- 財報編製流程
- 會計政策一致性

**關係人交易處理**
- 全面盤點
- 重訂定價 / 終止 / 重組
- 揭露制度

### T-12 至 T-6：運作期

**運作試行**
- 董事會、委員會開始正式運作
- 內控制度實施
- 三道防線實運作 12 個月以上（審查要求）

**揭露準備**
- 年報模擬編製
- 公司治理報告模擬
- 永續報告書（如適用）

### T-6 至 T-0：申請與路演

**主管機關審查**
- 承銷商輔導報告
- 會計師簽證
- 律師意見書
- 金管會 / 交易所審查

**市場準備**
- 公開說明書
- 路演（Roadshow）
- 分析師會議
- 媒體溝通

**IPO 日**
- 承銷價格決定
- 掛牌儀式
- 首日交易

### T+0 至 T+12：穩定期

**首年關鍵**
- 第一份公開季報 / 半年報 / 年報
- 第一次法說會
- 公司治理評鑑參評
- 股價穩定與預期管理

## 常見 IPO 治理缺陷與處理

### 缺陷 1：獨立董事準備不足

**症狀**：
- 獨董為創辦人朋友圈
- 獨董專業 gap 明顯
- 獨董未投入實質時間

**處理**：
- 以獵頭公司獨立招募
- 能力矩陣公開揭示 gap、招募補位
- 預聘 18 個月以上，讓獨董熟悉公司

### 缺陷 2：關係人交易過度集中

**症狀**：
- 關係人交易占營收 30% 以上
- 大股東個人公司為獨家供應商
- 家族成員租賃給公司

**處理**：
- 終止（小額）
- 重組（併入集團）
- 公平交易定價 + 公開揭露
- **原則**：IPO 後關係人交易應 ≤ 20%、關鍵原料分散至 2–3 家以上

### 缺陷 3：財務報表品質不佳

**症狀**：
- IFRS 轉換倉促
- 關鍵會計估計主觀性高
- 會計師事務所小型

**處理**：
- IFRS 轉換至少 T-18 完成
- 四大會計師事務所簽證（市場偏好）
- 重大會計估計建立審議機制

### 缺陷 4：三道防線不完整

**症狀**：
- 無獨立法遵 / 風管
- 稽核人力不足
- 無獨立升級機制

**處理**：
- T-24 前人選到位
- 章程獨立升級條款
- 稽核主管向審計委員會報告

### 缺陷 5：家族治理與公司治理混淆

**症狀**：
- 家族議題直入董事會
- 家族成員席次過多
- 家族憲法未建立

**處理**：
- 家族委員會先議
- 家族董事席次限制
- 家族憲法與公司章程對齊

### 缺陷 6：ESG 揭露不到位

**症狀**：
- 無永續報告
- TCFD 揭露空白
- 董事會無永續討論

**處理**：
- T-12 起啟動永續報告
- 永續委員會設置
- 董事會永續素養訓練

### 缺陷 7：股權結構問題

**症狀**：
- 家族持股過高（> 70%）
- 流通股數不足
- 員工持股偏低

**處理**：
- 家族階段性減持
- 員工持股信託
- ESOP（員工認股權）
- 配發計畫公開透明

### 缺陷 8：高管薪酬與績效脫鉤

**症狀**：
- 高管薪酬固定為主
- 無長期激勵（股票選擇權、RSU）
- 無追討條款

**處理**：
- 薪酬結構重設（固定 + 短期變動 + 長期激勵）
- 設立 ESOP / RSU
- 追討條款（Clawback）章程化

### 缺陷 9：揭露與溝通能力不足

**症狀**：
- IR 部門弱 / 無
- 高管不擅面對分析師
- 媒體回應風險

**處理**：
- IR 部門建立
- 高管路演培訓
- 媒體危機應變 SOP

### 缺陷 10：上市後的「第一年陷阱」

**症狀**：
- 上市當年業績下滑
- 與預期落差大
- 股價崩跌

**處理**：
- 上市前「保守預測」
- 上市後「穩定交付」
- 避免為上市衝刺非永續業績

## 承銷商與會計師選擇

### 承銷商選擇
- **市場能量**：能否撐起承銷規模
- **產業熟悉度**：相關產業 IPO 經驗
- **法人關係**：外資、投信、壽險連結
- **服務品質**：專案團隊資深度

### 會計師選擇
- **四大偏好**：KPMG、EY、PwC、Deloitte
- **產業團隊**：具備該產業深度
- **與公司契合**：溝通順暢、誠信
- **獨立性**：無與公司既有利益糾葛

### 常見錯誤
- 選擇「最便宜」承銷商 / 會計師 → IPO 品質打折
- 與中小型事務所 / 券商合作、IPO 後被質疑
- 過度依賴 / 信任單一服務商

## 路演（Roadshow）要點

### 目標對象
- 主要法人（外資、投信、壽險）
- 家族辦公室、私募基金
- 分析師
- 媒體

### 路演材料
- 公開說明書核心頁
- 投資故事（Equity Story）
- 成長預測
- 管理團隊介紹

### 常見問題
**投資人典型問題**：
- 成長動能來源？
- 毛利率永續性？
- 大客戶集中度？
- 競爭對手比較？
- 董事會獨立性？
- 關係人交易？
- 家族接班計畫？
- ESG 承諾？

**高管準備**：
- 練習 IR 語言（避免過度技術）
- 模擬問答 50+ 題
- 法律保留（Safe Harbor）

## 家族企業 IPO 的特殊準備

### 股權規劃
- 家族持股稀釋到多少？通常 50–60% 為合理
- 控制權安排（閉鎖性條款、特別股）
- 稅務規劃

### 家族治理與上市治理的對齊
- 家族憲法
- 家族委員會
- 家族辦公室
- 與董事會的介面

### 接班準備
- CEO 繼任規劃
- 家族二代職能證明
- 專業經理人的進入

### 家族文化的保留與變革
- 哪些文化要保留？（核心價值）
- 哪些需變革？（不合 IPO 需求）
- 如何向員工溝通？

## 海外 IPO 的額外考量

### 美國 IPO
- SEC 註冊：Form S-1 / F-1
- SOX 404 內控認證
- 季報 10-Q / 10-K
- 獨董過半、關鍵委員會全獨董

### 香港 IPO
- 聯交所上市規則
- 更強調公司治理守則
- 兩套準則（主板 / GEM）

### 選擇性考量
- 業務與市場連結
- 法規熟悉度
- 後續融資效率
- 治理成本

## 常見 IPO 失敗案例類型（匿名）

### 類型 1：治理脆弱被市場發現
上市後 1–2 年爆發治理爭議，股價重挫。

### 類型 2：業績泡沫
上市前衝刺業績、上市後回歸常態、落差引發質疑。

### 類型 3：關係人交易未處理乾淨
上市後被媒體揭露，股價下跌。

### 類型 4：承諾與實際落差
上市時承諾的成長未兌現。

### 類型 5：家族內訌外顯化
上市後公開化的家族衝突影響公司運作。

## EMBA 作業實務建議

### 個案分析重點
1. **治理升級時間軸**：是否 24 個月以上？
2. **獨董品質**：獨立性實質 / 能力 gap 補位
3. **關係人交易處理**：方法、進度、揭露
4. **三道防線**：部署完整度
5. **家族治理與公司治理對齊**
6. **上市後 3 年追蹤**（可選）：實際治理品質 vs. 上市前承諾
