# Taiwan Media Monitoring 台灣媒體監測指南

## 核心差異：為什麼台灣需要單獨處理

國際監測工具（Meltwater、Brandwatch、Talkwalker）的爬蟲覆蓋率在台灣有明顯盲區：

| 平台 | 國際工具覆蓋率 | 說明 |
|------|-------------|------|
| PTT | 差 | 部分工具抓得到，但往往延遲 24-48 小時且遺漏率高 |
| Dcard | 差～無 | 絕大多數國際工具不涵蓋 |
| LINE 群組 | 無 | 封閉生態，無法爬取 |
| Yahoo奇摩新聞 | 普通 | 需確認工具有設定繁體中文關鍵字 |
| 巴哈姆特 | 無～差 | 遊戲、ACG 品牌必須手動處理 |
| Mobile01 | 無～差 | 3C、汽車、生活品牌高度相關 |

**實務結論**：台灣監測必須「國際工具 + 本地工具 + 手動抽查」三層並用。

---

## 第一層：平台地圖

### 論壇類（輿論發酵地）

**PTT (ptt.cc)**
- 最具影響力的匿名論壇，年齡層 25-45 為主
- 關鍵看板：`Gossiping`（八卦板）、`Stock`、`Beauty`、`Tech_Job`、`PC_Shopping`
- 速度特性：危機事件在八卦板可在 2 小時內累積 200+ 推文，發酵速度快於主流媒體
- 監測重點：標題（推文轉發才是病毒擴散關鍵，不只是文章數量）

**Dcard (dcard.tw)**
- 主要用戶：18-25 歲大學生，實名制但匿名貼文
- 關鍵板塊：`時事`、`情感`、品牌官方板（如果有開設）、`工作板`
- 速度特性：比 PTT 慢，但 Z 世代品牌感知指標更準
- 注意：Dcard 對消費性品牌的負評擴散具有 IG 連動效應

**巴哈姆特 (gamer.com.tw)**
- 限定領域：遊戲、動漫、3C 周邊
- 非此領域品牌可略過

**Mobile01 (mobile01.com)**
- 3C、汽車、攝影、房地產品牌必監測
- 文章權重高（Google 搜尋結果常出現 Mobile01）

### 社群媒體類

| 平台 | 台灣滲透率 | 監測重點 |
|------|----------|---------|
| Facebook | ~75% 成人用戶 | 粉絲頁留言、社團（封閉社團為盲區）|
| Instagram | ~50%，18-35 為主 | hashtag、限時動態（無法監測）、貼文留言 |
| YouTube | ~85% | 影片留言、相關頻道提及 |
| TikTok (抖音台灣版) | 快速成長，18-30 | hashtag 趨勢、創作者提及 |
| X/Twitter | ~5-8%，科技/媒體人為主 | 意見領袖聲量，不代表大眾 |
| LINE | ~90%，但封閉 | 只能監測 LINE TODAY 新聞、LINE 官方帳號數據 |

### 新聞媒體類

**主流新聞網站（國際工具通常可抓）**
- 三立新聞、ETtoday、中時、聯合新聞網、自由時報、蘋果日報（已停刊，但存檔有影響力）
- TVBS、民視、東森

**原生數位媒體（部分國際工具有漏洞）**
- The Reporter（報導者）— 深度調查，被引用率高
- 關鍵評論網 (thenewslens.com) — 意見領袖受眾
- 換日線、天下、商業周刊 — B2B/品牌形象相關

## 第二層：工具矩陣

### 推薦工具組合（依預算）

**預算 A：< NT$5,000/月（新創、小品牌）**

```
Google Alerts (免費)
  → 設定品牌名、競品名、危機關鍵字
  → 缺點：台灣論壇覆蓋幾乎為零

PTT 手動監測 (免費)
  → 每日搜尋：https://www.ptt.cc/bbs/search?q={品牌名}
  → 重點：八卦板、Stock 板

Dcard 手動監測 (免費)
  → https://www.dcard.tw/search?query={品牌名}

Facebook Search (免費)
  → 在 FB 搜尋欄搜尋品牌名，篩選「最新」

總工時估計：約 30-45 分/天
```

**預算 B：NT$5,000-30,000/月（中型品牌）**

```
OpView 社群口碑資料庫 (台灣本土工具)
  → 涵蓋 PTT、Dcard、Mobile01、Yahoo 知識+
  → 最適合台灣市場的中價位選擇

搭配 Google Alerts 補新聞
搭配人工 PTT/Dcard 抽查（每週 2-3 次）
```

**預算 C：NT$30,000+/月（大型品牌、上市公司）**

```
Meltwater 或 Talkwalker (國際新聞覆蓋)
  搭配
OpView 或 QSearch (台灣社群論壇)
  搭配
人工監測小組或代理商（LINE 生態系、直播平台）
```

### OpView 關鍵設定

OpView 是台灣覆蓋率最廣的本土社群監測工具，設定時注意：

1. **繁體/簡體並列設定**：若品牌有簡體中文用戶，兩種字形都要加入
2. **同音異字**：品牌名如有常見打錯版本需一併列入（例：「星巴克」/「星把客」）
3. **負向關鍵字排除**：避免抓到不相關的同名詞（例：監測「蘋果」品牌須排除「蘋果派食譜」類內容）

## 第三層：PTT 監測 SOP

PTT 是台灣輿論最重要的發酵地，以下為完整監測程序。

### 每日快速掃描（15 分鐘）

```
1. 前往 https://www.ptt.cc/bbs/Gossiping/search?q={品牌關鍵字}
   → 按「最新」排序
   → 檢查過去 24 小時內有無新文章

2. 前往 https://www.ptt.cc/bbs/Stock/search?q={品牌關鍵字}
   （若為上市公司，此板必看）

3. 查看各板「熱門文章」前 20 名標題，目測有無品牌被帶入

判斷基準：
- 推文數 < 20：低風險，記錄即可
- 推文數 20-100：中風險，閱讀推文判斷情緒
- 推文數 > 100 或進入熱門：高風險，立即回報 PR 主管
```

### 推文情緒快速編碼

PTT 推文有內建的「推/噓/→」機制，可快速量化：

```
情緒指數 = (推文數 × 1) + (箭頭數 × 0) + (噓文數 × -1)
正規化後：情緒分數 = 情緒指數 / 總推文數

解讀：
 > +0.5 : 正面（社群支持）
 +0.2 ~ +0.5 : 偏正面
 -0.2 ~ +0.2 : 中性或分歧
 -0.2 ~ -0.5 : 偏負面
 < -0.5 : 強烈負面（危機警示）
```

**範例計算：**
某篇文章 推:45 →:12 噓:23
- 情緒指數 = 45 - 23 = +22
- 情緒分數 = 22 / 80 = +0.275（偏正面）

### 病毒擴散判斷

PTT 文章轉往主流媒體的判斷指標（非絕對，歷史觀察）：

| 條件 | 轉媒體機率 |
|------|----------|
| 推文數 > 500 且出現 24 小時內 | 高（約 60-70%） |
| 負面情緒分數 < -0.3 且推文數 > 200 | 高 |
| 被八卦板標記為「精華」或「爆」 | 極高 |
| 推文數 > 100 但情緒中性 | 低 |

## 第四層：台灣輿論情境對照表

台灣媒體生態有幾個本地特性，影響監測解讀：

### 媒體政治傾向

台灣主流媒體有明顯政治立場，**同一則新聞在不同媒體的標題和切入角度會不同**，這影響情緒分析準確度：

| 媒體 | 一般傾向 | 監測注意事項 |
|------|---------|------------|
| 三立新聞 | 偏綠/本土 | 若品牌涉及兩岸議題，報導角度會有立場 |
| 中時電子報 | 偏藍/親中 | 同上 |
| ETtoday | 相對中立，流量導向 | 標題聳動，但達成廣泛覆蓋 |
| 報導者 | 調查新聞，無明顯黨派 | 出現在報導者通常意味深度負面調查 |

**實務建議**：情緒分析不能只依賴工具，需人工確認政治脈絡下的報導方向。

### 節慶與輿論週期

台灣有幾個輿論高峰期，品牌監測需提前備戰：

| 時期 | 輿論特性 | 監測重點 |
|------|---------|---------|
| 農曆春節前 2 週 | 消費品輿論高峰 | 禮品評測、價格抱怨 |
| 228、清明前後 | 政治敏感期 | 品牌任何政治表態都會被放大 |
| 雙十一、黑五 | 電商輿論高峰 | 物流、客服負評密集爆發 |
| 選舉前 3 個月 | 整體政治敏感 | 避免品牌在任何議題上站隊 |
| 颱風季 (7-9月) | 民生情緒敏感 | 若品牌有服務中斷，準備好即時說明 |

## 危機偵測：台灣特有模式

### 模式一：PTT 燒到 ETtoday 鏈

**時間線**（典型案例）：
```
T+0h : PTT 八卦板出現負面文章
T+2h : 推文數破 100
T+4h : ETtoday 或三立記者看到，發出新聞
T+6h : LINE TODAY 推播，觸及 300-500 萬用戶
T+8h : Facebook 粉絲團、社團二次擴散
T+24h : 其他媒體跟進報導
```

**觸發條件**：PR 主管在 T+2h 必須知情，才有機會在 T+4h 前準備聲明。

### 模式二：Dcard 情緒爆炸（消費者品牌）

Dcard 的負面貼文病毒擴散機制與 PTT 不同：

```
Dcard 負評文 → IG Stories 截圖分享 → 被有粉絲的帳號轉發
→ Facebook 社團（如「靠北XX」類型社團）二次擴散
→ 主流媒體以「網友怒轟」角度報導
```

**關鍵監測點**：除 Dcard 本身，需監測 FB 上「靠北」類社團（各行業幾乎都有對應的消費者抱怨社團）。

### 模式三：KOL 點火

台灣 YouTube 科技類 KOL（如 JOEMAN、陳寗、3C 達人）的單一負面評測影片可以：
- 產生 20-100 萬次觀看
- 帶動 PTT/Dcard 後續討論
- 被媒體引用作為「評測界反應」

**監測動作**：訂閱主要 KOL 的 YouTube 頻道新影片通知，不能只靠關鍵字爬蟲。

## 競品監測：台灣執行細節

依照 SKILL.md 的 IRON LAW — 競品監測必須有觸發行動，以下是台灣場景的具體觸發設定：

### 競品監測觸發表（台灣版）

| 觸發條件 | 偵測方式 | 觸發動作 |
|---------|---------|---------|
| 競品在 PTT 八卦板出現 100+ 推文文章 | 每日手動掃描 | 24 小時內通知行銷主管，評估是否需回應 |
| 競品推出新產品（出現在科技媒體） | Google Alerts: "{競品} 新品" | 24 小時內給出競品分析摘要 |
| 競品負面事件爆發 | OpView 情緒異常警示 | 記錄時間點（可能成為市場機會），謹慎勿乘人之危 |
| 競品 KOL 合作影片上線 | 訂閱競品常合作 KOL | 評估我方是否需要相同 KOL 佈局 |

## 工具選型決策樹

```
品牌是否有台灣市場重要性？
└─ 是
   ├─ 月預算 < 5,000 NT?
   │   └─ 是 → Google Alerts + PTT/Dcard 手動監測 SOP
   │
   ├─ 月預算 5,000~30,000 NT?
   │   └─ 是 → OpView（必選） + Google Alerts
   │
   └─ 月預算 > 30,000 NT?
       └─ 是 → Meltwater/Talkwalker（新聞）+ OpView（論壇）
              + 人工 PTT 每日掃描（工具不可完全取代）

品牌是否屬於以下類別？
├─ 遊戲/ACG → 必須手動監測巴哈姆特
├─ 3C/汽車 → 必須手動監測 Mobile01
├─ 年輕消費品 → Dcard 優先
└─ 其他 → PTT 八卦板優先
```
