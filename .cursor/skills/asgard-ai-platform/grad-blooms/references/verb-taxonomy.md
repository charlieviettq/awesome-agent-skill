# Verb Taxonomy for Bloom's Revised Taxonomy

Action verbs are the operational core of Bloom's framework. A verb makes a learning objective **measurable**: it specifies the observable behavior an instructor can assess. This reference provides the complete verb inventory for all six cognitive levels, disambiguation rules for ambiguous verbs, and a worked example showing how verb choice changes the actual cognitive demand of an objective.

---

## The Problem with Generic Verbs

The following verbs appear in thousands of course syllabi but are **unacceptable** as Bloom's objectives because they are unobservable or cognitively ambiguous:

| Banned Verb | Why it Fails |
|-------------|--------------|
| understand | Internal state; not observable. Replace with: explain, classify, paraphrase |
| know | Same problem as "understand" |
| learn | Process, not outcome |
| appreciate | Affective, not cognitive |
| be familiar with | Unobservable gradient |
| grasp | Metaphor, not behavior |
| study | Activity, not outcome |
| cover | Course activity, not student behavior |

**Rule**: If you cannot write a test question or rubric criterion that directly observes the verb, replace it.

## Level 1 — Remember

**Cognitive demand**: Retrieve relevant knowledge from long-term memory. No transformation required.

**Canonical verbs:**

| Verb | Sample Objective Stem |
|------|-----------------------|
| define | Define the term "opportunity cost" |
| list | List the four stages of mitosis |
| name | Name the three branches of the US federal government |
| recall | Recall the date of the Treaty of Westphalia |
| recognize | Recognize which graph represents exponential growth |
| identify | Identify the key signature of a musical excerpt |
| label | Label the parts of a neuron on a diagram |
| match | Match each economist to their primary theory |
| reproduce | Reproduce the quadratic formula from memory |
| state | State Ohm's Law |

**Disambiguation — "identify" is overloaded:**  
At the Remember level, "identify" means picking the correct item from a set where no transformation is needed (e.g., "identify which of the following is a primary source"). At the Analyze level, "identify" means locating a pattern or structural element the student must derive (e.g., "identify the logical fallacy in this argument"). Context determines level, not verb alone.

**Typical assessment formats:**
- Multiple-choice with verbatim or near-verbatim answer
- Fill-in-the-blank (closed response)
- Matching exercises
- Short-answer recall (< 1 sentence)

---

## Level 2 — Understand

**Cognitive demand**: Construct meaning from instructional messages — translate, interpret, classify, compare, explain, infer. The student transforms the information but does not apply it to a new problem.

**Canonical verbs:**

| Verb | Sample Objective Stem |
|------|-----------------------|
| explain | Explain why inflation reduces purchasing power |
| describe | Describe the water cycle in your own words |
| summarize | Summarize the main argument of the article |
| paraphrase | Paraphrase the definition of natural selection |
| classify | Classify the following chemical reactions as endothermic or exothermic |
| compare | Compare the parliamentary and presidential systems |
| contrast | Contrast Keynesian and monetarist approaches to recession |
| interpret | Interpret the trend shown in the bar chart |
| illustrate | Illustrate the concept of scarcity with an original example |
| predict | Predict what will happen to demand if price increases |
| infer | Infer the author's attitude toward industrialization |
| generalize | Generalize the pattern from the given data set |

**Disambiguation — "explain" vs "analyze":**  
"Explain the causes of WWI" is Understand if students are expected to reproduce a taught causal account. It becomes Analyze if students must identify causal relationships in primary sources they haven't studied before.

**Disambiguation — "predict" at Level 2:**  
Predict is Understand when the prediction follows directly from a rule the student was taught (e.g., "predict the direction of a reaction using Le Chatelier's principle"). It escalates to Evaluate when the student must weigh competing evidence and justify a probabilistic judgment.

**Typical assessment formats:**
- Short paragraph explanations
- Concept maps (fill in vs. construct distinguishes level)
- Paraphrase exercises
- Classification tables with taught categories

## Level 3 — Apply

**Cognitive demand**: Execute or use a procedure in a given situation. The key marker is a **novel situation** — the student uses a known procedure but has not seen this exact problem before.

**Canonical verbs:**

| Verb | Sample Objective Stem |
|------|-----------------------|
| solve | Solve the following system of linear equations |
| calculate | Calculate the net present value of the investment |
| use | Use the SWOT framework to assess this startup |
| execute | Execute the SQL query to retrieve the requested records |
| implement | Implement a binary search algorithm in Python |
| demonstrate | Demonstrate proper hand-washing technique |
| apply | Apply Porter's Five Forces to the airline industry |
| construct | Construct a confidence interval for the given sample |
| produce | Produce a cash flow statement from the raw data |
| modify | Modify the code so it handles null input without crashing |
| sketch | Sketch the free-body diagram for the described scenario |
| simulate | Simulate the prisoner's dilemma with a partner |

**The novelty criterion:**  
Apply requires that the problem is **unfamiliar** to the student. If students have worked through the exact same type of problem in class, retrieval of the memorized procedure downgrades the objective to Remember. Apply means the student must recognize *which* procedure fits and then execute it.

**Typical assessment formats:**
- Problem sets with novel numerical inputs
- Case studies with scaffolded questions
- Lab practicals
- Code writing given a specification

## Level 4 — Analyze

**Cognitive demand**: Break material into constituent parts, determine relationships, distinguish relevant from irrelevant, identify organizational principles or logical structure.

**Canonical verbs:**

| Verb | Sample Objective Stem |
|------|-----------------------|
| analyze | Analyze the rhetorical strategies in the speech |
| differentiate | Differentiate between correlation and causation in the study |
| distinguish | Distinguish the valid from the invalid inferences |
| examine | Examine how the author's background influenced the text |
| deconstruct | Deconstruct the argument into premises and conclusion |
| break down | Break down the cost structure into fixed and variable components |
| attribute | Attribute the variance in sales to its contributing factors |
| compare | Compare the underlying assumptions of the two models |
| organize | Organize the following evidence into categories that support each hypothesis |
| question | Question the assumptions embedded in the policy proposal |
| test | Test whether the dataset violates the assumptions of linear regression |
| outline | Outline the logical structure of the argument |

**Disambiguation — "compare" appears at Levels 2 and 4:**  
At Understand, compare means describing similarities and differences between taught categories (e.g., "compare photosynthesis and cellular respiration using class notes"). At Analyze, compare means examining structural or inferential relationships to reach a conclusion about which is more appropriate, why a difference exists, or what the difference implies (e.g., "compare the sampling strategies of the two studies and explain how the difference affects generalizability").

**Typical assessment formats:**
- Structured analytical essays
- Case study decomposition
- Data interpretation with reasoning justification
- Argument mapping
- Peer review using a rubric

---

## Level 5 — Evaluate

**Cognitive demand**: Make judgments based on criteria and standards. Justify the judgment with evidence or logical reasoning.

**Canonical verbs:**

| Verb | Sample Objective Stem |
|------|-----------------------|
| evaluate | Evaluate the strength of the evidence for the hypothesis |
| judge | Judge which negotiation strategy best fits the scenario |
| critique | Critique the methodology of the research article |
| justify | Justify your recommended pricing strategy with data |
| defend | Defend your policy recommendation against a counterargument |
| argue | Argue for or against the proposed regulatory change |
| assess | Assess the reliability of each source |
| recommend | Recommend which vendor to select and explain why |
| prioritize | Prioritize the project risks by severity and likelihood |
| appraise | Appraise the artistic merit of the film using established criteria |
| rate | Rate the usability of the interface against Nielsen's heuristics |
| rank | Rank the following solutions by feasibility and impact |

**The criteria requirement:**  
Evaluate demands explicit criteria. An objective that asks students to "pick the best option" without requiring justification is actually Apply (execute a decision procedure) or even Remember (recall the preferred approach). The observable behavior must include the judgment *and* the reasoning grounded in stated standards.

**Disambiguation — "assess" at Levels 4 and 5:**  
"Assess the components of the system" (Analyze — decompose and describe). "Assess the adequacy of the system against the stated requirements" (Evaluate — judge against criteria).

**Typical assessment formats:**
- Position papers with counterargument response
- Structured debates
- Rubric-based peer evaluation with written justification
- Recommendation memos
- Grant or proposal review exercises

---

## Level 6 — Create

**Cognitive demand**: Put elements together to form a novel, coherent whole or reorganize elements into a new pattern or structure. The student produces something that did not exist before — a plan, hypothesis, design, or artifact.

**Canonical verbs:**

| Verb | Sample Objective Stem |
|------|-----------------------|
| design | Design a data collection instrument for the research question |
| create | Create a marketing campaign for the new product segment |
| construct | Construct a theoretical model explaining the observed pattern |
| compose | Compose an original piece in the style of the period |
| develop | Develop a training program for onboarding new analysts |
| generate | Generate three alternative hypotheses consistent with the data |
| plan | Plan an intervention study to test the proposed mechanism |
| produce | Produce a working prototype of the user interface |
| propose | Propose a revised organizational structure with rationale |
| formulate | Formulate a research question addressable with available data |
| invent | Invent a mnemonic system for learning the periodic table groups |
| synthesize | Synthesize findings from three conflicting studies into a unified account |

**The novelty requirement:**  
Create means the student generates something that is new to *them*, not necessarily new to the world. A student who follows a template step-by-step is likely at Apply. Create requires decisions about structure, content, or approach that are not pre-specified.

**The hierarchy note (from SKILL.md):**  
The revised taxonomy acknowledges that Evaluate and Create can occasionally reverse — some creative tasks (e.g., free improvisation) do not require prior evaluation, while some evaluation tasks require constructing a new evaluative framework. When in doubt, inspect the actual cognitive demand rather than treating the hierarchy as absolute.

**Typical assessment formats:**
- Original research proposals
- Design projects with open-ended constraints
- Capstone portfolios
- Creative works with reflective commentary
- Business plans or policy proposals

## Quick-Disambiguation Decision Tree

When you are unsure which level a verb belongs to, ask these questions in sequence:

```
1. Is the student RETRIEVING verbatim or near-verbatim information?
   YES → Remember

2. Is the student TRANSFORMING or EXPLAINING without a novel problem?
   YES → Understand

3. Is the student EXECUTING a known procedure on a novel problem instance?
   YES → Apply

4. Is the student DECOMPOSING structure, identifying relationships,
   or distinguishing relevant from irrelevant elements?
   YES → Analyze

5. Is the student JUDGING against explicit criteria and justifying?
   YES → Evaluate

6. Is the student PRODUCING a novel artifact, plan, or structure
   not specified in the prompt?
   YES → Create

If MULTIPLE answers apply, choose the HIGHEST level that is fully
satisfied by what the assessment requires.
```

## Worked Example: The Same Topic at Six Levels

Topic: **Supply and demand in a competitive market**

| Level | Objective | Why This Level |
|-------|-----------|----------------|
| Remember | State the law of demand. | Verbatim retrieval of a taught definition. |
| Understand | Explain in your own words why a price ceiling causes a shortage. | Student transforms a taught concept; no new problem. |
| Apply | Given the following supply and demand schedule, calculate the equilibrium price and quantity. | Novel numerical instance; uses known procedure. |
| Analyze | Examine the 2022 wheat price data and identify which supply-side and demand-side factors account for the observed price spike. | Decompose a real event into causal components using analytical framework. |
| Evaluate | Assess whether a price ceiling on rental housing is an effective policy for increasing affordability. Justify your position using economic evidence and criteria of efficiency and equity. | Judgment against explicit criteria with evidence-based justification. |
| Create | Design a market intervention proposal for a commodity of your choice, specifying the mechanism, predicted effects on price and quantity, and distributional consequences. | Student generates a novel policy design not pre-specified in the prompt. |

Notice: the topic is identical across all six rows. The cognitive demand shifts entirely through the verb and the task structure.

## Verb Collision Table

These verbs commonly appear at multiple levels. Use the contextual markers to resolve ambiguity.

| Verb | Lower Reading | Higher Reading | Distinguishing Marker |
|------|---------------|----------------|-----------------------|
| identify | Pick the correct item from a given set (Remember) | Locate a structural feature by reasoning (Analyze) | Is the feature given or must it be derived? |
| describe | Restate characteristics of a taught concept (Understand) | Characterize a newly encountered phenomenon (Analyze) | Is it familiar material or novel observation? |
| compare | List similarities and differences using taught categories (Understand) | Examine structural implications of differences (Analyze) | Does the comparison lead to a judgment or conclusion? |
| explain | Paraphrase a causal account from instruction (Understand) | Provide a mechanistic account the student constructs (Analyze) | Is the causal account taught or derived? |
| predict | Apply a rule to forecast an outcome (Apply) | Weigh competing evidence to probabilistic judgment (Evaluate) | Is the prediction rule-based or evidence-weighed? |
| assess | Decompose and describe components (Analyze) | Judge against explicit criteria (Evaluate) | Does the task specify criteria and require justification? |
| demonstrate | Show how a procedure works (Apply) | Prove a claim with evidence (Evaluate) | Is the student executing a procedure or making an argument? |
| construct | Build an artifact using specified steps (Apply) | Design a novel structure with open decisions (Create) | Are the design decisions pre-specified or student-generated? |

## Verb Lists by Knowledge Type

The knowledge dimension interacts with the cognitive level. The same verb ("apply") reads differently depending on what type of knowledge is being applied.

| Knowledge Type | Level 3 (Apply) Examples | Level 4 (Analyze) Examples |
|----------------|--------------------------|---------------------------|
| Factual | Use the periodic table to identify the element | Examine which factual discrepancies between sources affect the conclusion |
| Conceptual | Apply the concept of confirmation bias to evaluate a news article | Differentiate which conceptual framework better accounts for the observed behavior |
| Procedural | Execute the hypothesis test for the given data | Deconstruct where in the procedure the error was introduced |
| Metacognitive | Implement a self-monitoring strategy during the exam | Analyze which of your own cognitive biases is influencing your interpretation |

## Common Verb Mapping Errors

**Error 1 — Verb laundering**  
Writing "analyze the definition of X" when students are expected to recite a memorized analytical summary. The word "analyze" doesn't elevate a Remember task. The cognitive demand is in the *what is required*, not the verb on the page.

**Error 2 — Over-leveling foundational courses**  
Forcing Create-level objectives onto an introductory module where students lack the conceptual foundation. The Iron Law applies: you cannot create without something to build from. Introductory courses legitimately concentrate at Remember, Understand, and Apply.

**Error 3 — Under-leveling advanced courses**  
Retaining "define," "list," and "describe" objectives in graduate seminars where the expectation is critique and synthesis. Audit verb distribution: if > 70% of objectives are at Levels 1–2 in a 400-level course, the curriculum may be misaligned with student preparation.

**Error 4 — Ignoring the condition clause**  
"Analyze market conditions" without specifying *what* the student analyzes, *with what*, and *to what end* is too vague to assess. Complete objectives need: verb + content + condition/constraint. Example: "Analyze the pricing strategy of a provided case company using Porter's Five Forces and identify the dominant competitive pressure."
