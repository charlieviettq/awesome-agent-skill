# Example: 數位廣告支出對零售門市銷售的面板效應

## Scenario

**用戶問題**：一家台灣連鎖零售商（假設為「全島超市」）在 2019–2023 年間持有旗下 84 家門市的季度資料，包含門市銷售額、每季數位廣告預算、店長任期、門市所在縣市失業率，以及門市開幕年份。

研究員 Mei-Ling 想知道：「每多投入一元數位廣告，能帶來多少銷售增量？」她擔心各門市本身的「地段好壞」、「客群忠誠度」等不可觀察因素會混淆估計結果。行銷長另外懷疑過去季度銷售好不好也會影響本季廣告預算（反向因果）。

**資料規格**：
- N = 84 門市，T = 20 季（2019Q1–2023Q4）
- 平衡面板（無缺失期）
- 依變數：`ln_sales`（門市季度銷售額取對數）
- 主要解釋變數：`ln_adspend`（季度數位廣告支出取對數）
- 控制變數：`manager_tenure`（店長任期，季）、`city_unemp`（縣市失業率）、`store_age`（門市開幕年數）

---

## Analysis

### Step 1 — 探索面板結構

**面板基本統計**：

| 維度 | 值 |
|------|----|
| 門市數 (N) | 84 |
| 時間期數 (T) | 20 季 |
| 總觀測值 | 1,680 |
| 是否平衡 | 是 |

**組內 vs 組間變異分解**（`ln_adspend`）：

| 變異來源 | 標準差 |
|----------|--------|
| 整體 (overall) | 0.612 |
| 組間 (between) | 0.481 |
| 組內 (within) | 0.374 |

組內變異占整體約 61%，FE 估計有足夠辨識力。

> **注意**：`store_age` 每季固定增加 0.25，幾乎是時間的線性函數；納入時間固定效果後此變數將被共線性吸收，不列入主模型。

門市層級趨勢圖顯示各店銷售**截距**差異顯著（地段效果），但廣告彈性的**斜率**相對一致 — 符合固定效果設定的直覺。

---

### Step 2 — 估計 FE 與 RE 模型

**模型設定**（兩者皆使用門市層級 cluster-robust SE）：

$$\ln\_sales_{it} = \beta_1 \ln\_adspend_{it} + \beta_2 \text{manager\_tenure}_{it} + \beta_3 \text{city\_unemp}_{it} + \alpha_i + \lambda_t + \varepsilon_{it}$$

- $\alpha_i$：門市固定效果（FE）或隨機效果（RE）
- $\lambda_t$：季度時間固定效果（兩個模型皆納入，控制 COVID-19 等共同需求衝擊）

**估計結果**（括號內為 cluster-robust 標準誤）：

| 變數 | FE (β) | RE (β) |
|------|--------|--------|
| `ln_adspend` | **0.183** (0.041)*** | **0.224** (0.037)*** |
| `manager_tenure` | 0.008 (0.004)* | 0.011 (0.004)** |
| `city_unemp` | −0.092 (0.031)** | −0.079 (0.028)** |
| 時間固定效果 | ✓ | ✓ |
| R² (within) | 0.347 | 0.312 |
| 觀測數 | 1,680 | 1,680 |

FE 估計廣告彈性 0.183（支出增加 10% → 銷售增加約 1.8%）；RE 稍高（0.224），暗示若 RE 不一致，方向是高估。

> **IRON LAW 確認**：FE 控制了「地段好壞」、「客群忠誠度」等時間不變因素，但**時間變動的混淆因素**（如競爭對手季節性促銷）仍未完全排除。

---

### Step 3 — Hausman 檢定

**H₀**：個體效果與解釋變數無相關（RE 一致）  
**H₁**：個體效果與解釋變數相關（僅 FE 一致）

| 統計量 | 數值 |
|--------|------|
| Hausman χ²(3) | 14.72 |
| p 值 | 0.002 |
| 決策 | **拒絕 H₀ → 採用 FE** |

主要驅動來自 `ln_adspend` 係數差異（Δ = 0.041）。直覺上合理：公司傾向將更多行銷資源分配給地段優越的門市，RE 因此高估廣告效果約 22%。

**結論**：FE 係數 0.183 為主要報告值。

---

### Step 4 — 動態 GMM（處理反向因果）

行銷長擔心「本期銷售好 → 下期廣告預算增加」形成反向因果。在 FE 模型加入 `ln_sales_{t-1}` 後，Nickell 偏誤存在（T = 20 尚可但不可忽略），採用 **Arellano-Bond System GMM**。

**工具變數策略**：
- `ln_adspend` 的 t−2、t−3 落後期作為差分方程工具
- `ln_adspend` 的一階差分作為水平方程附加工具（System GMM 條件）
- 工具變數總數：22（使用壓縮矩陣，避免 instrument proliferation）

**GMM 估計結果**（cluster-robust SE）：

| 變數 | System GMM (β) |
|------|----------------|
| `ln_sales_{t-1}` | 0.341 (0.087)*** |
| `ln_adspend` | **0.147** (0.053)** |
| `manager_tenure` | 0.006 (0.005) |
| `city_unemp` | −0.081 (0.034)* |

**診斷檢定**：

| 檢定 | 統計量 | p 值 | 結果 |
|------|--------|-------|------|
| AR(1) | −4.21 | < 0.001 | 預期顯著 ✓ |
| AR(2) | 1.18 | 0.238 | 通過（無二階自相關）✓ |
| Hansen J (df = 14) | 18.63 | 0.181 | 工具有效性通過 ✓ |

廣告彈性在控制銷售動態後降至 0.147，反向因果確實存在，但規模有限；FE 模型僅高估約 25%。

---

## Result

## Panel Data Analysis: 全島超市數位廣告效果研究（2019–2023）

### Panel Structure

| 維度 | 值 |
|------|----|
| 門市數 (N) | 84 |
| 時間期數 (T) | 20 季（2019Q1–2023Q4） |
| 是否平衡 | 是 |

### Estimation Results

| 變數 | FE (β) | RE (β) | GMM (β) |
|------|--------|--------|---------|
| `ln_adspend` | 0.183 (0.041)*** | 0.224 (0.037)*** | 0.147 (0.053)** |
| `manager_tenure` | 0.008 (0.004)* | 0.011 (0.004)** | 0.006 (0.005) |
| `city_unemp` | −0.092 (0.031)** | −0.079 (0.028)** | −0.081 (0.034)* |
| `ln_sales_{t-1}` | — | — | 0.341 (0.087)*** |

*p<0.05, **p<0.01, ***p<0.001；cluster-robust SE at 門市層級

### Model Selection

| 檢定 | 統計量 | p 值 | 決策 |
|------|--------|-------|------|
| Hausman χ²(3) | 14.72 | 0.002 | 採用 FE |
| AR(2) | 1.18 | 0.238 | 通過 |
| Hansen J (df = 14) | 18.63 | 0.181 | 工具有效 |

### Key Findings

- 數位廣告的銷售彈性為 **0.147–0.183**（靜態 FE 至動態 GMM），廣告支出增加 10% 對應銷售增加 1.5–1.8%
- Hausman 檢定確認地段優越的門市同時獲得更多廣告資源，RE 高估廣告效果約 22%，FE 為一致估計量
- 加入銷售動態後廣告彈性略降，反向因果規模有限（約 0.036），FE 結果仍具實務參考價值
- 縣市失業率每升高 1 個百分點，門市季度銷售下降約 8–9%，宏觀環境效果顯著

### Limitations

- 嚴格外生性假設可能因競爭對手廣告行為（時間變動混淆）而違反；建議後續納入縣市層級競爭強度指標
- GMM 使用 t−2、t−3 期工具，隱含廣告決策需滯後兩季才有效 — 需與業務單位確認預算規劃週期是否支持此假設
- `store_age` 與時間固定效果高度共線，無法單獨識別「門市老化效應」
- 樣本限於單一零售商，跨業態外部效度有限
