# Meta Ads Creative Guidelines

## Creative Formats & Specs

| Format | Placement | Recommended Size | Aspect Ratio | Max File Size | Notes |
|--------|-----------|-----------------|--------------|---------------|-------|
| Single Image | Feed, Stories, Reels | 1080×1080 / 1080×1920 | 1:1 / 9:16 | 30 MB | JPG or PNG |
| Single Video | Feed | 1080×1080 | 1:1 | 4 GB | MP4, 15s–3min |
| Stories Video | Stories | 1080×1920 | 9:16 | 4 GB | MP4, ≤15s for full-screen |
| Reels | Reels | 1080×1920 | 9:16 | 4 GB | MP4, ≤60s |
| Carousel | Feed | 1080×1080 per card | 1:1 | 30 MB per card | 2–10 cards |
| Collection | Feed | Cover: 1200×628 | 1.91:1 | — | Requires catalog |

**Safe zone rule**: Keep text and key visuals in the center 80% of the frame. Stories and Reels crop top/bottom on some devices — anything within 250px of edges may be hidden.

## The 3-Second Hook Rule

Meta研究顯示，用戶決定停留或滑走的時間 < 3 秒。廣告前3秒必須：

1. **視覺衝擊**：動態、對比強烈的色彩、或人臉（眼神接觸效果尤佳）
2. **問題點燃**：在字幕或旁白中點出受眾的痛點或慾望
3. **品牌識別**：Logo或產品在3秒內出現（不要留到影片尾段）

**不可做的事**：前3秒放LOGO動畫或品牌片頭。這是品牌廣告的舊思維——在Meta上等同於浪費預算。

## Text Overlay 規則

Meta已移除「20%文字規則」的硬性限制，但演算法仍懲罰文字過多的圖片。

**實務原則**：
- 圖片文字 ≤ 20% 圖面面積（視覺判斷）
- 廣告文字（Primary Text）：前125字元最關鍵，超過需點「more」才顯示
- Headline（標題欄）：27字元內，超過截斷
- Description（說明欄）：27字元，僅部分版位顯示

```
Primary Text 結構建議：
[痛點/情境]（1句）→ [解法/產品]（1句）→ [社會認同/數字]（1句）→ [CTA]（1句）

範例：
台灣有 85% 的上班族有肩頸痠痛問題。
WorkFit 人體工學椅採用腰椎自適應系統，讓久坐不再是折磨。
已有 12,000+ 用戶告別肩頸問題。
👉 限時免費試坐 → [了解更多]
```

## Creative Type 選擇決策樹

```
你的目標是？
│
├─ 直接轉換（購買/填單）
│   ├─ 產品是視覺型（服飾/食品/家居）→ Carousel 或 Collection
│   ├─ 產品需要展示功能 → Video（展示使用流程，15-30秒）
│   └─ 服務型/B2B → Single Image + 強文案
│
├─ 再行銷（曾看過網站/加購）
│   ├─ 電商 → Dynamic Product Ads（DPA，自動抓購物車商品）
│   └─ 服務 → Carousel（展示不同方案/評價）
│
└─ 品牌認知
    ├─ 預算充足 → Reels/Video（完整故事）
    └─ 預算有限 → Single Image（強視覺，高頻次）
```

## Video Creative 結構框架（15-30秒）

| 時間區段 | 內容 | 目的 |
|---------|------|------|
| 0-3s | 視覺鉤子 + 痛點 | 阻止滑走 |
| 3-8s | 解法展示（產品使用） | 建立興趣 |
| 8-20s | 社會認同（數字/評論）或差異點 | 消除疑慮 |
| 20-30s | 明確CTA + 優惠 | 驅動行動 |

**字幕是必要的**：Meta預設靜音播放，85%用戶不開音效。沒有字幕 = 失去85%受眾。

**開頭不用Fade-in**：直接切入動態畫面。Fade-in在靜音靜態的Feed中完全失效。

## Carousel 最佳實踐

Carousel適合兩種策略：

### 策略A：連續故事型（提升完閱率）
每張卡片是下一張的懸念，迫使用戶向右滑：
```
卡1: 「你知道你每天損失多少時間嗎？」（問題）
卡2: 「平均工作者每天浪費 2.3 小時在低效任務」（數字）
卡3: 「原因是這個...」（分析）
卡4: 「TaskFlow 解決了它」（解法）
卡5: 「限時方案 →」（CTA）
```

### 策略B：產品展示型（電商）
每張卡片是獨立商品，最後一張放品牌主頁或優惠：
- 前2張放最暢銷/最高ROAS的商品
- 每張卡片有獨立的Landing Page URL
- 標題欄統一格式：`{商品名} — NT${價格}`

## A/B Test 執行規範

**單變數原則**：每次只測試一個元素。同時改圖片和文案，你不知道是哪個起作用。

### 測試優先順序（依影響力排序）

1. **Visual Hook**（圖片/影片前3秒）— 影響最大，先測這個
2. **Offer**（優惠/CTA）— 影響轉換率
3. **Primary Text 結構**（痛點先 vs 解法先）
4. **Headline**

### 統計顯著性門檻

不要用直覺判斷勝負。使用以下規則：

```
最小樣本（每變體）：
- 印象測試（CTR）：各 2,000 impressions
- 點擊測試（CVR）：各 200 clicks  
- 轉換測試（CPA）：各 50 conversions

信心水準：≥ 95%（p < 0.05）才宣告勝出
```

**實務計算**：

若A組 CTR = 2.1%（n=5,000），B組 CTR = 2.8%（n=5,000）

差距 = 0.7%，相對提升 = 33%

用 Chi-square test 或 [線上A/B計算器](計算工具)驗證，不要只看「感覺差很多」。

## Creative Fatigue 診斷與處理

**疲勞信號**（每週檢查）：

| 指標變化 | 判斷 |
|---------|------|
| CTR 下降 > 30%（vs 前7天均值） | 疲勞警示 |
| Frequency > 3.5（冷受眾） | 疲勞確認 |
| CPM 上升 + CTR 下降同時發生 | 高度疲勞，立即換素材 |
| CPA 上升 > 20% 且budget未變 | 先排除素材疲勞再查其他原因 |

**疲勞週期經驗值**：
- 冷受眾（CAC）：2-4週
- 再行銷（暖受眾）：1-2週（接觸頻率高）
- Reels/短影音：7-14天（演算法分發快，燒耗快）

**處置流程**：
```
偵測到疲勞信號
│
├─ Frequency 高但CTR尚可 → 擴大受眾（增加受眾規模）
├─ CTR下降但CVR穩定 → 換圖/影片，保留文案
├─ CVR下降但CTR穩定 → 換文案/Offer，保留視覺
└─ 全面下降 → 全新素材組，包含新角度/訴求
```

不要只是「重複使用舊素材+微調顏色」。演算法會識別相似素材並給予相同分發限制。

## iOS ATT 後的創意策略調整

iOS 14.5後，Meta歸因視窗縮短，部分轉換無法追蹤，直接影響：

- **再行銷受眾縮水**：自訂受眾（網站訪客）規模下降30-60%
- **演算法學習變慢**：轉換數據不完整，Target CPA/ROAS學習期拉長

**創意層面的對應**：

1. **增加Broad受眾創意比例**：依賴精準再行銷的策略失效 → 改用高品質Broad素材讓演算法自己找人
2. **強化廣告內的信任信號**：因為再行銷受眾縮水，冷受眾看到廣告更多 → 在廣告素材中加入評價數量、媒體報導、認證等社會認同
3. **影片優先**：影片觀看可在Meta平台內追蹤（不依賴iOS pixel），建立暖受眾的可靠方式

## Dynamic Creative Optimization (DCO)

Meta的DCO功能自動組合多組素材元素並找出最佳組合。

**適用情境**：
- 有多個素材方向但不確定哪個最好
- 受眾規模 > 50萬（太小則每個組合分配不到足夠流量）
- 月預算 > NT$30,000（讓演算法有足夠數據）

**DCO設定建議**：
```
上傳數量：
- 圖片/影片：3-5個（不同視覺方向，非同一圖的小變體）
- Primary Text：2-3個版本
- Headline：2-3個版本
- CTA按鈕：1-2個

避免：
- 上傳10+素材（每個組合分配流量太少）
- 所有圖片是同一款商品的不同角度（差異不足以測試）
```

## Taiwan 市場創意特殊考量

**語言**：繁體中文為主，但年輕族群（18-34）可混用英文關鍵詞（保持酷感）。避免簡體字，台灣用戶對此敏感。

**視覺偏好**：
- 台灣市場對「高CP值」、「限時」、「免運費」等促銷訴求反應強烈
- B2B/高單價產品：專業簡潔風格，過度活潑的設計降低信任感
- 美食/生活類：真實感比精修過度的圖更有效（素人感 > 廣告感）

**LINE整合**：台灣用戶習慣LINE生態，廣告Landing Page若能導向LINE官方帳號而非填表單，往往有更高的詢問轉換率。在Meta廣告素材上可直接標示「加LINE詢問」。

**假日節慶排程**：
| 節點 | 備貨時間 | 創意調整 |
|------|---------|---------|
| 雙11 (11/11) | 提前2週上新素材 | 倒數計時、最大折扣 |
| 母親節 (5月第2週日) | 提前1.5週 | 情感訴求優先 |
| 農曆年前 | 年前3週開始 | 送禮、囤貨訴求 |
| 開學季 (8月底) | 提前1週 | 學生/家長族群 |
