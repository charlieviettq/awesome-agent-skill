# Poll Reading — 民調判讀與引用準則

> 政治新聞最常出錯的地方之一就是民調引用。一個數字加上「最新民調顯示」就能誤導大量讀者。本檔列出**引用民調的最低標準**與**判讀框架**。

## 一、引用任一民調必含的五項資訊

**缺一不可。沒有就標 `[待查證]`,不要先引用再說。**

| 項目 | 為什麼重要 | 範例寫法 |
|------|----------|---------|
| **樣本數 (n)** | 樣本太小代表性差;n=500 與 n=1500 的可信度差異很大 | 「樣本數 1,068 人」 |
| **抽樣誤差 (MOE) 與信賴水準** | 沒有 MOE 的數字不能比較 | 「在 95% 信賴水準下,抽樣誤差 ±3.0 個百分點」 |
| **調查方法** | CATI(電話)、CAWI(網路)、面訪、混合 — 各方法代表性不同 | 「採用市內電話與行動電話雙底冊抽樣」 |
| **委託方 / 執行方** | 委託方影響中立性判讀;執行方反映專業性 | 「由 X 黨委託、Y 民調公司執行」 |
| **調查日期** | 一週前的民調與當下民意可能相差很多;選戰期間更敏感 | 「2026 年 4 月 8-10 日進行」 |

## 二、進階判讀:你應該額外想到的問題

### 1. 加權方式
- 多數民調會按 **年齡 / 性別 / 地區 / 教育程度** 加權,使樣本貼近母體結構。
- 加權後的數字 vs 原始數字 — 較佳實務是兩者都報告。
- 警訊:某些網路民調未做加權,結果會嚴重偏向高參與群體(年輕、城市、高教育)。

### 2. 問題用詞 (question wording)
- 「您支持 / 反對 X 政策?」vs「您是否支持政府推動 X?」— 後者隱含背書,結果會偏高。
- 「您是否支持 X,以解決 Y 問題?」— 「以解決 Y」是引導,違反中立。
- 重大爭議性題目應公布完整題目,不只報「贊成 / 反對 / 無意見」比例。

### 3. 順序效應 (order effect)
- 候選人列名順序、選項排列順序會影響回答。
- 嚴謹民調會輪換順序;若未輪換,結果可能受偏好順序影響。

### 4. 拒答率 / 完訪率 (response rate)
- 拒答率高代表完訪樣本可能不具代表性。
- 國際標準:面訪 50%+、電話 30%+ 為合理。低於此值需特別說明。

### 5. 時間點與事件
- 民調時段是否包含某重大事件之前 / 之後?
- 例:某事件爆發後 24 小時內的民調,反映「事件衝擊期」民意,不代表穩定民意。

### 6. 樣本框 (sampling frame)
- 「20 歲以上具投票權國民」vs「網路使用者」vs「特定縣市選民」— 框架決定誰被涵蓋。
- 報導應明示樣本框,讓讀者判斷是否代表「全民」「選民」「該縣市民」。

## 三、解讀數字的常見陷阱

### 1. 兩個民調差 3% = 有差別嗎?
- 若兩民調 MOE 各為 ±3%,實質差異需大於 6% 才稱「顯著」。
- 「A 候選人 47%、B 候選人 44%」搭配 MOE ±3% — 兩人實質可能相當,標題不應寫「A 領先 B」。

### 2. 同一民調不同次差異
- 同一機構、同樣方法,兩週相差 5% — 是真的變動還是誤差波動?
- 看趨勢線,不要看單點。連續 3-4 次同方向才有趨勢可言。

### 3. 不同機構民調差異
- 不同機構、不同方法、不同問題用詞的民調**不應直接相減比較**。
- 應標註方法差異,讀者才能判斷。

### 4. 「無意見」的位置
- 「贊成 40% / 反對 35% / 無意見 25%」— 無意見比例高,代表政策認知低,不是「贊成多」。
- 報導應呈現完整三項,不要只報贊成率。

### 5. 民調 vs 預測
- 民調是當下民意快照,不是未來預測。
- 投票率變化、未表態者投票傾向、轉換率等因素都會讓投票結果偏離民調。
- 標題避免「X 將當選」「Y 民調勝出已成定局」等預測式語氣。

## 四、選舉民調 vs 一般民意調查

| 項目 | 選舉民調 | 一般民意調查 |
|------|---------|-------------|
| 受訪者範圍 | 通常限「該選區具投票權」 | 一般 20 歲以上國民 |
| 需問「投票意向」 | 是 | 否 |
| 「未表態 / 不知道」處理 | 重要,通常單獨計分 | 多歸入「無意見」 |
| 加權重點 | 過往投票率 + 人口結構 | 主要看人口結構 |
| 公布禁制期 | 投票日前 10 日(查證最新) | 無 |

## 五、實作檢查清單(發稿前)

- [ ] 五項基本資訊是否都標明?
- [ ] 是否使用「顯示」「指出」等中性動詞,而非「證明」「預測」「決定」?
- [ ] 數字差異是否在 MOE 範圍內?(若是,不應寫「領先 / 落後」)
- [ ] 委託方是否與議題有立場?(若有,應在內文點明)
- [ ] 是否與其他民調比較?(若是,有無說明方法差異?)
- [ ] 是否在禁制期內公布選舉民調?
- [ ] 是否將民調呈現為預測?(避免)

## 六、引用範本

**完整版**(深度報導):
> 全國產業總工會委託、A 民調公司於 2026 年 4 月 8-10 日針對 1,068 位 20 歲以上民眾進行的電話訪問顯示,62% 的受訪者認為今年最低工資調幅不足以反映物價上漲。本調查在 95% 信賴水準下,抽樣誤差為 ±3.0 個百分點,採市內電話與行動電話雙底冊抽樣,並依年齡、性別、地區及教育程度進行加權。

**精簡版**(短訊):
> 全國產業總工會於 4 月 10 日公布的民調(樣本 1,068、誤差 ±3.0%、電話訪問)顯示,62% 民眾認為最低工資調幅不足。

**錯誤示範**:
> 最新民調顯示,六成民眾認為最低工資調幅不足。
> *(缺:委託方、執行方、樣本數、誤差、方法、日期 — 六項都沒有)*
