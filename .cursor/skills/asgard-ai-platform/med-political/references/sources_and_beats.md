## 政府消息來源層級(分級慎用)

1. **官方公告(蓋印的公報、官網新聞)**:可直接引,標明出處。
2. **新聞稿(部會發出)**:可引,但須標「依 XX 部新聞稿」,提醒讀者這是機關自述。
3. **記者會逐字稿**:準官方,可引。注意主持人問答 vs 主動發言的差別。
4. **背景簡報(off-the-record / on-background)**:極度小心。這類訊息的權威性建立在機關信譽上,但無責任歸屬。需獨立查證。
5. **不具名「政府高層」**:除非有具體風險與編輯台核可,不應採用。台灣政治新聞慣用「府院高層」「黨內高層」過度,讀者已對其失去判讀能力。

## 政黨與政治組織

- 各政黨中央黨部:官方立場、新聞稿、黨團會議結論。
- 黨團辦公室:每週黨團會議後通常有立場聲明。
- 智庫:財團法人國家政策研究基金會(藍)、新境界文教基金會(綠)、台灣民主基金會、中央研究院政治所(學術)。立場差異需在引用時點明。
- 公民團體:勞工陣線、台灣人權促進會、地球公民基金會、婦女新知、原住民族青年陣線等。引用時標明組織立場。

## 民調機構

- **學術導向**:中央研究院、政大選研中心、台大政治系、TVBS 民意調查中心、臺灣民意基金會。
- **媒體/政黨委託**:聯合報系、TVBS、ETtoday、自由時報、各黨內部民調。
- **新型網路民調**:Yahoo、PTT 民調、菱傳媒、美麗島電子報。注意:網路民調樣本代表性弱於電話民調,引用時須標明調查方法。
- **國際**:Pew Research、YouGov、Gallup(關於台灣議題的國際數據)。

## 採訪 Beat 配置(典型分工)

| Beat | 主要場域 | 必訪對象 |
|------|----------|---------|
| 府院線 | 總統府、行政院 | 發言人、重大政策推動部會 |
| 立法院線 | 議場、各委員會、黨團 | 提案立委、委員會召委、黨團幹事長 |
| 黨政線 | 各黨中央 + 黨團 | 主席、秘書長、黨團三長 |
| 選戰線(選舉期間) | 候選人陣營 | 競選總部、發言人、政策主管 |
| 司法/檢調 | 司法院、各級法院、地檢署 | 院長、發言人、重大案件主辦 |
| 縣市政府 | 各縣市府 | 首長、發言人、重大政策局處 |
| 民代/議會 | 縣市議會、鄉鎮代會 | 議長、各黨團、委員會召委 |

## 何時應該打給多少人

| 報導類型 | 最少訪問數 |
|---------|----------|
| 即時短訊(< 300字) | 1 名(具名)+ 公開資料 |
| 政策追蹤(800-1500字) | 推動方 + 反對方 + 中立專家 (3+) |
| 選戰報導 | 該事件涉及之所有候選人 + 至少 1 名中立分析者 |
| 深度調查 | 5+ 名,跨陣營跨層級,含一手文件 |
| 評論/op-ed | (不適用,但事實基礎仍應交叉查證) |
