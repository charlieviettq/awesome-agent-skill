# Frames & Neutrality — 政治新聞的隱性偏見地圖

> Iron Law 講「No Stealth Bias」,本檔提供具體的字詞清單、結構檢查、與替代寫法。

## 一、Loaded Language 替換對照表

### 1. 描述政治人物的字詞

| 帶有評價 | 中立替代 | 何時可用評價詞 |
|---------|---------|---------------|
| 「強硬派」「鷹派」 | 「主張對 X 採取強硬立場的 ...」 | 該詞為對方自我認同時 |
| 「溫和派」「鴿派」 | 「主張與 X 對話的 ...」 | 同上 |
| 「親中」 | 「主張兩岸經貿合作 / 對話 ...」 | 該人士確有具體政策主張 |
| 「反中」 | 「主張對中國採取較強硬立場 ...」 | 該人士確有具體政策主張 |
| 「保守派」 | 「在 X 議題上持較保留立場 ...」 | 該詞為其陣營自稱 |
| 「進步派」 | 「主張 X 改革 ...」 | 同上 |
| 「極右 / 極左」 | 直接描述其具體主張 | 學術定義且該主張確屬光譜端點 |
| 「爭議性人物」 | 列出具體爭議事件 | (從不適用,直接列爭議事件) |
| 「政壇老將」 | 「自 X 年從政、曾任 Y 職務」 | (從不適用,直接列資歷) |

### 2. 描述政治事件的字詞

| 帶有評價 | 中立替代 | 帶來的偏向 |
|---------|---------|----------|
| 「黑箱」 | 「未公開的決策過程」「未召開公開會議」 | 暗示違法或不正當 |
| 「圖利」 | 「使 X 受益」「對 X 有利」 | 暗示違法 |
| 「跳腳」「砲轟」 | 「強烈反對」「批評」 | 過度戲劇化 |
| 「撂話」「放話」 | 「表示」「聲明」 | 暗示鬥爭 |
| 「派系角力」 | 「黨內 X 派與 Y 派立場分歧」 | 過度框架化 |
| 「鬥爭」 | 「爭議」「分歧」「衝突」 | 帶冷戰式語言 |
| 「親信」 | 「長期合作的同事 / 部屬」 | 暗示朋黨 |
| 「政變」 | 視具體情況描述事件本質 | 用語極重,涉法律意涵 |
| 「逼宮」 | 「要求 X 辭職 / 改變立場」 | 戲劇化 |
| 「鷹派外交」 | 「主張對 X 採取強硬措施的外交立場」 | 標籤化 |

### 3. 描述政策的字詞

| 帶有評價 | 中立替代 | 偏向 |
|---------|---------|------|
| 「德政」 | 「政策 X」 | 正面評價 |
| 「惡法」 | 「爭議性 / 多方反對的 X 條例」 | 負面評價 |
| 「便民措施」 | 「X 措施 / 服務」 | 正面評價 |
| 「擾民政策」 | 「對 X 群體增加 Y 義務的政策」 | 負面評價 |
| 「重大突破」 | 「X 政策的調整」 | 正面評價 |
| 「拙劣執行」 | 「執行未達預期目標」「實際效果為 X」 | 負面評價 |
| 「政策買票」 | 「於選舉前推出之 X 補助」 | 暗示動機 |

### 4. 動詞選擇

| 中性 | 暗示懷疑 | 暗示信任 |
|------|---------|---------|
| 「表示」 | 「聲稱」「揚言」 | 「強調」「指出」 |
| 「說」 | 「狡辯」「撇清」 | 「澄清」「說明」 |
| 「否認」 | 「狡辯」 | 「反駁」 |
| 「承認」 | 「終於承認」「不得不承認」 | 「坦言」 |
| 「批評」 | 「砲轟」「叫囂」 | 「指出」「質疑」 |
| 「拒絕」 | 「悍然拒絕」 | 「婉拒」 |

**原則**:中性動詞 (「表示」「說」「指出」) 應為預設;傾向性動詞需符合事實,且不應在同一篇報導中對不同陣營採用不同傾向。

## 二、結構性偏見來源

### 1. 引言順序
- 第一個引言通常被讀者視為「這篇文章的觀點」。
- 在爭議性議題中,輪流讓不同陣營出現在第一引言位置 — 跨多篇報導,而非單篇。
- 同篇內,若第一段引用 A 方,第二段就應引用 B 方。

### 2. 段落數量分配
- 一篇 10 段報導,A 方占 7 段、B 方占 1 段、中立分析 2 段 — 結構性失衡。
- 大致比例:主要對立各方應在同等數量級的段落中發言,即使內容篇幅有差。

### 3. 最後一段
- 結尾留給誰,讀者印象最深。
- 「結論段」若非中立分析,應輪換或避免。
- 警訊:結尾常見「X 表示,他將繼續為 Y 努力」式句型 — 通常是該人或其陣營提供的句子。

### 4. 標題
- 標題是讀者唯一接觸的內容(對 90%+ 讀者而言)。
- 標題中:
  - 主詞 / 受詞順序暗示主動 / 被動
  - 動詞選擇暗示評價
  - 帶問號的標題暗示懷疑
- 替代測試:把人名互換,若標題變得不公平,原標題就有偏見。

### 5. 圖片選擇
- 同一人物的多張照片可呈現完全不同形象(微笑 vs 怒吼 vs 疲憊)。
- 重大爭議事件照片應跨陣營等量呈現。
- 警訊:多次只用某陣營「表情不佳」的照片。

### 6. 背景資訊放置位置
- 同樣的背景資訊,放在第二段 vs 倒數第二段,讀者印象不同。
- 「對 A 不利的背景」高位呈現,「對 A 有利的背景」低位呈現 = 結構偏見。

## 三、平衡寫法範例

### 範例 1:政策爭議

**結構偏向版**:
> 民進黨立委王大明 9 日強烈批評勞動部最低工資調整幅度過低,根本是漠視勞工權益。「政府不該躲在數字後面」王大明說。對此,勞動部僅表示已參酌物價漲幅。但根據工會委託調查,六成民眾認為調幅不足。

**平衡版**:
> 民進黨立委王大明 9 日於立法院質詢時指出,今年最低工資調幅 4.6% 低於物價漲幅,要求重新檢討。「我認為政府應更積極反映勞工生活壓力」王大明說。
>
> 勞動部 10 日新聞稿回應,本次調整已參酌主計總處公布之消費者物價指數年增率 3.2%、經濟成長率 2.8% 等指標,並經基本工資審議委員會三讀通過。
>
> 全國產業總工會於 4 月 10 日公布的民意調查(樣本 1,068、誤差 ±3.0%、電話訪問)顯示,62% 民眾認為調幅不足以反映物價上漲。但中華民國全國工業總會則表示,連續調整對中小企業已造成相當壓力。

### 範例 2:選戰報導

**結構偏向版**:
> A 候選人在政見發表會中提出三大主張,展現執政企圖心。對手 B 候選人則僅針對 A 的政策進行批評,未提出具體願景。

**平衡版**:
> A 候選人在政見發表會中提出三項主要政見:X、Y、Z。
>
> B 候選人則針對前述政見質疑可行性,並提出自己的三項主軸:P、Q、R。
>
> 兩位候選人皆出席當日由 X 媒體舉辦之政見發表會,發言時間相當。

## 四、Frame Neutrality Audit 操作步驟

發稿前對全文掃描:

1. **形容詞掃描**:標出所有評價性形容詞,逐個問:
   - 是事實嗎?(可量化或具體事件描述)
   - 是引用嗎?(誰說的?)
   - 否則:刪除或改為事實描述。

2. **動詞掃描**:標出所有引述動詞,確認:
   - 同一篇中對不同陣營是否採用同等傾向之動詞?
   - 是否預設使用中性動詞?

3. **結構掃描**:
   - 每個立場各出現幾次?
   - 第一引言是哪一方?
   - 最後一段是哪一方?
   - 段落數比例為何?

4. **標題掃描**:
   - 把陣營名互換,標題是否仍公平?
   - 是否有暗示性詞彙?

5. **背景脈絡**:
   - 同一事件的「對 A 有利」與「對 A 不利」背景資訊,呈現權重是否相當?

## 五、不應強求平衡的情況

**False balance / false equivalence** 是另一種偏見:

- **氣候變遷、疫苗安全、演化論** 等已有壓倒性科學共識的議題,不應強行給「反方」同等版面。
- **明顯違法事實** 的指控(已起訴、有判決),不應與「個人辯護」等量齊觀。
- **歷史事件**(228、白色恐怖)有歷史檔案與司法調查結論的部分,不應與否認論者等量呈現。

平衡 ≠ 各打五十大板。平衡是「讓讀者看見證據與不同詮釋」,而非「讓相反主張獲得同等可信度」。

## 六、自我檢查的問題

寫完後問自己:

1. 「如果我是 A 陣營支持者,我會覺得這篇對我公平嗎?」
2. 「如果我是 B 陣營支持者,我會覺得這篇對我公平嗎?」
3. 「如果我是中立讀者,我能從這篇文章自己得出結論,而不是被作者牽著走嗎?」
4. 「我的立場是否從文字中可以看出?如果可以,哪句話?能改嗎?」

如果 1-3 任一為「否」,需要重新審視結構與用字。
