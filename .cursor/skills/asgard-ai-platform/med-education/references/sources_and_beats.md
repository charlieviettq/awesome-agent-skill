# Education Sources & Beats — Where to Find Things

## 教育主管機關(Tier 1: Public Record)

| 來源 | 用途 | URL pattern |
|------|------|-------------|
| 教育部全球資訊網 | 政策公告、新聞稿、統計資料、法令修正 | edu.tw |
| 國家教育研究院 | 教育研究報告、課程綱要詮釋、教學資源 | neri.org.tw |
| 聯合招生委員會中心 | 升學制度、招生數據、榜單統計、申請指引 | uac.edu.tw (或各區聯招) |
| 主計總處 | 教育相關統計、教育預算、決算資料 | dgbas.gov.tw |
| 各縣市政府教育局 | 地方教育政策、學校統計、特色課程 | 各縣市網址(e.g., 台北市:education.taipei.gov.tw) |
| 國際教育評估機制 | PISA(經濟合作暨發展組織)、TIMSS(國際數學科學教育成就評比)、PIRLS(國際閱讀素養進展研究) | oecd.org、iea.nl |

## 教育新聞典型採訪 Beat 配置

| Beat | 主要場域 | 必訪對象 |
|------|----------|---------|
| **政策線** | 教育部、國教院 | 部長、政策司長、研究員 |
| **大學線** | 各大學校長、教務長、國際事務處 | 招生政策、教學改革、國際合作 |
| **中小學線** | 各校校長、教學主任、教師會會長 | 學習成果、課程執行、教師待遇 |
| **升學線** | 聯招中心、各大學招生組 | 招生人數、錄取率、申請制度異動 |
| **教育研究** | 各大學教育研究中心、國教院 | 教學效果研究、政策評估報告 |
| **家長焦慮** | 家長團體、補習業者、教育部家長溝通窗口 | 家長需求、升學壓力、課外負擔 |

## 何時應該打給多少人

| 報導類型 | 最少訪問數 | 備註 |
|---------|----------|------|
| 即時短訊(< 300字) | 1 名(教育部發言人或學校發言人) + 公開資料 | 政策公告 |
| 教育政策追蹤(800-1500字) | 推動方 + 反對方(工會或家長) + 中立專家(教育學者) (3+) | 平衡原則 |
| 教育研究新聞(1000-2000字) | 研究者 + 批判學者 + 政策相關人士 (3+) | 效果量必訪 |
| 升學制度異動 | 教育部 + 聯招中心 + 大學招生主管 + 學生/家長聲音 (4+) | 多層級視角 |
| 校園事件 | 學校發言人 + 受影響者/家長 + 教育專家評論 (3+) | 避免單一聲源 |

## 教育新聞常見誤讀與查證點

- **「升學率 90%」**: 確認分母是誰(報考人? 畢業生全數? 錄取人數?)與官方來源
- **「排名升 2 位」**: 查 PISA / TIMSS 國家排名誤差範圍(通常 ±3-5 位在統計雜訊內)
- **「研究發現 X 改善成績」**: 必查效果量與樣本代表性
- **「全校英文成績平均 85 分」**: 確認年級、試卷難度、是否含補習生
