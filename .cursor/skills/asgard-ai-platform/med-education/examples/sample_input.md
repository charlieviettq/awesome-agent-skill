## 素材 1: 國家教育研究院研究摘要(2024 年 3 月發表)

**標題**: 國小英語能力與升學相關性研究:自編教材與常規教材比較

**摘要節錄**:

```
本研究比較自編雙語補強教材與常規英文教材對台灣國小學生英語閱讀理解的影響。

樣本:640 名台灣中部四所公立國小五年級學生,其中實驗組 320 名使用自編教材 12 週,
對照組 320 名使用常規教材。樣本性別比約 1:1;家庭社經地位多為中產至上中產。

工具:英文閱讀理解測驗(TCLE Reading Comprehension Test),15 題單選題,滿分 100 分。
事前測平均分數:實驗組 M=52.3 (SD=14.1),對照組 M=51.8 (SD=14.5),未達統計差異(p=0.72)。

結果:事後測實驗組 M=67.8 (SD=12.2),對照組 M=60.5 (SD=13.8),
差異達統計顯著(t(638)=5.42, p<0.001)。
效果量(Cohen's d) = (67.8 - 60.5) / 13.0 = 0.56 (95% CI: [0.38, 0.74]),屬中等至大效果。

追蹤:三個月後追蹤測試(n=580,追蹤率 90.6%),實驗組 M=66.1,對照組 M=59.8,
效果量 d=0.48,表現大致維持。

限制:(1)樣本侷限於中部四校,社經地位偏中上,不代表全國;
(2)追蹤期短(3 個月),長期效果未知;
(3)學測升學相關性未納入本研究(需後續調查)。

發表期刊:《教育研究季刊》(peer-reviewed)
經費:國科會補助 (OR 教育部委託),無廠商贊助。
```

## 素材 2: 教育部新聞稿(2024 年 3 月 15 日)

```
教育部發布新聞稿:推廣雙語教育政策,強化學生國際競爭力

教育部今(15)日宣布推廣雙語教育政策進展,自 2030 年起,全國公立國小
將普及英文強化課程,並提供教材補助與師資培訓。

教育部指出,全球化時代下,英語能力為學生升學與就業關鍵競爭力。
本部已委託國家教育研究院進行相關研究(見《教育研究季刊》2024 年 3 月發表),
證實自編教材有助提升國小學生英文閱讀理解能力。

政策目標:
- 2025-2027 年試點階段:100 所公立國小試行
- 2028-2030 年推廣階段:全國 2,500+ 公立國小納入

預算與實施:
- 年度預算編列 5 億元(教材開發、師訓、評估)
- 家長參與為自願制,學校得依在地資源調整

教育部強調,本政策旨在培養學生第二語言素養,不排擠本土語言教學時數。
同時將加強弱勢校補助,縮小城鄉差距。

〔聯絡人與電話欄位〕
```

## 素材 3: 家長與升學數據片段

**家長焦慮訪談節錄**(匿名,取得同意):

「我小孩讀中部某國小,班級已經開始試行新的英文課程。我很糾結,一方面希望他英文能力強,怕未來升學時輸人;但另一方面,國中聯招(會考)也考英文,國小多上英文會不會反而造成他壓力?」

「之前看到教育部說『強化英文』,讀起來就像『以後不會英文就上不了好大學』。但我不知道這件事的證據有多強。」

**高中英文招生數據(虛構示範)**:

2023 年大學學科能力測驗英文科:
- 滿分(85-100):全國 8.2%
- 一級(75-84):全國 16.5%
- 二級(65-74):全國 24.1%
- 三級(54-64):全國 28.6%
- 四級(44-53):全國 17.2%
- 五級(0-43):全國 5.4%

頂大英文系、國際企業系等「英文加權」科系之錄取生:約 60-70% 為英文一級或滿分。
基層工作(物流、營造、農業等)對英文需求相對低,該類職業英文學測通常二級以上即足。

## 素材 4: 國內外升學焦慮背景(剪報片段)

「雙語國家目標」政策自 2018 年提出,目標 2030 年成為雙語國家。
近年家長對子女英文能力焦慮加劇,補習班英文課程人數年增 8-12%。

國際對比:日本、南韓國小英文起始年級分別為 3、5 年級;台灣 2001 年起在國小 3-6 年級納入英文。

## 寫作要求

- 字數:約 1,000 字
- 新聞型式:教育政策新聞 + 研究新聞混合
- 平衡性:呈現研究證據、政策推廣意圖、家長焦慮、以及證據的限制
- 關鍵點:明確標示效果量(d=0.56)、樣本特性(中部四校,中上社經)、追蹤期限、以及何時適合推廣、何時需謹慎
- 避免:
  - ❌「雙語教育成果斐然」(誇大,無具體效果量)
  - ❌「研究人員發現英文能力與升學成功高度相關」(升學相關性未納入本研究)
  - ❌「全國國小應立刻推廣」(證據侷限於中部四校)
  - ✅「一項針對中部 640 名五年級學生的 12 週研究,發現自編教材使英文閱讀成績提升約 0.56 個標準差,三個月追蹤維持效果。教育部據此推廣,但研究者呼籲需後續在多區域驗證。」
