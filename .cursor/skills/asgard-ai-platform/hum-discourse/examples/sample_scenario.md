# Example: 外送平台「夥伴關係」聲明的話語分析

## Scenario

用戶：我剛看到 FleetRun（台灣外送平台）針對勞動部調查所發出的聲明，感覺哪裡怪怪的，但說不上來。可以幫我分析一下這段話在「說」什麼，以及它在「做」什麼嗎？

**原文（FleetRun 官方聲明，2026-03-18）：**

> FleetRun 一直以來致力於為夥伴創造靈活、自主的工作機會。我們的平台讓超過 8 萬名夥伴能夠自由選擇接單時段、掌控自己的收入節奏。針對近期外界對勞動關係的討論，FleetRun 重申：夥伴是獨立的服務提供者，而非雇員。我們將持續強化對夥伴的支持措施，包括意外險補貼與線上培訓資源，以確保夥伴在追求彈性工作的同時，也能獲得妥善保障。我們期待與各方利害關係人共同打造更永續的平台經濟生態。

---

## Analysis

### 步驟 1：背景脈絡

- **生產者**：FleetRun 公關部門（被勞動部調查中）
- **受眾**：媒體、勞動部官員、投資人、一般大眾（外送員看不到這份稿）
- **目的（表面）**：回應勞動關係爭議
- **目的（潛藏）**：在法律判決前鞏固「承攬關係」的論述框架，規避雇主法律責任
- **類型**：企業危機聲明

---

### 步驟 2：詞彙分析

| 文中用語 | 可用的替代詞 | 詞彙選擇的效果 |
|----------|-------------|----------------|
| **夥伴** | 勞工、員工、騎手 | 抹除雇傭關係的法律意涵；「夥伴」暗示對等關係 |
| **靈活、自主** | 缺乏排班保障、無底薪 | 將結構性不穩定重新包裝為個人自由 |
| **掌控自己的收入節奏** | 收入不穩定、無法預期 | 用「掌控」遮蔽平台演算法對接單的實際控制 |
| **獨立的服務提供者** | 從屬勞工（勞基法定義） | 使用法律術語先行定性，搶佔詮釋框架 |
| **支持措施** | 勞工保障、法定義務 | 將義務轉化為恩賜；暗示是公司好意而非法律要求 |
| **利害關係人** | 外送員、勞工團體 | 稀釋外送員的主體地位，與投資人、政府並列為抽象群體 |
| **平台經濟生態** | 外包勞動模式 | 生態學比喻暗示自然、共生，淡化剝削結構 |

---

### 步驟 3：主體位置分析

- **施事者（Agent）**：FleetRun（「致力於」、「重申」、「將持續強化」）——公司是積極的行為主體
- **受事者（Patient）**：夥伴（「讓…能夠」、「獲得妥善保障」）——外送員是被賦予機會的被動接受者
- **缺席者**：
  - 具體的受傷或收入受損外送員（無個案、無數字）
  - 勞動部的調查發現
  - 反對平台勞動模式的勞工團體聲音

---

### 步驟 4：預設分析（未經陳述的前提）

1. **「靈活」本身是好的**：聲明預設彈性工作是外送員共同追求的目標，未承認部分外送員希望穩定雇傭關係。
2. **夥伴關係是雙方自願選擇的結果**：預設外送員有足夠的替代選項，而非因缺乏其他工作機會而不得不接受此模式。
3. **現行法律框架可能不適用**：「針對近期…討論」這種措辭預設這只是「討論」，而非可能違法的事實。

---

### 步驟 5：互文性

- 援引「平台經濟」話語（源自矽谷科技公司，如 Uber、Airbnb 的公關語言）
- 「靈活工作」呼應後疫情時代「斜槓」、「自由工作者」的正面文化想像
- 「生態系」比喻常見於企業永續報告，暗示公司是環境中自然存在的物種，非人為設計的剝削結構

---

## Result

# Discourse Analysis: FleetRun 勞動關係聲明（2026-03-18）

## Context
- **Producer**: FleetRun 公關部門（正接受勞動部調查）
- **Audience**: 媒體、監管機關、投資人
- **Purpose（表面）**: 回應勞動爭議
- **Purpose（潛藏）**: 鞏固承攬關係論述、規避雇主法律責任
- **Genre**: 企業危機聲明

## Framing Analysis

| Element | 文中語言 | 替代框架 | 效果 |
|---------|---------|---------|------|
| 工作性質 | "靈活、自主的工作機會" | "缺乏保障的零工工作" | 將結構性不穩定美化為個人自由 |
| 勞動關係 | "夥伴是獨立的服務提供者" | "從屬勞工" | 用法律術語預先定性，搶占話語主導權 |
| 公司行為 | "支持措施" | "法定義務" | 恩賜框架：將義務包裝成好意 |
| 問題性質 | "近期外界的討論" | "勞動部調查" | 降低事件嚴重性，模糊官方調查的法律性質 |

## Subject Positions

- **施事者（主動）**: FleetRun——「致力於」、「強化」、「期待」
- **受事者（被動）**: 夥伴——被「讓」他們選擇、被「保障」
- **缺席者**: 具體的受傷或欠薪外送員、勞工工會、勞動部調查具體發現

## Presuppositions

- 彈性工作是外送員普遍偏好的（而非部分人被迫接受的）
- 8 萬名「夥伴」都認同並接受現行的合作模式
- 現行爭議只是「討論」而非潛在違法事實

## Power Analysis

- **此論述框架有利於**：FleetRun（規避雇主責任）、投資人（維持低人力成本的商業模式）
- **此論述框架不利於**：外送員（固化其「非雇員」身份，使集體協商、職災保障、勞健保請求更為困難）

## Key Findings

表面讀法：「FleetRun 表示自己重視外送員，提供靈活工作並給予保障。」

話語分析的發現：

1. **「夥伴」替換「勞工」是核心戰略**：全篇迴避「員工」、「勞工」等詞，是刻意的法律防禦姿態，而非描述性語言。
2. **能動性的反轉**：外送員的自主性（「自由選擇」）被強調，但演算法對接單率、排名的控制機制在文中完全不存在。
3. **調查事實的消音**：勞動部調查被稀釋為「外界討論」，壓制了調查可能指向違法的論述空間。
4. **意外險補貼作為替代品**：用「支持措施」取代「法定職災保障」，在維持成本優勢的同時製造出「已有保障」的形象。

> **結論**：這份聲明的主要功能不是溝通，而是在法律判決前對「勞動關係」的定義進行話語佔領。
