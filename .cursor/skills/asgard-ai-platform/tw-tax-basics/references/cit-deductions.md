# 營利事業所得稅扣除項目 (CIT Deductions)

## 應稅所得計算公式

```
應稅所得 = 收入 - 銷貨成本 - 營業費用 - 非營業損失 + 非營業收益 ± 稅務調整項目

應納稅額 = 應稅所得 × 20%

特殊門檻：
  ≤ NT$120,000          → 免稅
  NT$120,001–500,000   → (應稅所得 - NT$120,000) × ½ × 20%
  > NT$500,000          → 應稅所得 × 20%
```

---

## 扣除項目分類

### 1. 薪資與人事費用

| 項目 | 可扣除條件 | 上限 |
|------|-----------|------|
| 員工薪資 | 實際支付、有薪資明細 | 無上限 |
| 勞健保保費（雇主負擔） | 依法定費率計算 | 無上限 |
| 員工退休金（勞退提撥） | 6% 薪資提撥上限 | 依勞退條例 |
| 員工教育訓練費用 | 有發票、與業務相關 | 無特別上限 |
| 董監事酬勞 | 章程規定範圍內 | 依章程 |

> **注意**：給付個人薪資超過每月一定金額需辦理扣繳，但雇主端薪資費用本身仍全額可扣除。

---

### 2. 租金與辦公費用

| 項目 | 文件要求 | 注意事項 |
|------|---------|---------|
| 辦公室租金 | 租約 + 統一發票或收據 | 房東若為個人，需為其扣繳 10%；但扣除額仍為全額租金 |
| 水電費 | 統一發票 | 若辦公室兼住家，按使用比例認列 |
| 電話、網路費 | 統一發票 | 混合個人/公司用途需分攤 |
| 辦公耗材 | 統一發票 | 超過 NT$80,000 之設備需列固定資產，不得一次費用化 |

---

### 3. 固定資產折舊

依《固定資產耐用年數表》（財政部公告）攤提，採直線法為主。

```
年折舊額 = (取得成本 - 殘值) ÷ 耐用年數

殘值 = 取得成本 × 10%（一般預設）
```

**常見耐用年數**：

| 資產類別 | 耐用年數 |
|---------|---------|
| 電腦設備 | 3 年 |
| 辦公傢俱 | 5 年 |
| 交通工具（轎車） | 5 年 |
| 機械設備 | 依行業別 5–15 年 |
| 建物改良 | 依結構 10–50 年 |

**範例**：
- 購入筆記型電腦 NT$60,000，耐用年數 3 年，殘值 10%
- 年折舊 = (60,000 − 6,000) ÷ 3 = **NT$18,000/年**
- 三年可扣除合計：NT$54,000

> 若資產取得成本 ≤ NT$80,000，可選擇當年度一次費用化，不需逐年折舊。

---

### 4. 研究發展費用加計扣除（產創條例）

這是新創公司最重要的稅務優惠，**可扣除金額超過實際支出**。

#### 適用條款

依《產業創新條例》第 10 條：

```
選項 A（當年度）：研發費用 × 15% → 直接抵減應納稅額（稅額抵減）
選項 B（三年分攤）：研發費用 × 10% → 三年平均抵減稅額
```

**注意**：這是「稅額抵減」（tax credit），不是「費用加計」。計算時分兩層：

```
Step 1：研發費用正常列入費用扣除（已減少應稅所得）
Step 2：再另外抵減稅額（從算出的稅額中扣掉）

實際效益 = 研發費用 × 20%（費用扣除效果）+ 研發費用 × 15%（稅額抵減）
         = 研發費用 × 35%（綜合節稅效果）
```

#### 可列為研發費用的項目

| 項目 | 可計入 | 不可計入 |
|------|--------|---------|
| 研發人員薪資 | ✓ 全額 | 非研發職能人員 |
| 研發設備折舊 | ✓ 研發用途比例 | 行政/業務設備 |
| 研發材料消耗 | ✓ 有紀錄者 | 量產用材料 |
| 委外研究費用 | ✓ 合約明確 | 一般顧問費 |
| 專利申請費 | ✓ | 維護費（另案處理） |
| 雲端運算費（研發用） | ✓ 需能區分 | 全公司 IT 費 |

#### 申請要件

1. 設立**研究發展費用專帳**，逐筆記錄
2. 每年申報時附「研究發展活動費用專案報告」
3. 若研發費用超過 NT$1,000 萬，需請工業局或科技部預先認定
4. 保留研發日誌、工作紀錄、版本控制紀錄等佐證文件

---

### 5. 呆帳損失（壞帳）

```
可認列條件（需符合其一）：
  a. 債務人宣告破產或清算，有法院文件
  b. 債權達一年以上且有訴訟紀錄
  c. 小額債權（≤ NT$80,000）逾期兩年，有催收紀錄

可認列金額 = 實際呆帳金額（需沖銷帳面應收帳款）
```

無法提供催收紀錄或法院文件，**不得認列**。提列「備抵呆帳」不等於稅務可扣除。

---

### 6. 交際費（Entertainment Expenses）

有明定上限，依當年度淨收入計算：

```
交際費扣除上限：
  製造業：   淨收入 × 0.15%
  買賣業：   淨收入 × 0.10%
  服務業：   淨收入 × 0.20%
  其他行業： 淨收入 × 0.15%

超過上限部分 → 稅務調整（加回應稅所得）
```

**文件**：每筆需有統一發票、用餐人員名單、業務目的。

---

### 7. 廣告費

原則上全額可扣除（有統一發票且與業務相關），無上限。

但以下狀況需留意：
- 預付廣告費（如買斷廣告版位跨年）→ 需按效益期間分攤
- 贈品/試用品費用 → 計入廣告費但需有明確促銷活動紀錄

---

### 8. 保險費

| 保險類型 | 可扣除 | 說明 |
|---------|--------|------|
| 員工團體保險 | ✓ | 以員工為被保人且受益人為員工/家屬 |
| 公司財產保險 | ✓ | 辦公設備、存貨等 |
| 董監事責任險 | ✓ | 以公司為被保人 |
| 負責人個人壽險 | ✗ | 以個人為受益人者不可扣除 |
| 員工壽險（公司受益） | 部分可 | 需視保單結構，建議事先諮詢 |

---

## 稅務調整項目（加回應稅所得）

帳面上的費用，稅務上**不得扣除**，申報時需加回：

| 調整項目 | 說明 |
|---------|------|
| 罰款、滯納金 | 稅務罰鍰、交通罰單等 |
| 股東個人費用 | 誤列為公司費用的個人消費 |
| 超限交際費 | 超過淨收入比例的部分 |
| 未合格憑證費用 | 無統一發票、無收據的費用 |
| 非業務相關費用 | 與本業無關的支出 |
| 捐贈超限部分 | 對非公益團體或超過所得額 10% 的捐贈 |

---

## 捐贈費用特殊規則

```
對政府機關/公立學校捐贈：全額扣除
對合法公益團體（有免稅資格）捐贈：≤ 所得額 10%
對政黨捐贈：≤ 所得額 10%，且上限 NT$500,000
對私立學校捐贈：≤ 所得額 25%（依教育捐贈獎勵辦法）
```

## 完整試算範例

**情境**：一家台灣 B2B SaaS 新創，2025 年度數字如下：

```
收入：
  軟體授權收入       NT$8,000,000
  顧問服務收入       NT$1,500,000
  利息收益（非營業）    NT$20,000
  合計              NT$9,520,000

費用（帳面）：
  薪資（含勞健退）    NT$4,800,000
  辦公室租金         NT$600,000
  設備折舊（電腦×10台）NT$180,000
  研發費用（工程師薪資+雲端）NT$1,200,000
  廣告費             NT$300,000
  交際費             NT$50,000
  AWS費用（非研發）   NT$120,000
  無統一發票支出      NT$80,000
  合計              NT$7,330,000
```

**Step 1：帳面利潤**
```
8,000,000 + 1,500,000 + 20,000 - 7,330,000 = NT$2,190,000
```

**Step 2：稅務調整（加回不可扣除項目）**
```
交際費上限檢查：
  服務業 → 淨收入 NT$9,500,000 × 0.20% = NT$19,000
  帳列 NT$50,000 > NT$19,000
  加回：NT$50,000 - NT$19,000 = NT$31,000

無憑證費用加回：NT$80,000

稅務調整合計：+NT$111,000
```

**Step 3：應稅所得**
```
應稅所得 = NT$2,190,000 + NT$111,000 = NT$2,301,000
```

**Step 4：基本稅額**
```
> NT$500,000 → 全額 20%
應納稅額 = NT$2,301,000 × 20% = NT$460,200
```

**Step 5：產創條例研發稅額抵減**
```
研發費用 NT$1,200,000 × 15% = NT$180,000 稅額抵減

實際應納稅額 = NT$460,200 - NT$180,000 = NT$280,200
```

**Step 6：有效稅率**
```
NT$280,200 ÷ NT$9,500,000（收入）= 約 2.95%
（相較於未申請研發抵減的 4.84%，節省約 NT$180,000）
```

## 憑證要求速查

| 費用類型 | 最低文件要求 |
|---------|------------|
| 國內採購（有開立統一發票者） | 統一發票 |
| 個人房東租金 | 租約 + 收據（公司已扣繳 10%） |
| 海外軟體訂閱（如 AWS、Figma） | 信用卡明細 + 收據（英文可） + 扣繳申報紀錄 |
| 員工薪資 | 薪資條、轉帳紀錄、勞健保繳費證明 |
| 研發費用 | 專帳 + 工作日誌 + 佐證文件 |
| 折舊 | 資產購買發票 + 固定資產台帳 |

> **關鍵原則**：沒有統一發票的支出，稅務上仍可能認列（如個人房東、海外費用），但需要替代憑證並完成相關扣繳義務。完全無憑證者一律加回。

## 暫繳稅款（9 月）與年度申報的關係

```
暫繳金額 = 上一年度應納稅額 × 50%
申報期間：9 月 1–30 日

年度申報（5 月）：
  應納稅額（重新計算）- 暫繳金額 - 研發抵減等 = 補繳或退稅
```

暫繳是預付，不影響全年可扣除項目的計算，但**現金流規劃**需納入。
