# 台灣法律研究資源

## 一手法源（Primary Sources）

### 全國法規資料庫
**網址**: laws.moj.gov.tw  
**主管機關**: 法務部  
**用途**: 查詢現行有效法律條文、歷史修正版本、立法理由

搜尋技巧：
- 以「條文」搜尋時，輸入「第12條」找不到，要輸入「12」或法律名稱
- 選「條文內容」欄位搜尋比「法規名稱」搜尋範圍廣
- 每條條文頁面底部有「立法理由」連結——立法理由是IRAC中Rule陳述的重要補充

**IRAC對應**：Rule階段首選。引用格式：「《勞動基準法》第12條第1項第4款」

---

### 司法院法學資料檢索系統
**網址**: jirs.judicial.gov.tw  
**用途**: 判決書、解釋文、決議全文搜尋

資料庫分類：
| 資料庫 | 內容 | IRAC用途 |
|--------|------|---------|
| 最高法院判決 | 終審判決 | Rule + Application 的解釋標準 |
| 高等法院判決 | 二審判決 | Application 類比事實用 |
| 行政法院判決 | 行政爭訟 | 行政法議題的Application |
| 司法院解釋 | 大法官解釋 | 憲法層次的Rule |
| 憲法法庭判決 | 2022年後新制 | 取代部分大法官解釋功能 |

搜尋策略（以勞動法為例）：
```
1. 進入「裁判書查詢」
2. 法院類別：選「最高法院」
3. 案由：「終止勞動契約」
4. 裁判日期：設定近5年（避免已廢棄見解）
5. 全文關鍵字：「比例原則」（若要找比例原則的適用論理）
```

**網址**: lawbank.com.tw  
**性質**: 付費服務（各大學圖書館多有授權）  
**特色**: 法律見解整理、實務見解彙整，比司法院系統更易使用

免費功能：

- 法律問題討論（各廳研討結果）

付費功能（學校帳號可用）：
- 「法律問題」資料庫：各法院提請最高法院或最高行政法院討論的法律問題及決議
- 「實務見解」彙整：同一法條下的歷年判決見解整理

**IRAC對應**：Application階段找類比判決最快速

## 行政解釋與函釋

### 公文查詢
- **主計處函釋**: dgbas.gov.tw
- **勞動部函釋**: mol.gov.tw → 政策法規 → 法規查詢
- **金管會函釋**: fsc.gov.tw → 法規資訊

函釋在IRAC中的地位：
- 函釋是行政機關對自身主管法規的解釋，有行政拘束力但無司法拘束力
- Rule陳述時應標注「主管機關解釋認為…」而非「法律規定…」
- 法院可不採行政函釋見解——Application階段需說明函釋與判決的衝突點

## 法院判決引用格式

### 標準引用格式
```
最高法院 114 年度台上字第 OOO 號民事判決
臺灣高等法院 113 年度勞上字第 OOO 號判決
臺灣臺北地方法院 112 年度勞訴字第 OOO 號判決
```

格式拆解：
- `114 年度` = 中華民國114年（2025年）
- `台上字` = 最高法院上訴案件
- `勞上字` = 高等法院勞動上訴案件
- `勞訴字` = 地方法院勞動訴訟案件

### 大法官解釋引用
```
司法院大法官解釋第 XXX 號（釋字第XXX號）
憲法法庭 111 年憲判字第 XX 號判決（2022年後）
```

**IRAC注意**：2022年憲法訴訟法施行後，新案件為「憲法法庭判決」；2022年前為「大法官解釋」。二者並存，勿混淆。

## 判決可信度層級

在Application階段引用判決時，應注意判決的拘束力層級：

```
憲法法庭判決 / 大法官解釋
        ↓ 最高拘束力，所有法院均應遵守
最高法院判例（廢止前）/ 決議
        ↓ 實務上具高度參考效力
最高法院近年判決（多數見解）
        ↓ 下級法院通常遵循但不必然
高等法院判決
        ↓ 同院參考，他院不受拘束
地方法院判決
        ↓ 最低，個案拘束力
```

**2019年重要變革**：最高法院廢止「判例」制度。過去有拘束力的「判例」改為「裁判要旨」，效力降低。查詢時注意判決是否標注「判例」字樣——若是，確認該判例是否已被廢止。

廢止判例查詢：jirs.judicial.gov.tw → 判例查詢 → 廢止判例

## 快速查找流程（IRAC實作）

### Step 1：確認Issue適用的法律領域
```
民事 → 民法、特別民事法（消保法、公司法等）
刑事 → 刑法、特別刑事法
行政 → 行政程序法、各目的事業主管法規
憲法 → 憲法本文、憲法訴訟法、大法官解釋
```

### Step 2：找Rule（法條）
1. 全國法規資料庫 → 確認現行條文
2. 同頁「立法理由」→ 了解立法目的（法律解釋的依據）
3. 若條文有「經主管機關認定」等授權語→ 找行政機關函釋

### Step 3：找Application素材（判決）
1. 司法院系統 → 最高法院判決 → 全文關鍵字搜尋法條號碼
2. 篩選近5年判決（除非要找早期見解演變）
3. 找判決摘要中與你的事實最接近的案件
4. 確認該見解是否為多數見解（可搜尋相同關鍵字看有幾件類似判決）

### Step 4：確認見解是否有爭議
1. 若找到相反見解的判決→ Application中必須交代兩種見解的存在
2. 若學說有爭議→ 月旦法學知識庫找近年評析
3. 若大法官有解釋→ Rule中直接引用，不需討論下級法院分歧
