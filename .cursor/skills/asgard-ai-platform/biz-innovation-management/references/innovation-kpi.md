# 創新 KPI、組織設計與激勵方案筆記

## 創新 KPI 設計的挑戰

### 為何創新 KPI 難設計

1. **產出延遲**：投資到回報可能 3–10 年
2. **高不確定性**：80%+ 失敗率
3. **多層面**：營收、品牌、人才、文化
4. **本業邏輯干擾**：短期 ROI 思維扼殺長期投資

## KPI 四大類型

### 1. 投入指標（Input）

**研發投入**
- R&D 投入 / 營收（R&D intensity）
- 國際對照（業界平均）

**創新預算**
- H1 / H2 / H3 配置比例
- 絕對金額

**人才投入**
- 創新團隊規模
- 創新人員 / 總員工

**外部合作投入**
- 合作夥伴數
- 合作案金額

**風險**：只看投入容易變成「燒錢不產出」。

### 2. 過程指標（Process）

**Stage-Gate 相關**
- 各階段案件數
- 通過率（Kill Rate）
- 階段平均天數

**Time-to-Market**
- 從 Idea 到 Launch 的平均時間
- 行業對照

**創新活動**
- 創新 workshop 次數
- 提案數
- 外部掃描件數

**專案健康度**
- 預算達成率
- 里程碑達成率

### 3. 產出指標（Output）

**新產品 / 服務收入**
- Vitality Index：過去 3 年推出產品的營收占比
- 業界標竿：消費品 30%、科技 40–60%

**專利**
- 申請 / 核准件數
- 引用次數（品質）
- 商業化率

**新市場 / 新客戶**
- 進入新市場數
- 新客戶數、新客群

**CVC 組合**
- 被投資市值變化
- 退出 / 併購案件
- 策略訊號數

### 4. 學習指標（Learning）

**Kill Rate**
- 各 Gate Kill 率
- 平均 Kill 所需時間（愈短愈好）

**Pivot 次數**
- 專案平均 Pivot 次數
- 每 Pivot 學習紀錄

**失敗分析報告**
- 每失敗案撰寫 Lessons Learned
- 次年參考

**知識共享**
- 內部創新論壇次數
- 跨事業部知識轉移案例

## Vitality Index 詳解

### 定義
「**當期營收中，來自過去 N 年新推出產品的比例**」。

N 通常為 3 或 5 年。

### 計算公式
```
Vitality = （當期新產品營收）/（當期總營收）
```

**新產品定義**：
- 過去 N 年內首次上市
- 與既有產品差異顯著（非小改版）

### 業界標竿

| 產業 | 建議 Vitality Index |
|---|---|
| 消費品（食品、日用） | 20–30% |
| 消費品（電子、服飾） | 40–60% |
| 工業品 | 20–40% |
| 製藥 | 依專利過期 |
| 軟體 / 服務 | 50–70% |
| 金融業 | 15–25% |

### 陷阱

- **小改版被當新產品**：需清晰定義
- **大幅延長 N**：N=10 年會誤導
- **忽略組合平衡**：Vitality 高但都是 Core 小創新

## 組織設計選項

### 選項 1：集中式（Centralized Innovation）

**結構**：
- 獨立創新處 / 創新事業群
- 統一創新長管理
- 涵蓋 H2 / H3

**優**：
- 專業化、聚焦
- 資源集中
- 避免事業部干擾

**劣**：
- 與事業部整合困難
- 可能脫離市場

**適合**：多事業部、需要集中突破

### 選項 2：分散式（Decentralized）

**結構**：
- 各事業部自己做創新
- 集團層級設「創新長」協調

**優**：
- 貼近市場
- 事業主管有當責

**劣**：
- 資源重複
- 難以協調、出現組合失衡

**適合**：多元化集團、事業間協同少

### 選項 3：雙軌制（Dual Structure）

**結構**：
- H1：既有組織
- H2 / H3：獨立事業群 / 子公司

**優**：
- 保護新事業免於本業扼殺
- 各自最佳化

**劣**：
- 整合時衝突
- 兩套 KPI、文化

**適合**：大型企業、需保護新事業

### 選項 4：混合式（Hybrid）

**結構**：
- 本業 + CVC + 加速器 + 獨立事業群

**優**：
- 多軸並進
- 靈活

**劣**：
- 治理複雜
- 人才需求高

**適合**：大型集團、創新戰略多元

### 選擇原則

- 公司規模
- 事業多元度
- 變革速度
- 資源能量
- 人才供給

## 創新長職位設計

### 創新長的三大職責

1. **戰略**：
   - 創新組合設計
   - 資源配置建議
   - 與 CEO / 董事會對話

2. **治理**：
   - Stage-Gate 主持
   - 組合審議會議
   - KPI 設計

3. **整合**：
   - 跨事業部協調
   - 外部夥伴洽談
   - 文化塑造

### 職位層級

- **集團 Level**：副總經理、執行副總經理
- **直接向 CEO 報告**
- **薪酬同 CFO / COO 等級**

### 共同陷阱

- **無預算**：只有位置、無錢
- **無授權**：事業部不甩
- **KPI 模糊**：無法衡量績效
- **任期過短**：創新需 3–5 年見成果

## 創新人才激勵方案

### 顯性激勵

**現金獎勵**
- 里程碑獎金
- 專利獎金
- 成功商業化分紅

**股票 / 選擇權**
- 一般員工股票選擇權
- 新事業股權（Phantom Equity）
- 創新事業 earn-out

**職涯加速**
- 創新成果 → 晉升快軌
- 跨部門輪調

### 隱性激勵

**認可**
- 年度創新獎
- 公開表揚
- CEO / 董事會的關注

**自主**
- 20% Time（Google）
- 年假「創新週」
- 自由選專案

**心理安全**
- 失敗不懲罰
- 公開失敗分享

**成長機會**
- 創新訓練
- 國際會議
- 外部顧問身份

### 台灣企業的激勵設計難題

**薪資結構僵化**
- 難給創新人才頂級薪資
- 補救：股權、獎金、其他福利

**股權制度**
- 上市公司 RSU 普遍
- 非上市需「虛擬股權」

**創新失敗的「隱形懲罰」**
- 雖然嘴上說允許、實際邊緣化
- 修正：顯性保護機制（輪調、保留職位）

## 創新文化塑造

### 文化指標

**公開對話頻率**
- CEO 公開談創新的次數
- 創新故事的傳播

**失敗的處理**
- 失敗者的後續（升 / 平 / 降 / 離職）
- 失敗案例是否公開分享

**內部挑戰**
- 員工敢不敢質疑決策
- 跨部門爭議的處理

**外部連結**
- 員工參與外部活動的支持度
- 與新創、學研的互動

### 文化轉型的信號

- CEO 時間投入（實際花多少時間在創新？）
- 資源分配（從口號到錢）
- 人事決策（誰升官？誰被保護？）
- 空間設計（是否有創新實驗室？）

## 創新績效管理的五個常見錯誤

### 錯誤 1：本業 KPI 套用新事業
- ROI、Margin 套 H3 → 立刻 Kill
- 修正：差異化 KPI

### 錯誤 2：單一 KPI 依賴
- 只看專利件數 → 追數量、犧牲質量
- 修正：多面向 KPI 組合

### 錯誤 3：年度 KPI 評估創新
- 年度績效殺死長期投資
- 修正：里程碑 + 年度 + 滾動

### 錯誤 4：懲罰失敗
- 顯性失敗有 KPI 扣分
- 修正：設「學習獎」

### 錯誤 5：CEO 不介入
- 把創新下放給創新長 → 預算被 CFO 砍
- 修正：CEO 公開承諾保護
