# 繼任規劃框架與企業實踐筆記

## 繼任規劃（Succession Planning）的核心問題

> 「此人若明天不在，組織能否持續運作？」

繼任規劃回答此問題於三個時間尺度：
- **立即（0–6 個月）**：能否指派代理人？
- **短期（1 年內）**：能否培養出準備好的接班人？
- **中期（3–5 年）**：能否從組織內產出下一代領導？

## 基本框架

### 1. 關鍵職位識別（Critical Role Identification）
- 衝擊度 + 替代難度 矩陣
- 不超過「全部主管職」的 15%
- 每年重新校準（組織結構變化大者每半年）

### 2. 接班人候選池（Successor Pool）
- 每個關鍵職位至少 2–3 個候選人
- 層級：**Ready Now** / **Ready 1–2 Year** / **Ready 3–5 Year**
- 候選人不限內部，可含外部鎖定對象

### 3. 差距分析（Gap Analysis）
- 候選人 vs. 職位職能要求的落差
- 以**職能地圖**可視化

### 4. 發展行動（Development Action）
- 每位候選人對應 IDP
- 70-20-10 法則
- 年度 checkpoint

### 5. 治理回饋循環（Governance Loop）
- 董事會 / 家族委員會 / 執委會定期審查
- 候選人異動（升遷、離職）即時通報

## Charan Leadership Pipeline（領導發展管線）

**Charan et al. (2001)** 將領導發展分為 **7 個過渡**：

1. **管理自己 → 管理他人**
2. **管理他人 → 管理經理**
3. **管理經理 → 管理職能**
4. **管理職能 → 管理事業**
5. **管理事業 → 管理多事業**
6. **管理多事業 → 管理企業**
7. **管理企業 → 董事會角色**

**每一過渡需要三種改變**：
- **技能**（Skills）：新工作所需能力
- **時間運用**（Time Application）：做什麼事、花多久
- **價值觀**（Values）：看重什麼

### EMBA 應用
- 診斷主管「是否卡在過渡」：常見第 2、3、5 階段卡關
- 「提升職級但沒改變時間運用與價值觀」= 過渡失敗
- 台灣企業常見：創辦人從未完成「管理多事業 → 管理企業」（仍當自己是大事業總經理）

## Ram Charan 3 類型 CEO 繼任

### Type 1：Insider Successor（內部繼任）
- 優：文化連續、風險低
- 劣：可能延續既有問題、無法帶來變革
- 適用：穩定成長階段、文化健康

### Type 2：Outsider Successor（外部繼任）
- 優：新視角、變革動能
- 劣：文化整合、短期績效壓力
- 適用：需大幅變革、產業顛覆

### Type 3：Hybrid（混合繼任）
- 以外部 COO 培養 2–3 年後內升
- 結合兩者優勢
- 多數大型家族企業偏好此路徑

## Pipeline 建構的五類企業實踐

### GE「Talent Review System」（傳統金標）
- 年度 Session C 會議
- 頂層 500 人深度盤點
- Jack Welch 時期每月討論 1 位高潛
- **啟示**：CEO 親自投入是關鍵

### Unilever「Global People Committee」
- 每季度審視跨市場高潛
- 強制跨國輪調
- 多元性 KPI（女性、新興市場、年齡）

### P&G「Build from Within」
- 100+ 年傳統：幾乎全部高管內部培養
- CEO 候選通常 25+ 年司齡
- 優勢：文化深植；劣勢：變革緩慢

### Toyota「層層遞進」
- 強調現場經驗（Gemba）
- 每位高管必歷經工廠多年
- 接班透明度低（日本文化）

### 台積電「技術 + 管理雙軌」
- 技術軌最高可達 VP / CTO
- 管理軌 CEO / COO
- 繼任以內部培養為主，外部為輔

## 家族企業的繼任特殊框架

### Tagiuri & Davis 三圈模型
- 家族、所有權、經營
- 繼任規劃需三圈同步設計，不可只談經營

### Gersick et al.《Generation to Generation》四階段
1. **創辦期**：一人同時持有家族、所有權、經營
2. **兄弟姐妹合夥**：第二代分工
3. **堂表親聯盟**：第三代多元化
4. **家族王朝**：第四代以後制度治理

### 家族憲法（Family Constitution）典型內容
- 家族成員進入公司條件
- 公司 vs. 家族利益衝突處理
- 股權轉讓限制
- 分紅政策
- 家族衝突解決機制
- 慈善 / 公益承諾

### 台灣家族企業常見繼任路徑
1. **指定繼承**：父親指定長子（或某一子女）
2. **競爭繼承**：子女各自經營分公司，績效決定
3. **專業化**：家族退出經營、擔任股東 / 董事
4. **分家**：事業按子女人數切分
5. **混合**：部分事業家族經營、部分專業化

### 台灣家族企業繼任的特殊挑戰
- **長子迷思**：預設長子接班
- **女兒代際突破**：新世代女兒接班比例上升
- **入贅 / 女婿議題**：家族歸屬感與忠誠度
- **稅務與股權傳承**：贈與稅、遺產稅規劃
- **家族信託**：近年興起的治理工具

## 繼任規劃的常見陷阱

### 陷阱 1：「名單即承諾」
候選人知道自己在名單上 → 期待升遷 → 未升時失望離職。
**防範**：明確溝通「候選人≠保證」，定期更新期望管理。

### 陷阱 2：「名單政治」
主管把自己愛將放名單、異己排除。
**防範**：HR 校準、外部董事挑戰、候選人多元性 KPI。

### 陷阱 3：「培養過度延遲」
候選人 40 歲就鎖定但 50 歲還沒給位置。
**防範**：設定「最遲決議日期」；若未決議，接受候選人離職的風險。

### 陷阱 4：「接班前一刻才發現 gap」
以為候選人準備好了，上任後崩盤。
**防範**：在現職期間提供「stretch assignments」逐步驗證。

### 陷阱 5：「純內部思維」
只看內部，錯失市場機會。
**防範**：每個關鍵職位同時維護「外部鎖定名單」。

### 陷阱 6：「離職面談才知道流失」
明星離職面談才發現未被賞識。
**防範**：每季直接與關鍵人才 1-on-1（skip-level），CEO 應認識前 50 位關鍵人才。

## 董事會在繼任規劃的角色

### 上市公司（金管會要求）
- CEO 繼任計畫應定期呈報董事會
- 獨立董事應有機會單獨接觸候選人
- 2023 年起，部分上市公司被要求揭露繼任規劃概況

### 家族企業（非上市）
- 家族委員會 + 董事會雙軌
- 獨立董事（若有）提供中立視角
- 家族憲法應明定董事會對繼任的角色
