# Example: 研究台灣獨立遊戲開發者社群的創作文化

## Scenario

一位遊戲產業研究員 Maya Chen 任職於某獨立遊戲發行商 Pixel Harbor，公司計畫進入台灣獨立遊戲市場。Maya 需要了解台灣 indie 開發者社群（主要活躍於 Discord 伺服器「台灣獨立遊戲協作站」、Plurk 社群與 TGDF 實體活動）的創作動機、合作文化與對商業化的態度，以協助公司制定合作策略。她問：「我想用民族誌方法研究這個社群，但我只有四個月，夠嗎？怎麼規劃？」

---

## Analysis

### Step 1: Gain Access and Establish Rapport

**Field sites identified (multi-sited ethnography):**

| 場域 | 類型 | 入口策略 |
|------|------|----------|
| Discord「台灣獨立遊戲協作站」（約 1,200 名成員）| 線上主場域 | 以個人研究者身份申請加入，公開揭露研究目的 |
| Plurk #indiegame #gamedev 話題串 | 半公開線上場域 | 觀察公開貼文，對互動頻繁者私訊告知研究身份 |
| TGDF 2025（台灣遊戲開發者論壇，9 月） | 實體場域 | 購票參加，帶名片，不以發行商代表身份出席 |

**研究者角色定位：** Observer-as-participant — 公開研究者身份，積極參與討論但不主導，避免影響社群日常互動。

**關鍵動作（第 1-4 週）：**
- 在 Discord 的 `#自我介紹` 頻道發文，說明研究目的與使用方式
- 連續 2 週只潛水觀察，不發言（netnography 的 lurk 階段）
- 第 3 週起開始在 `#作品分享` 頻道提供真誠回饋，建立信任

**注意：** 四個月在民族誌尺度屬於短期，可達「密集觀察」層次，但無法捕捉跨年週期（如 Global Game Jam 到 TGDF 的完整生命週期）。研究者需明確界定研究聲稱的範圍限制。

---

### Step 2: Conduct Participant Observation

**田野筆記系統（每日記錄）：**

```
[描述層] 2025-06-14 22:30
Discord #作品回饋 頻道：開發者 @koalastudio 分享新作截圖，
標題「逃跑的鉛筆」，美術風格為手繪素描。15 分鐘內收到 23 則
回應，幾乎全為美術讚美，僅 2 則提及遊玩體驗。

[反思層]
「美術優先」的回饋模式反覆出現。是社群規範（不批評同伴的
遊玩性）？還是這個社群以視覺創作者為主？需要持續觀察。

[方法層]
下週起追蹤「同一個作品在不同頻道被討論時的語調差異」—
#作品回饋 vs #商業討論 vs 實體 TGDF 的非正式對話。
```

**觀察重點追蹤（6 月-9 月）：**

- Discord 訊息量：平均每日 80-120 則；高峰在 Global Game Jam 截止前（日均 340 則）
- 反覆出現的 emic 詞彙：「做自己想做的」、「不要為了市場妥協」、「肝」（通宵工作）、「養肝」（規劃可持續的開發節奏）
- 社群制裁事件：一名成員在 `#作品回饋` 發行銷連結，72 小時內被版主移除並警告，引發 30+ 則討論，多數成員支持版主決定

---

### Step 3: Produce Thick Description

**田野片段（TGDF 2025，展場走廊，2025-09-07）：**

> 攤位收攤後，五名開發者圍著一張摺疊桌喝手搖，討論某大型發行商當天的演講。Lin（28 歲，獨立開發三年）說：「他們講的 monetization，聽起來像在跟我們說怎麼賣靈魂。」其他人笑，但沒有反駁。Chen（化名，曾與一家發行商合作後解約）說：「簽之前要看 IP 誰的，這才是重點。」全桌點頭，這句話沒有被挑戰——它像是已被反覆驗證的格言，不需要辯論。
>
> **解釋：** 「賣靈魂」作為社群隱喻，不只是對商業化的反對，而是指涉 IP 控制權的喪失。創作自主性（creative autonomy）是這個社群最核心的價值符號；商業合作的可接受性以「IP 歸屬」為判斷閾值，而非收入金額。

---

### Step 4: Analyze and Write the Ethnographic Account

**核心文化主題：**

| Theme | Emic Term | Observed Practices | Interpretation |
|-------|-----------|-------------------|----------------|
| 創作神聖性 | 「做自己想做的」 | 拒絕市場需求導向的功能建議；讚美美術多於遊玩性 | 創作純粹性是社群聲望的主要貨幣 |
| 勞動英雄主義 | 「肝」 | 凌晨更新開發進度截圖；以通宵工作為榮 | 過勞被轉化為奉獻與熱情的證明 |
| 邊界維護 | （無單一詞彙） | 版主移除行銷貼文；正式場合與非正式場合的語言切換 | 社群積極維護非商業的「安全創作空間」 |
| IP 主權 | 「IP 誰的」 | 合作談判時優先確認 IP 歸屬；以此為接受或拒絕合作的首要條件 | 主權框架取代純利潤框架——賣出 IP = 失去創作身份 |

**社會結構：**
- 核心圈（10-15 人）：TGDF 常態參展者、Discord 版主——定義社群規範
- 半外圍（~200 人）：活躍參與者，跟隨規範但不制定
- 外圍（~1,000 人）：潛水者，觀察多於參與

---

## Result

```markdown
## Ethnographic Analysis: 台灣獨立遊戲開發者社群

### Field Context
- Setting: Discord「台灣獨立遊戲協作站」+ Plurk #indiegame + TGDF 2025 實體場域
- Duration: 2025年6月–9月（4個月密集觀察；非完整年度週期）
- Researcher role: Observer-as-participant，公開研究者身份
- Access strategy: Discord 公開揭露研究目的，TGDF 以個人研究者（非發行商）身份出席

### Cultural Themes
| Theme | Emic Term | Observed Practices | Interpretation |
|-------|-----------|-------------------|----------------|
| 創作神聖性 | 「做自己想做的」 | 抗拒市場導向建議；美術讚美優先 | 純粹性是聲望貨幣 |
| 勞動英雄主義 | 「肝」 | 凌晨更新進度；過勞敘事 | 過勞 = 奉獻的社群證明 |
| 邊界維護 | （無） | 版主移除行銷貼文，社群支持 | 積極保護非商業安全空間 |
| IP 主權 | 「IP 誰的」 | 合作談判首要條件 | 主權框架優先於利潤框架 |

### Thick Description Excerpt
> TGDF 收攤後的走廊對話顯示：「賣靈魂」不是對金錢的排斥，而是對 IP 控制權喪失的具體恐懼。「IP 誰的」已成為社群的判斷格言，在無需辯論的情況下被全體接受——這表明它已完成從個人意見到集體規範的轉化。

### Social Structure and Power
- Key actors and roles: Discord 版主（5人）+ TGDF 常態參展者（10-15人）定義規範；兩群體高度重疊
- Norms and sanctions: 行銷內容在創作頻道發布會被移除，且獲社群主動支持，顯示規範內化程度高
- Tensions and contradictions: 「肝」文化被部分成員反思為不可持續，但批評聲音在公開頻道中受到壓制（沉默螺旋）

### Researcher Reflexivity
- Positionality: 研究者任職發行商，即使未公開此身份於社群，仍影響哪些成員願意深度訪談（2 名受訪者事後得知後退出）
- Impact on setting: 研究者的出現使 `#商業討論` 頻道訊息量在第 3 週增加約 40%，部分成員可能有「表演給研究者看」的效應

### Implications for Pixel Harbor
1. **IP 條款是合作門檻，非談判籌碼**：任何未在第一次接觸時明確保障開發者 IP 所有權的提案，都會在社群核心圈形成負面口碑，且擴散速度極快
2. **直接接觸核心圈（版主/TGDF 常態參展者），不走大量撒網策略**：此社群聲望高度集中，核心圈的背書 > 任何廣告預算
3. **「養肝」論述有利可圖**：部分成員對過勞文化存在隱性反思，以「可持續開發支援」為合作定位，可與主流「賣靈魂」敘事形成差異
```

**對 Maya 的方法回應：**

四個月足以完成「密集觀察」等級的 netnography + 有限實體田野，但研究聲稱必須明確界定：你描述的是 2025 年夏秋季的社群快照，而非跨年週期的文化動態。論文或報告中需標注此限制，並建議以「初探性民族誌」（exploratory ethnography）定位，而非完整民族誌研究。
