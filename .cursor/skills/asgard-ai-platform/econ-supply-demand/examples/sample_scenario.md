# Example: 台灣珍珠奶茶市場 — 砂糖進口關稅調漲的衝擊

## Scenario

用戶是一位在台灣經營連鎖手搖飲料品牌「珍珠時代」（30 家門市）的營運總監。2026 年 3 月，政府宣布對進口砂糖課徵每公斤 NT$8 的進口關稅（現行稅率為 NT$2），將於 2026 年 7 月 1 日生效。

用戶的問題：
> 「砂糖關稅從 NT$2 漲到 NT$8，我一杯珍奶要漲多少？消費者會減少購買嗎？我該怎麼預測接下來的市場變化？」

背景數字：
- 目前市場均衡價格：NT$65 / 杯（500ml 中杯）
- 目前市場月交易量：約 4,000 萬杯（台灣全市場）
- 每杯珍奶使用砂糖約 20 克
- 台灣手搖飲料約 60% 砂糖來自進口

---

## Analysis

### Step 1：定義市場

- **財貨**：台灣現做手搖珍珠奶茶（中杯，500ml）
- **範疇**：台灣全國市場，2026 年第三季起

---

### Step 2：識別衝擊

- **事件**：進口砂糖關稅調漲 NT$6/kg（從 NT$2 → NT$8）
- **受衝擊的曲線**：**供給曲線**
  - 砂糖是手搖飲料的投入要素（input cost）
  - 投入成本上升 → 供給曲線左移（減少）
- **需求曲線**：不受影響（消費者所得、偏好、替代品價格均無變化）

---

### Step 3：計算供給曲線位移量（每杯成本衝擊）

```
每杯使用砂糖：20 克 = 0.02 kg
進口砂糖占比：60%
每杯受影響砂糖量：0.02 kg × 60% = 0.012 kg

關稅增加：NT$6 / kg
每杯成本增加：0.012 kg × NT$6 = NT$0.072 ≈ NT$0.07 / 杯
```

供給曲線向左位移量：**NT$0.07 / 杯**（廠商在每個數量水準上，願意接受的最低售價上漲 NT$0.07）

---

### Step 4：估算均衡變化（依彈性分攤稅負）

**需求彈性估算**（手搖飲料）：
- 珍奶屬非必需品，有替代品（咖啡、自備水、其他飲料）
- 估計需求價格彈性：`Ed ≈ -1.2`（略有彈性）

**供給彈性估算**（短期）：
- 現有門市設備固定，短期擴縮產量有限
- 估計供給價格彈性：`Es ≈ 0.6`（短期偏無彈性）

**稅負分攤公式**：
```
買方承擔比例 = Es / (Es + |Ed|) = 0.6 / (0.6 + 1.2) = 0.33
賣方承擔比例 = |Ed| / (Es + |Ed|) = 1.2 / (0.6 + 1.2) = 0.67
```

**結論**：
- 每杯 NT$0.07 的成本增加，廠商（賣方）吸收約 NT$0.047，轉嫁給消費者約 NT$0.023
- 實際售價預計上漲 **NT$0.02–0.03 / 杯**（極小幅度，不足以漲價一個整數級距）

> **⚠️ 關鍵洞察**：此衝擊遠小於業者通常的「最小調價單位」（NT$5）。短期內，「珍珠時代」**不會因為這次關稅單獨調漲售價**，而是等待成本累積到足夠門檻。

---

### Step 5：量的變化

```
價格上漲 NT$0.023 → 漲幅 = 0.023 / 65 = 0.035%
需求量變化 = Ed × 漲幅 = -1.2 × 0.035% = -0.042%
市場月交易量減少：4,000 萬 × 0.042% ≈ 1.7 萬杯（全市場）
```

量的衝擊幾乎可以忽略。

---

## Result

# Supply-Demand Analysis: 台灣手搖珍奶市場（進口砂糖關稅調漲）

## Market Definition
- **Good**: 現做手搖珍珠奶茶（中杯 500ml）
- **Scope**: 台灣全國市場，2026 Q3

## Shock Identification
- **Event**: 進口砂糖關稅 NT$2→NT$8（+NT$6/kg），2026-07-01 生效
- **Affected curve**: 供給曲線
- **Direction**: 左移（供給減少）
- **Mechanism**: 投入成本上升，廠商在相同產量下需要更高售價才能維持獲利，供給曲線上移 NT$0.07/杯

## Equilibrium Impact
|  | Before | After | Change |
|---|--------|-------|--------|
| Price | NT$65.00 | NT$65.02 | ↑ NT$0.02 |
| Quantity (月) | 4,000 萬杯 | 3,998 萬杯 | ↓ 約 1.7 萬杯 |

## Analysis

此次關稅調漲對手搖飲料市場的直接影響**極為有限**：

1. **每杯成本衝擊僅 NT$0.07**：砂糖僅占每杯原料的一小部分，且進口砂糖只佔 60%。關稅衝擊被稀釋後，遠低於業者的實際調價門檻（NT$5）。

2. **廠商吸收大部分成本**：因短期供給彈性（0.6）低於需求彈性（1.2），廠商承擔約 67% 的稅負，利潤空間略受壓縮，但不至於引發調價。

3. **長期效應需重新評估**：若同期疊加其他成本衝擊（原物料漲價、最低工資調升），累積效應可能觸發漲價門檻。長期供給彈性較高（廠商可調整配方、尋替代糖源），轉嫁比例可能下降。

4. **「珍珠時代」的建議**：
   - 短期：無需調整售價，監控砂糖用量佔比並評估替代甜味劑可行性
   - 中期：若同年度有其他成本壓力，可將關稅成本納入年度漲價幅度
   - 供應商談判：鎖定國內糖廠配額或提前簽訂遠期合約，規避進口糖價格波動

---

> **Gotcha 提醒**：用戶原始問題「關稅漲了我要漲多少？」隱含了「成本=售價漲幅」的錯誤假設。正確分析需要透過彈性分攤計算消費者實際承擔比例，而非直接把成本增加量全數轉嫁。
