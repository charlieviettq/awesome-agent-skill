# 董事會組成：能力矩陣、多元性、獨立性實務

## 董事會能力矩陣（Board Skills Matrix）

### 什麼是能力矩陣
以表格列出董事會「整體需要具備哪些能力」，並對應到個別董事，用於：
- 診斷現有董事會的能力 gap
- 指導新董事招募
- 向投資人揭露董事會的整體實力

### 台灣上市公司典型能力清單

| 能力領域 | 中文 | 目標席次 |
|---|---|---|
| Industry expertise | 產業專業 | 2–3 |
| Financial / accounting | 財務會計 | 2 |
| Legal / compliance | 法律法遵 | 1 |
| Risk management | 風險管理 | 1 |
| Digital / technology | 數位科技 | 1 |
| Cybersecurity | 資訊安全 | 1 |
| International / cross-border | 國際跨境 | 1 |
| ESG / sustainability | 永續發展 | 1 |
| HR / talent | 人才管理 | 0–1 |
| Marketing / branding | 行銷品牌 | 0–1 |

### 能力矩陣的填寫方式

| 董事 | 產業 | 財會 | 法律 | 風險 | 數位 | 資安 | 國際 | ESG |
|---|---|---|---|---|---|---|---|---|
| 董事 A |  ✓ |     |     |  ✓  |     |     |  ✓  |     |
| 董事 B |     |  ✓  |     |     |     |     |     |  ✓  |
| 獨董 C |     |  ✓  |     |     |     |     |     |     |
| 獨董 D |     |     |  ✓  |     |     |     |     |     |
| 獨董 E |     |     |     |     |  ✓  |  ✓  |     |     |

**診斷**：ESG 與國際僅 1 席、風險管理僅 1 席 → 屬中度 gap。

### 能力矩陣的常見陷阱
- **過度填寫**：每位董事都自評「什麼都懂」→ 失去診斷價值
- **未定期更新**：能力矩陣只在公開說明書出現、未年度審視
- **能力認定鬆散**：「有產業經驗」被濫用（退休 20 年是否仍算？）
- **忽略軟技能**：領導、溝通、衝突處理未被系統化評估

## 董事會多元性

### 多元性的維度
- **性別**
- **年齡 / 世代**
- **國籍 / 族裔**
- **職業背景**
- **地理 / 市場經驗**
- **思維風格 / 認知多元**

### 台灣性別多元相關法規
- 2022 起上市公司單一性別董事席次比例揭露強制
- 2026 起上市公司單一性別董事席次不得超過 2/3
- 2030 目標 30% 女性董事

### 性別多元的研究發現
- Catalyst 等研究顯示「性別多元與財務績效正相關」
- 但因果方向有爭議（可能是「好公司較容易吸引女性董事」）
- 多元性對決策品質的影響較穩健（Page, 2007）

### 年齡 / 世代多元
- 近年趨勢：引入 30–40 世代年輕董事
- 優：數位、ESG、消費者洞察
- 挑戰：資歷與經驗的平衡

### 多元性的正向機制（Erhardt et al. 2003）
1. 擴展資訊池
2. 減少「群體思考」（Groupthink）
3. 更好地代表利害關係人
4. 增強創新

### 多元性的負向機制
- 衝突成本上升
- 決策時間增長
- 整合挑戰

## 獨立性實務判斷

### 四重標準

**1. 法規獨立**
符合證交所 / 交易所的獨立性定義（具體見台灣公司治理中心公告）。

**2. 經濟獨立**
- 獨董酬金不應為其主要收入來源
- 獨董不依賴該公司其他業務收入
- 獨董公司 / 配偶不與該公司有重大商業關係

**3. 關係獨立**
- 與董事長 / CEO 無深度個人關係（大學同窗、長期球友、姻親等）
- 與其他董事 / 高管無利益結盟
- 不受某一派系邀請上任

**4. 心理獨立**
- 敢於挑戰管理階層
- 不迎合多數意見
- 能以少數股東利益為考量

### 法規 vs. 實質的落差

台灣案例中，**形式獨立但實質不獨立**常見模式：
- 獨董是 CEO 的大學同學 / 教練
- 獨董兼任公司長期法律顧問所屬事務所的律師
- 獨董是董事長重大捐款基金會的董事
- 獨董是董事長家族辦公室的顧問

這些情況可能符合法規、但實質獨立性存疑。EMBA 個案分析應挖掘此類「實質關係」。

## 獨立董事的「實質化」機制

### 1. 獨立聘顧問權
- 獨董遇重大議題可獨立聘會計師、律師、產業顧問
- 費用由公司負擔
- 需章程明文授權

### 2. 獨立會議（Executive Session）
- 獨董每季至少 1 次不含管理階層的會議
- 討論 CEO 評估、敏感議題
- 會議紀錄保密、僅關鍵決議呈報董事會

### 3. 獨立資訊管道
- 內部稽核主管可直接聯絡獨董
- 吹哨者信箱由外部獨立單位接收後轉獨董
- 財報季後獨董與會計師單獨會面

### 4. 書面意見權
- 對重大議題可提書面意見納入議事錄
- 對反對意見可公開揭露（符合規範）

### 5. 時間承諾
- 獨董每年對該公司投入應不少於 200 小時
- 兼任不應超過 5 家
- 關鍵時期（危機、IPO）大幅增加投入

## 董事會運作的實務細節

### 議程設計
- **CEO Report**：通常占會議 40%
- **策略 / 長期議題**：應占 25%，但常被壓縮
- **財務報告**：15%
- **風險與合規**：15%
- **其他**：5%

**常見問題**：CEO Report 拖長 → 策略討論被推遲 → 董事會成為「打卡」。

### 董事會議資料分發
- 建議會議前 5 個工作天送達
- 資料應含「議題、背景、選項、建議、決議事項」
- 避免「當天發資料」（常見於家族企業）

### 會議時間
- 每月 / 每季會議 2–4 小時為主
- 年度策略會議 1–2 天
- 特別會議（危機）隨時

### 董事會評估（Board Evaluation）
- **自評**：每位董事填表
- **他評**：董事互評
- **外部評估**：每 3–5 年請外部顧問
- 結果用於改選、能力補強、議程改革

## 家族企業的董事會特殊性

### 家族董事 vs. 非家族董事
- 家族董事：代表家族利益、長期導向
- 非家族 / 獨立董事：代表整體股東、獨立判斷
- 比例建議：非上市家族企業獨立董事 ≥ 1/3

### 家族董事間的協調
- 家族董事應在家族委員會先達成共識
- 避免家族分歧直接帶入董事會（家醜外揚）
- 家族內部衝突需有長輩 / 外部仲裁機制

### 姻親 / 第三代的董事會準備
- 第三代可從「觀察員」起步
- 姻親通常不進董事會（避免婚變衝擊）
- 家族憲法明訂進入條件

## 國際最佳實踐對照

### 紐交所 / 那斯達克標準
- 獨立董事多數
- 審計 / 薪酬 / 提名委員會必須全獨董
- 獨立董事定期獨立會議
- CEO Chair 分離（建議）

### 英國 UK Corporate Governance Code
- 非執行董事至少一半
- Chair 應為獨立非執行董事（IPO 當時起算）
- 每年董事會評估、每 3 年外部評估

### 德國雙層制（Dual Board）
- 監事會（Supervisory Board）與執行董事會（Management Board）分離
- 勞方代表參與監事會
- 利害關係人治理模式

### 日本 Corporate Governance Code
- 獨立董事至少 1/3（Prime Market 2022 起）
- 強調「對話型」治理（股東與董事會）
- 「Stewardship Code」機構投資人盡職治理

## EMBA 個案分析建議

### 診斷題目模板
1. **董事會能力矩陣**：列表與 gap 分析
2. **獨立性實質檢視**：不只法規、挖實質關係
3. **委員會運作評估**：出席率、決議品質、獨立會議頻率
4. **多元性評估**：性別、年齡、背景
5. **與國際標竿比對**：選 2–3 家標竿公司對照
6. **治理升級建議**：具體、可檢驗、時程明確

### 資料來源
- 年報 / 公司治理報告
- 證交所「重大訊息公告」
- 公司治理評鑑結果
- 媒體報導（注意可靠性）
- 董事會議事錄（若可得）
- 學術論文與案例研究
