# Example: 外送平台補貼政策對訂單量的因果效應

## Scenario

**FoodRocket** 是台灣一家外送平台，2024 年 Q2 在台北市推出「首單免運費」補貼活動（補貼上限 NT$60），活動起始日為 2024-04-01。活動僅限台北市，高雄、台中、台南未參與。

資料科學團隊被問到：**這個補貼真的帶動台北市訂單量成長，還是只是季節性效應？**

可用資料：
- 4 個城市、2024 年 1 月至 6 月、每週聚合訂單量
- 週維度 panel，共 24 週 × 4 城市 = 96 觀測值
- 控制變數：平均氣溫、週末/假日佔比、活躍用戶數（Log）

---

## Analysis

### Step 1 — 定義處置組與對照組

| 元素 | 設定 |
|------|------|
| 處置組（Treat = 1） | 台北市 |
| 對照組（Treat = 0） | 高雄、台中、台南（三城市合併） |
| 政策上線日 | 2024-04-01（W14） |
| 前期（Pre） | W1–W13（2024-01-01 至 2024-03-31） |
| 後期（Post） | W14–W24（2024-04-01 至 2024-06-16） |

可比性確認：補貼前四城市的平均訂單量成長率、用戶年齡分布、餐廳類別比例相近，無同期競爭性活動。

---

### Step 2 — 平行趨勢測試

以事件研究規格跑 W1 為基期，估計各週前期係數：

```
Log(Orders_it) = α_i + γ_t + Σ δ_k × (Treat_i × Week_k) + ε_it
k ∈ {W2,...,W24}, W1 為基準
```

**前期 leads（W2–W13）估計結果：**

| 前期週次 | δ 係數 | S.E. | p-value |
|----------|--------|------|---------|
| W2 (t-12) | +0.012 | 0.019 | 0.53 |
| W4 (t-10) | -0.008 | 0.021 | 0.71 |
| W6 (t-8)  | +0.015 | 0.018 | 0.41 |
| W8 (t-6)  | -0.003 | 0.020 | 0.88 |
| W10 (t-4) | +0.011 | 0.017 | 0.52 |
| W12 (t-2) | +0.006 | 0.016 | 0.71 |
| W13 (t-1) | -0.009 | 0.018 | 0.62 |

前期所有係數均不顯著（F-test p = 0.84），視覺圖形顯示台北與其他三市趨勢高度平行。**平行趨勢假設通過。**

---

### Step 3 — 估計 DID 模型

基礎迴歸方程式：

```
Log(Orders_it) = β₀ + β₁×Treat_i + β₂×Post_t + β₃×(Treat_i × Post_t) + Controls_it + α_i + γ_t + ε_it
```

標準誤在城市層級做 clustering（4 clusters，需注意樣本數偏少，見 Gotchas）。

**估計結果：**

| 規格 | β₃ (Treat×Post) | S.E. | p-value | N |
|------|-----------------|------|---------|---|
| 基準（雙向固定效果） | +0.187 | 0.041 | 0.003 | 96 |
| 加入氣溫、假日、Log(用戶數) | +0.163 | 0.037 | 0.004 | 96 |
| 排除台南（敏感性） | +0.171 | 0.043 | 0.011 | 72 |

**解讀：** 補貼上線後，台北市訂單量相較對照城市額外成長約 **16–19%**（ATT），加入控制變數後效果略縮小但仍顯著。

---

### Step 4 — 穩健性檢定

**安慰劑測試（假處置日）：** 將處置日設為 2024-02-05（W6），前期資料重跑 DID，β₃ = +0.021，p = 0.61，**通過**。

**假處置組測試：** 將高雄設為假處置組（對照為台中、台南），β₃ = +0.008，p = 0.79，**通過**。

**Wild Bootstrap（補償 cluster 數不足）：** 4 個 cluster 遠低於 50 的安全門檻，改用 wild cluster bootstrap（1,000 次重複）。Bootstrap p-value = 0.018，結論不變。

---

## Result

```markdown
## DID Analysis: FoodRocket 台北免運費補貼（2024-Q2）

### Research Design
| Element | Description |
|---------|-------------|
| Treatment group | 台北市（1 城市） |
| Control group | 高雄、台中、台南（3 城市） |
| Treatment date | 2024-04-01（W14） |
| Pre-treatment periods | 2024-01-01 至 2024-03-31（W1–W13） |

### Parallel Trends Test
| Pre-period lead | Coefficient | S.E. | p-value |
|-----------------|-------------|------|---------|
| t-12 | +0.012 | 0.019 | 0.53 |
| t-6  | -0.003 | 0.020 | 0.88 |
| t-2  | +0.006 | 0.016 | 0.71 |
| t-1  | -0.009 | 0.018 | 0.62 |

F-test (all pre-period leads = 0): p = 0.84 ✓

### DID Estimate
| Specification | β (Treat×Post) | S.E. | p-value | N |
|---------------|----------------|------|---------|---|
| Baseline | +0.187 | 0.041 | 0.003 | 96 |
| With controls | +0.163 | 0.037 | 0.004 | 96 |
| Wild bootstrap | +0.163 | — | 0.018 | 96 |

### Robustness
- Placebo test (fake date W6): β = +0.021, p = 0.61 ✓ 通過
- Fake treatment group (高雄 vs 台中/台南): β = +0.008, p = 0.79 ✓ 通過
- Alternative control (排除台南): β = +0.171, p = 0.011 ✓ 結論一致

### Limitations
- Cluster 數僅 4 個，已採 wild bootstrap；若能拆分為更細行政區可改善
- 本估計為 ATT（台北市處置組的平均處置效果），無法外推至其他城市
- 未能排除台北競爭對手同期未推出反制活動（無競品資料）
- 補貼期間用戶組成可能改變（低頻用戶被吸引），存在輕微 composition bias 風險
```

**結論：** 台北市免運費補貼使訂單量增加約 **16%（ATT）**，效果在多重穩健性檢定下一致。若補貼成本低於每週增量訂單所帶來的毛利，政策值得擴大推廣；但推廣前應評估高雄、台中的平行趨勢是否對這兩市同樣成立（需重新建立各城市的 DID 設計，而非直接假設效果可外推）。
