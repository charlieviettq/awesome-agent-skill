# ADR reference templates

## Standard (MADR-style) excerpt

```markdown
## Decision drivers

* Must have ACID compliance
* Must support complex queries
* Team familiarity

## Considered options

### Option A
- **Pros:** ...
- **Cons:** ...

### Option B
- **Pros:** ...
- **Cons:** ...

## Decision
We will use **Option A** because ...

## Related decisions
- ADR-0002: ...
```

## Y-statement

```markdown
In the context of [situation],
facing [forces],
we decided for [option]
and against [alternatives],
to achieve [outcome],
accepting [tradeoff].
```

## Deprecation ADR

When replacing a prior decision, reference the superseded ADR, migration plan, and lessons learned.
