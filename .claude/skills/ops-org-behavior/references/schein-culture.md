# Schein 組織文化三層次：深度解析與 EMBA 應用

## Schein 其人與貢獻

Edgar H. Schein（1928–2023），MIT Sloan 教授，組織心理學奠基者。核心著作：
- *Organizational Culture and Leadership*（1985，2017 第 5 版）
- *The Corporate Culture Survival Guide*（2009）
- *Humble Inquiry*（2013）

他的核心命題：**文化是領導者唯一真正重要的工作**。其他事情都是可被代理、外包、制度化的；只有文化，需要 CEO 親自經營。

## 三層次模型（詳解）

### Level 1：Artifacts（人造物）

**定義**：任何可觀察、可聽到、可感受到的文化表現。

**類別**：
- 物理環境：辦公室配置、裝潢、服裝、標誌、產品
- 語言：公司用語、口號、專業術語
- 儀式：會議形式、節慶、獎項
- 故事：創辦神話、英雄故事、失敗教訓
- 行為規範：誰先打招呼、會議遲到如何處理

**特徵**：
- **容易觀察、容易誤解**
- 一個「開放式辦公室」可能代表「扁平、協作」（如矽谷），也可能代表「監視、壓力」（如某些血汗工廠）
- 不能從 Artifacts 直接推論文化

### Level 2：Espoused Values（信奉價值）

**定義**：組織公開聲明、書面化的信念與價值。

**來源**：
- 使命宣言（Mission Statement）
- 願景（Vision）
- 核心價值觀（Core Values）
- 策略文件、員工手冊
- 領導者公開發言

**關鍵洞察**：
- 多為**理想的、應然的**價值
- 與實際行為可能有落差（「說一套做一套」）
- Schein 稱此為「**公開宣稱但未必體現**」

**EMBA 診斷重點**：
- 觀察 Espoused Values 與 Artifacts 是否一致
- 若不一致 → 文化是病態的、或處於變革陣痛中

### Level 3：Basic Underlying Assumptions（基本假設）

**定義**：組織成員未經思考、視為理所當然的信念，形塑實際行為。

**七大領域**（Schein, 2017）：
1. **與環境的關係**：組織能控制環境還是順應？
2. **現實與真理的本質**：真理來自權威、科學、還是辯論？
3. **人性的本質**：人是善、惡、中立、可變？（連結 McGregor X/Y）
4. **人類活動的本質**：應主動行動還是被動適應？
5. **人際關係的本質**：階層、合作、競爭？
6. **時間的本質**：過去、現在、未來導向？短期或長期？
7. **空間與隱私**：開放或封閉？

**特徵**：
- **難以觀察**：成員自己不自覺
- **極難改變**：觸及這層會引發強烈情緒防衛
- **真正驅動行為**：當 Espoused Values 與 Basic Assumptions 衝突，行為會遵循 Basic Assumptions

### 三層之間的關係

```
Artifacts（看得到） ── 表現
   ↑
Espoused Values（說得出） ── 宣稱
   ↑
Basic Assumptions（想不到要說） ── 真正驅動
```

**診斷提問**：
- Artifacts 支持還是抵觸 Espoused Values？
- Espoused Values 與推論出的 Basic Assumptions 是否一致？

## 文化的三個起源

Schein 指出文化由三個來源形塑：
1. **創辦人的信念與價值觀**（最強烈）
2. **群體成員共同的學習經驗**（危機、成功）
3. **新成員帶入的新假設**

→ **創辦人效應**：為何家族企業文化極難改？因為根基是一個人的基本假設。

## 文化學習機制（Learning Mechanisms）

Schein 區分兩種：

### 正向強化（Positive Problem-Solving）
組織面對挑戰 → 採取方法 → 成功 → 「這個方法有用」成為假設
→ 這類假設**穩定但偶爾需更新**

### 焦慮降低（Anxiety Reduction）
組織面對威脅 → 建立規則減少焦慮 → 「這樣做比較安全」成為假設
→ 這類假設**極難改變**（改變 = 回到焦慮狀態）

**EMBA 啟示**：
- 許多「我們一直這樣做」其實是焦慮降低機制，不是有效做法
- 變革時要承認恐懼、提供新的心理安全感，否則組織會退回舊假設

## 文化改造（Cultural Change）

### Schein 的變革模型（基於 Lewin 三階段）

1. **解凍（Unfreezing）**
   - 製造「生存焦慮」（現狀不可持續）
   - 降低「學習焦慮」（有路徑、有支持）
   - **核心**：學習焦慮 < 生存焦慮 時，改變才會發生

2. **認知重構（Cognitive Restructuring）**
   - 新資訊、新視角、新經驗
   - 通常需要離開舒適圈（工作輪調、外派、教練）

3. **再凍結（Refreezing）**
   - 新行為產生新結果 → 獲得肯定
   - 寫入制度、儀式、人事評價

### 變革機制工具

Schein 列出 CEO 能使用的「**主要嵌入機制**」：
1. **領導者關注、衡量、控制什麼**
2. **領導者對危機與關鍵事件的反應**
3. **資源分配的優先順序**
4. **審慎的角色示範與指導**
5. **獎酬與地位的分配**
6. **招聘、晉升、離退的標準**

**次要嵌入機制**：
1. 組織設計與結構
2. 制度與程序
3. 儀式與典禮
4. 物理空間、外觀、建築
5. 故事、傳奇、神話
6. 正式的組織理念聲明

### 主要 vs. 次要的差別
主要機制反映**領導者實際的優先順序**（不可假裝），次要機制可被政治操作。
**看 CEO 時間表比看願景聲明更能讀懂文化。**

## EMBA 個案診斷的五步法（Schein 式）

### Step 1：進入現場（Artifacts 蒐集）
- 辦公室走動、列席會議、看資料
- **不要急於解讀**

### Step 2：詢問 Espoused Values
- 查閱公開文件
- 訪談領導層：「你們重視什麼？」

### Step 3：觀察落差
- Artifacts 是否支持 Espoused Values？
- 若不支持 → 哪些 Basic Assumptions 可能在運作？

### Step 4：假設化驗證
- 提出 2–3 個 Basic Assumptions 的假設
- 透過後續訪談、關鍵事件分析驗證

### Step 5：組織成員共同詮釋
- Schein 強調：**文化診斷是與組織對話的過程**
- 讓組織成員檢視並修正外部診斷
- 診斷的目的是促進組織自我覺察，不是貼標籤

## 常見 EMBA 個案誤用

### 誤用 1：只寫 Artifacts + Espoused Values
- 學員：「這家公司文化是開放、創新、重視員工」（抄網站）
- 問題：停留在表層，沒觸及基本假設
- 修正：追問「如果真的開放，為何離職率 25%？」

### 誤用 2：把 Artifacts 直接當 Basic Assumptions
- 學員：「他們有開放式辦公室，所以基本假設是扁平」
- 問題：Artifacts 可支持多種假設
- 修正：用多個 Artifacts + 行為證據交叉驗證

### 誤用 3：期待文化短期改變
- 學員：「建議 6 個月內導入新文化」
- 問題：Basic Assumptions 平均 18–36 個月才能動搖
- 修正：短期改 Artifacts + 制度，中期改 Espoused Values，長期才有 Basic Assumptions 變化

### 誤用 4：以為「強文化」必然好
- 學員：「公司文化強、凝聚力高、值得學習」
- 問題：強文化遇環境劇變可能是陷阱（柯達、諾基亞的文化慣性）
- 修正：檢視文化適應力、迭代機制

## 與其他文化模型的對照

### Hofstede 國家文化六維
- 比較「社會層」文化
- 用於跨國管理、國際併購
- 與 Schein 互補：Schein 看「單一組織內部」，Hofstead 看「國家群體間」

### Cameron & Quinn CVF（競爭價值框架）
- 四類文化：氏族、創新、市場、科層
- OCAI 問卷可量化
- **工具性強但深度弱**：適合快速盤點、不適合深度文化診斷

### Denison 組織文化模型
- 四維度：使命、適應、一致、參與
- 強調績效連結（有實證資料）
- 適合做組織對比、文化健檢

### Trompenaars 文化七維
- 強調跨文化管理
- 與 Hofstede 類似，但更細緻（普遍主義 vs. 特殊主義等）

**EMBA 論文常用組合**：Schein（深度）+ CVF 或 Denison（量化）

- 「設備不壞就不要動」

### 科技業 / 新創
- 「快還要更快」
- 「失敗是常態、成功是例外」
- 「工程師優於業務 / 營運」
- 「加班是投入的象徵」

### 金融 / 保險
- 「穩定優於創新」
- 「合規是天條」
- 「資歷定位階」

**警示**：這些是**預設假設池**，用於引導診斷思考，**不可直接套在任何個案上**。每個組織需回到三層次模型自己驗證。
