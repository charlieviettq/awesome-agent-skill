# 9-box Grid：歷史、變體與實證研究筆記

## 起源

9-box grid 起源於 1970 年代 General Electric 與 McKinsey 合作的「GE-McKinsey Matrix」，原用於「事業組合評估」（以市場吸引力與事業競爭地位為軸）。

1980 年代後被 GE 的人資系統改造為「人才 9-box」，以 **績效** × **潛力** 為雙軸，成為全球 HR 標配工具。2000 年代後期被多家顧問公司（Korn Ferry、DDI、Mercer）系統化。

## 標準架構

### 雙軸定義
- **績效（Performance）**：過去 12 個月的工作成果
- **潛力（Potential）**：未來 2–5 年晉升到更高職位的可能性

### 九格分類
```
         低績     中績     高績
高潛  │ 疑問   │ 新星  │ 明星★ │
中潛  │ 可觀察 │ 核心  │ 高績骨幹│
低潛  │ 低效   │ 穩定  │ 穩定高手│
```

## 主要變體

### 變體 1：Korn Ferry 4-Box（簡化）
兩軸各兩級：高／低。4 格。
- **優**：校準快、爭議少
- **劣**：缺層次、中間人沒有區隔

### 變體 2：DDI 6-Box（績效 3 級 × 潛力 2 級）
聚焦於「高潛與非高潛的區隔」，績效分三級。
- **優**：強調潛力獨立評估
- **劣**：非高潛者的發展資源可能被忽視

### 變體 3：5×5 Grid
兩軸各五級，25 格。
- **優**：極細緻
- **劣**：校準困難、主管疲勞

### 變體 4：「**準備度 × 關鍵性**」
軸改為「晉升準備度」與「崗位關鍵性」，用於繼任規劃階段。
- **優**：直接對接繼任決策
- **劣**：無法用於一般人才盤點

### 變體 5：「3+1」（敬業度追加）
9-box 外加第三維度「敬業度 / 投入度」。
- **優**：識別「高潛但流失風險高」的雙面明星
- **劣**：視覺化困難、需資訊系統支援

## 「潛力」的核心指標

Korn Ferry（2014）定義「Learning Agility」為潛力核心，包含五維：

| 維度 | 定義 | 問句 |
|---|---|---|
| **Mental Agility（心智敏捷）** | 複雜問題分析能力 | 面對未知議題能否從多角度切入？ |
| **People Agility（人際敏捷）** | 理解、影響多元他人 | 能否和各類型人建立互信？ |
| **Change Agility（變革敏捷）** | 擁抱變化、驅動實驗 | 對模糊情境的容忍度如何？ |
| **Results Agility（結果敏捷）** | 在新情境仍產出結果 | 第一次做某事能否做成？ |
| **Self-Awareness（自我覺察）** | 知道自己的強弱 | 能否真實描述自己的 gap？ |

### 其他潛力指標
- **Lominger 67 competencies**
- **DDI 領導潛力評估工具**
- **Gallup StrengthsFinder**
- **Mercer 潛力評鑑中心**

## 實證研究觀察

### 潛力評估的信度

多項研究（Church & Rotolo, 2013；Silzer & Church, 2009）指出：
- **單一主管評潛力**：信度約 0.3–0.5（偏低）
- **加入 360 評鑑**：信度提升至 0.5–0.7
- **加入評鑑中心（Assessment Center）**：信度可達 0.7–0.85

**啟示**：重要人才評估（例如繼任候選）不應僅靠主管主觀判斷。

### 績效 × 潛力的相關性

傳統誤解：「績效好 → 潛力高」。
實證（Silzer & Church, 2009）：
- 高績效員工中，約 30% 為高潛、40% 中潛、30% 低潛
- 高潛員工中，約 50% 高績效、30% 中績、20% 低績
- 兩者相關係數約 **0.3–0.4**（中度正相關但非同一概念）

**啟示**：潛力須獨立評估；不能以績效代理。

### 9-box 的實際使用問題

**Ledford & Kochanski (2004)** 調查 Fortune 500：
- 76% 使用 9-box 或類似工具
- 但僅 34% 認為其「顯著影響」晉升決策
- 主要問題：主管的「評高效應」、政治妥協、校準失敗

## 常見「評級偏誤」

| 偏誤 | 表現 | 防範 |
|---|---|---|
| **集中趨勢** | 大家都評 5/10 | 強制分佈 |
| **寬鬆誤差** | 主管不敢評低 | 校準會議 + HR 挑戰 |
| **嚴苛誤差** | 主管全評低 | 跨主管比較 |
| **月暈效應** | 一項好 → 全部好 | 分維度獨立評 |
| **近因誤差** | 最近事件主導 | 全年度事件清單 |
| **相似偏誤** | 像我 → 高評 | 多主管交叉評 |
| **性別 / 年齡偏誤** | 歸因刻板印象 | HR 挑戰、匿名校準 |

## 校準會議（Calibration Meeting）設計

### 目的
- 統一主管對「明星」「新星」等標準的理解
- 揭露主管的偏誤
- 使盤點結果可跨部門比較

### 流程（典型 3 小時）
1. HR 主持說明規則（15 分鐘）
2. 每部門主管呈報本部門盤點結果（各 20 分鐘）
3. 跨部門質疑與討論（各 15 分鐘）
4. 最終評級決議（可以覆寫主管評級）（30 分鐘）
5. 爭議個案留待後續會議

### 強制分佈（Forced Distribution）
- **支持方**：避免「全民明星」虛胖、迫使主管思考差異
- **反對方**：扭曲真實、影響團隊合作
- **折衷**：對「明星 / 新星」設上限（如不超過 20%）、對「低效」設下限（如至少 5%）

## 9-box 的批評與回應

### 批評 1：「人不能被格子化」
- **回應**：9-box 是**討論工具**不是**判決**。格子能促成對話，比「主管腦中印象」更透明。

### 批評 2：「激發政治與焦慮」
- **回應**：正確做法不公開個人評級，只公開集體分佈與資源配置。

### 批評 3：「強化現有偏誤」
- **回應**：引入 360、評鑑中心、多次校準，可部分抵消。

### 批評 4：「高流動率時代已不適用」
- **回應**：在 2 年內可能換一批人的環境，9-box 應改為半年度重校準。

## 9-box 與個人發展的連結

### 各格的發展資源配比（參考）

| 分類 | 建議資源投入 | 重點發展活動 |
|---|---|---|
| 明星 | 最高（15%） | 跨事業輪調、董事會曝光、外部教練 |
| 新星 | 高（15%） | 關鍵專案、導師配對 |
| 高績骨幹 | 中（25%） | 專精深化、保留方案 |
| 核心骨幹 | 中（20%） | 職涯選擇支援 |
| 穩定高手 | 中（10%） | 技術軌晉升、知識傳承角色 |
| 疑問 | 中（5%） | 診斷 + 調崗機會 |
| 其他 | 低（10%） | 日常培訓 |

## 9-box 結果的用途

✅ 合適用途
- 年度人才盤點討論的起點
- 繼任規劃候選人的篩選
- 培訓資源配置
- 升遷、關鍵專案機會分派

❌ 不合適用途
- 薪酬決定的唯一依據
- 裁員名單的直接來源
- 員工公開張貼

## 台灣企業的特殊考量

- **面子文化**：不公開評級、避免集體排名
- **家族成員**：建議雙軌盤點（家族 vs. 專業）
- **年資文化**：高年資員工常被「自動進」核心骨幹，需挑戰
- **跨世代衝突**：Z 世代年輕員工用舊標準評可能失真
