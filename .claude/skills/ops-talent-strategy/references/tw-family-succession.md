# 台灣家族企業接班與股權規劃筆記

## 為什麼家族企業是台灣 EMBA 的重要主題

- 台灣上市櫃公司中家族企業占比 65–70%（依定義）
- 中小企業家族比例更高（超過 90%）
- 第一代創辦人多 1950–1970 年代出生，2026 年多在 60–80 歲
- 未來 10 年將是大規模世代交替期
- EMBA 學員中大量為「二代接班人」或「家族企業專業經理人」

## 三圈模型再述（Tagiuri & Davis, 1996）

```
     家族 Family
       ╱    ╲
      ╱      ╲
     ╱   重疊  ╲
    ╱          ╲
所有權 Ownership ── 經營 Business
```

**任一人身份落點決定其視角**：
- 純家族成員：情感驅動
- 純股東：分紅驅動
- 純員工：績效驅動
- 多圈重疊：高度政治

## 家族企業生命週期

### 第一代：創業期
- 創辦人同時持有三圈
- 決策迅速、權威集中
- 家族即公司、公司即家族
- 重要決策：何時讓下一代進入？以什麼身份？

### 第二代：兄弟姐妹期（最常見衝突階段）
- 多子女共同繼承，三圈開始分離
- 「誰是老大」的繼承政治
- 股權、職位、分紅三軌可能各自分配
- 重要決策：是否引入專業經理人？家族憲法是否制定？

### 第三代：堂表親期
- 多房多分支、家族人數倍增
- 股權高度分散（每人持股 < 5%）
- 親疏有別，對公司感情淡化
- 重要決策：是否維持家族經營？或轉為純投資人？

### 第四代以後：王朝期
- 類似歐洲百年家族
- 家族辦公室取代家族企業經營
- 慈善、投資、傳承為主軸

**統計**：全球家族企業能跨過「第一代→第二代」的僅 30%，跨過「第二代→第三代」的僅 12%。

## 台灣家族企業接班的特殊議題

### 1. 女兒接班的上升

- 傳統重男輕女漸變
- 2020 後女兒接班比例顯著提升（國泰、裕隆、台玻、華新麗華等案例）
- 困難：老員工接受度、供應商 / 客戶印象、家族內部政治
- 建議：
  - 女兒進公司前建立「自己的事業經驗」（外商、創業、顧問）
  - 由外部董事 / 家族長輩公開背書
  - 接班路徑從「非家族核心事業」起步累積戰功

### 2. 媳婦 / 女婿的定位

- 「入贅」傳統已淡、但「姻親進入公司」仍有爭議
- 風險：婚變時的股權處理
- 建議：
  - 原則上姻親不進核心職位（但可擔任顧問或周邊事業）
  - 股權若給予，需婚前 / 婚後協議規範
  - 家族憲法明訂姻親進入條件

### 3. 家族成員 vs. 專業經理人

#### 三種模式
**Model A：家族主導**
- 家族成員任 CEO、關鍵職位
- 專業經理人任執行層
- 例：多數中部傳產

**Model B：專業主導**
- 家族退居股東 / 董事
- CEO、CFO 皆專業經理人
- 例：早期宏碁、部分科技業

**Model C：雙軌並行**
- 核心事業家族經營、新事業專業化
- 例：多數大型集團

#### 選擇決策樹
- 家族成員是否具備職能？→ 若否，Model B / C
- 家族是否能放下控制？→ 若否，Model A（但要承認上限）
- 組織規模與複雜度？→ 大而複雜 → Model B / C

### 4. 股權傳承

#### 常見工具
- **直接贈與**：簡單但稅負高（贈與稅 / 遺產稅）
- **股份控股公司**：第二層控股拉長稅務處理時間
- **家族信託**：近年興起，適合高資產（>10 億台幣）
- **公益信託 / 慈善基金會**：社會形象與稅務雙重考量
- **員工持股信託**：部分股權分散至員工、增強忠誠度

#### 台灣遺產稅與贈與稅（2025 現況）
- 稅率：10–20%（累進）
- 免稅額：遺產 1,333 萬、贈與 244 萬 / 年
- 規劃建議：提早 10–20 年分年贈與、搭配股份控股公司

#### 股權鎖定條款
- 家族成員內部轉讓優先
- 離婚 / 離職 / 死亡的強制買回機制
- 外部收購的家族優先購買權

### 5. 家族衝突的制度化處理

#### 家族憲法（Family Constitution）
- 由家族共創、非老闆獨寫
- 內容涵蓋：進入條件、薪酬、分紅、衝突解決、股權、家族活動
- 每 5 年審議

#### 家族委員會（Family Council）
- 家族成員代表組成（不限定經營者）
- 與董事會平行運作
- 處理家族議題、向董事會提案

#### 家族辦公室（Family Office）
- 適用高資產（> 50 億台幣）
- 管理家族投資、稅務、慈善、下一代教育
- 獨立於公司本業

#### 獨立仲裁機制
- 家族衝突無法解決時的外部仲裁人（通常是長期律師、會計師、牧師等）
- 預先指定、寫入家族憲法

## 二代接班人的養成路徑

### 典型路徑

| 階段 | 年齡 | 重點 |
|---|---|---|
| 海外教育 | 18–25 | 大學 + 碩士（多為美國 MBA） |
| 外部歷練 | 25–32 | 外商 / 顧問業 5–7 年 |
| 進入公司（基層） | 32–35 | 從產線、門市、業務做起 |
| 中階主管 | 35–40 | 某事業部副總 / 新事業負責人 |
| 高階主管 | 40–45 | 事業部總經理 / COO |
| 接班 CEO | 45–55 | 完全接棒 |

### 常見失敗模式
- **空降 CEO**：直接接班無基層經驗 → 被老員工抵制
- **永遠接班**：50 歲還是「副總」，第一代不肯放手
- **海外回來跟不上**：脫離台灣產業太久、與現場斷層
- **被安排假職位**：名義主管實無權、學不到東西

### 成功因素
- 第一代有明確退出時間表
- 二代有「小王國」試煉（子公司 / 新事業）
- 家族外部導師支持
- 家族憲法與董事會治理配套

## 台灣家族企業的產業分布特色

### 傳統產業（紡織、食品、五金、機械）
- 家族色彩最濃
- 跨代傳承風險大（人才短缺）
- 經常面臨「收掉」或「賣掉」的選擇

### 金融保險
- 家族色彩漸淡（法規要求）
- 但控制權仍常在家族手中（如國泰、富邦、新光）
- 接班更強調「資本運用能力」

### 科技業
- 家族色彩較低（部分）
- 但創辦人仍有強烈影響（如廣達、仁寶）
- 接班議題近 5 年才開始浮現

### 零售 / 服務業
- 家族比例高
- 品牌與家族高度連結（如全聯、統一）
- 接班強調「消費者洞察」傳承

## 案例：台灣家族企業接班的公開討論

### 宏碁 / 施家
- 施振榮「傳賢不傳子」，建立專業經理人制度
- 困難：經理人選擇、文化傳承
- 啟示：家族退出經營不等於退出影響

### 國泰金 / 蔡家
- 家族內部分工、跨世代協同
- 家族憲法運作相對成熟
- 啟示：金融家族的治理化路徑

### 裕隆 / 嚴家
- 第二代嚴凱泰英年早逝後，嚴陳莉蓮接班
- 女性接班的典範
- 啟示：配偶接班的制度化準備

### 長春集團 / 林家
- 低調家族、第三代逐步接棒
- 家族憲法運作健全
- 啟示：專業化與家族的平衡

**警示**：以上案例為公開資訊整理，細節請以當事人官方聲明為準。EMBA 個案分析時避免揣測未公開事項。

## EMBA 個案寫作的建議

### 取材原則
- **優先寫自家企業**（學員本人為二代時）
- **訪談需取得長輩同意**
- **敏感議題（股權、衝突）應匿名化處理**
- **避免揣測競爭對手家族事務**

### 分析框架建議
1. **三圈模型**：先釐清各家族成員身份
2. **職能盤點**：家族成員與專業人士分開評估
3. **股權現況**：製作股權結構圖
4. **繼任規劃**：短中長三層接班人
5. **家族治理**：現行機制與建議優化
6. **學習與反思**：不只是診斷、也要有「如果是我會怎麼做」

### 學術誠信紅線
- 家族隱私不可外洩
- 數字如屬未公開上市資料，需脫敏
- 引用訪談需徵詢同意
- 作業結論應以「建議」而非「揭發」定位
