# 台灣合約法：民法典關鍵條文與實務差異

> 本文件補充說明 `law-contract` 技能中「台灣特殊性」的部分，聚焦於台灣民法與英美合約法（common law）的關鍵差異，以及特定契約類型的強制規定。

---

## 1. 台灣民法的契約形成要件

台灣契約法源自大陸法系（civil law），規定於《民法》第 153 條起。與英美 common law 最關鍵的差異：

### 1.1 不需要 Consideration（對價）

Common law 要求雙方均須提供「有價值之事物」（consideration）才能成立契約。台灣民法**不採此要件**。

| 要件 | Common Law | 台灣民法 |
|------|-----------|---------|
| 要約（Offer） | ✅ 必要 | ✅ 必要（§154） |
| 承諾（Acceptance） | ✅ 必要 | ✅ 必要（§153） |
| 對價（Consideration） | ✅ 必要 | ❌ **不需要** |
| 合法性（Legality） | ✅ 必要 | ✅ 必要（§71-72） |
| 當事人能力（Capacity） | ✅ 必要 | ✅ 必要（§75-85） |

**實務影響**：台灣法下，一方承諾免費提供服務的契約，只要雙方合意即可成立並具拘束力，不需對方提供任何回報。這使得贈與契約、無償委任在台灣具備直接的法律效力。

### 1.2 要約與承諾的規則（§153-162）

**要約的拘束力：**

```
要約人發出要約後 → 要約人原則上受拘束，不得任意撤回
例外：要約明示可撤回，或依情形要約本質上可撤回
```

**承諾的時限：**

| 情形 | 規則 | 民法條文 |
|------|-----|---------|
| 要約定有承諾期限 | 逾期承諾無效 | §158 |
| 要約未定期限（對話間） | 即時承諾 | §156 |
| 要約未定期限（非對話間） | 通常可期待到達之時間內 | §157 |
| 遲到的承諾 | 視為新要約 | §160 |

**承諾內容的鏡像規則（Mirror Image Rule）：**

台灣民法 §160 第 2 項：承諾附有條件或限制者，視為拒絕原要約，並為新要約。這與 common law 的鏡像規則一致，但台灣法明文規定更清晰。

## 2. 定型化契約的強制規定（消費者保護法）

當契約一方為消費者（自然人、非以商業目的）時，適用《消費者保護法》（消保法），對「定型化契約」有強制限制。

### 2.1 什麼是定型化契約？

**定義（消保法 §2 第 7 款）：**

> 企業經營者為與多數消費者訂立同類契約之用，所預先擬定之契約條款。

判斷標準（三要素）：
1. 企業經營者預先擬定
2. 供多數消費者使用
3. 消費者無實質議價空間

### 2.2 無效條款類型（消保法 §12）

定型化契約條款若有下列情形，**當然無效**（不需法院介入即自始無效）：

| 類型 | 典型條款範例 | 為何無效 |
|------|------------|---------|
| 顯失公平 | 「本公司保留最終解釋權」 | 排除消費者爭議權利 |
| 免除故意/重大過失責任 | 「本公司對任何損害不負責任」 | §222 民法亦禁止 |
| 單方修改權 | 「本公司得隨時修改服務條款」（無通知義務） | 剝奪消費者知情權 |
| 不合理限制求償 | 退款限制在付款後 3 日內（明顯不合理） | 限制消費者法定權利 |

**IRON LAW 強化**：在台灣 B2C 情境，「合法性」要件不只指契約目的合法，還包括個別條款不得違反消保法強制規定。一個整體目的合法的契約，可能因個別條款無效而產生空白或重新議定的需求。

### 2.3 主管機關公告的「應記載及不得記載事項」

主管機關（行政院消保處）針對特定產業公告定型化契約範本，違反者條款無效：

| 行業 | 規範重點 |
|------|---------|
| 預售屋買賣 | 價金找補、坪數誤差處理方式 |
| 補習班 | 退費計算公式（依上課比例退費） |
| 健身房 | 月費制 vs. 點數制；倒店退費機制 |
| 電商平台 | 7 日鑑賞期不得以任何理由排除 |
| 手機門號 | 提前解約違約金上限 |

**補習班退費公式（主管機關公告版）：**

```
退費金額 = 已繳學費 × (剩餘課時 / 總課時)
```

若業者在契約中訂定不同計算方式（如「報名費不退」、「行政手續費 20%」），**該條款無效**，仍依上述公式計算。

## 3. 特定契約類型的強制規定

### 3.1 勞動契約（勞動基準法）

勞動契約雖為私法契約，但受《勞動基準法》強制介入，當事人不得以約定排除勞基法最低保障。

**不得以契約排除的勞工權利：**

| 項目 | 勞基法規定 | 契約能否調整 |
|------|-----------|-----------|
| 基本工資 | 現行 27,470 元/月（2024） | 只能往上，不能往下 |
| 每日正常工時 | 8 小時 | 不得增加 |
| 加班費計算 | 平日延長：前2小時 4/3，後續 5/3 | 不得降低 |
| 特休假 | 依年資累計 | 不得減少 |
| 資遣費 | 依年資計算 | 不得排除 |

**常見無效的勞動契約條款：**

```
❌ 「本人同意放棄特休假，以現金補償代替。」
→ 雖有書面同意，仍無效。特休假是法定休假，不能事前放棄。

❌ 「離職後一年內不得至同業任職，違者賠償 100 萬元。」
→ 競業禁止需符合比例原則；賠償金額若顯不相當，法院得酌減（民法 §252）。

✅ 「離職後六個月內不得以同類技術服務直接競爭客戶，公司應支付競業補償金每月 50% 原薪。」
→ 限制合理、有補償，較可能被法院認可。
```

### 3.2 競業禁止條款的有效性檢驗（勞基法 §9-1）

2015 年修法後，競業禁止須符合四要件：

```
① 雇主有應受保護的正當營業利益
② 勞工職位與接觸機密有關
③ 競業禁止期間、地域、職業、就業對象合理（期間上限 2 年）
④ 雇主給付合理補償（不低於離職前平均薪資 50%）
```

**四要件缺一，競業禁止條款即屬無效。**

決策流程：

```
契約中有競業禁止條款？
    ↓ Yes
雇主有正當利益可保護？（商業秘密、客戶名單、核心技術）
    ↓ Yes
勞工職位會接觸該利益？（一般行政人員通常不符合）
    ↓ Yes
期間 ≤ 2 年，地域/職業範圍合理？
    ↓ Yes
雇主承諾補償 ≥ 50% 原薪？
    ↓ Yes
→ 條款可能有效，但仍視法院認定
    ↓ 任一 No
→ 條款無效，勞工不受拘束
```

## 4. 契約解除 vs. 終止的法律差異

台灣法明確區分「解除」與「終止」，英文合約翻譯時常混用 "termination"，但中文法律效果截然不同：

| 概念 | 法律效果 | 適用情形 |
|------|---------|---------|
| **解除**（§258-263） | 溯及既往，回復原狀（返還已受領給付） | 一次性履行契約（買賣、承攬） |
| **終止**（§263準用） | 向後失效，已發生的部分仍有效 | 繼續性契約（租賃、僱傭、委任） |

**實務判斷：**

```
問：這是一次性給付還是繼續性給付的契約？

一次性：買賣、承攬、贈與
→ 違約時，他方可「解除」契約 + 請求損害賠償
→ 解除後，雙方應回復原狀（退錢、退貨）

繼續性：租賃、勞動、委任、授權
→ 違約時，他方通常「終止」契約 + 請求損害賠償
→ 終止後，僅停止未來的義務，過去履行部分仍有效
```

**常見合約草擬錯誤：**

```
❌ 「任一方得解除本授權協議。」
→ 授權是繼續性契約，應用「終止」。「解除」會要求雙方回復原狀，
  意味著被授權方須返還所有已依授權使用所獲得的利益，法律效果
  差異極大。

✅ 「任一方得終止本授權協議，終止前已授予之授權不受影響。」
```

## 5. 民法 §148 誠信原則的實務影響

台灣民法 §148 第 2 項：「行使權利，履行義務，應依誠實及信用方法。」

誠信原則在台灣契約法中是「兜底條款」，法院可據此：

1. **補充契約漏洞**：契約未明定事項，依誠信原則推定當事人意思
2. **限制契約條款**：即使條款明文，若行使方式有悖誠信，法院可拒絕支持
3. **附隨義務**：即使契約無明文，誠信原則可課予當事人通知、協力、保密等附隨義務

**案例型態：**

```
情境：SaaS 合約未約定停機通知義務，系統維護停機 12 小時，
      客戶因此損失訂單。

契約條款分析：
- 契約無明文通知義務 → 但依 §148 誠信原則
- 法院可能認定：提供服務的一方負有「合理提前通知」的附隨義務
- 結果：即使契約未明訂，供應商仍可能需負損害賠償責任
```

**對合約起草的影響**：不要依賴「契約未明定即不負責」的邏輯，誠信原則可填補空白，且方向不一定對起草方有利。重要的附隨義務應明文規定清楚（包括你**不**願意承擔的責任）。

## 6. 違約金與懲罰性賠償

台灣民法允許違約金條款（§250），但有重要的法院酌減機制。

### 違約金的兩種性質（§250）

| 性質 | 說明 | 預設 |
|------|-----|------|
| **損害賠償預定性** | 違約時，以違約金取代損害賠償，無需證明損害 | 台灣法預設為此 |
| **懲罰性違約金** | 違約金之外，仍可另外請求損害賠償 | 需在契約中明文約定為「懲罰性」 |

### 法院酌減違約金（§252）

若違約金「過高」，法院**得**（非「應」）酌減至相當金額。

酌減考量因素：
- 實際損害額
- 違約情節輕重
- 當事人之資力差距
- 債務人有無可歸責事由

**實務建議：**

```
設定違約金時，應能說明其合理性（對應預期損失）。
若合約對方是消費者（B2C），過高的違約金條款
依消保法 §12 可能直接無效（顯失公平）。

合理的違約金設定參考：
- SaaS 提前解約：剩餘月數 × 月費 × 50-100%
- 工程承攬延遲：每日罰款 = 合約金額 × 0.1-0.3%
- 保密協議洩露：具體損害難以估算者可設定較高定額，但需可合理解釋
```

## 快速檢查清單：台灣法下的契約審查附加步驟

以下為在通用 Contract Review 流程之外，台灣法特有的額外檢查項目：

```
□ 是否為 B2C 定型化契約？
  → 檢查消保法強制規定，免除/限制條款是否有效

□ 是否為勞動契約？
  → 勞基法最低標準是否均已符合或優於

□ 有無競業禁止條款？
  → 確認勞基法 §9-1 四要件均已滿足

□ 契約性質：一次性 vs. 繼續性？
  → 確認使用「解除」或「終止」用語是否正確

□ 違約金是否顯然過高？
  → B2C：可能消保法無效；B2B：法院仍可酌減

□ 適用法律條款是否指定台灣法？
  → 若無明文，涉外契約依《涉外民事法律適用法》判斷準據法

□ 主管機關是否有針對該產業公告定型化契約應記載事項？
  → 補習、健身、預售屋、電商等特定行業需額外核對
```
