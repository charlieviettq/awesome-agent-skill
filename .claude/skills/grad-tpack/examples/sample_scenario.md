# Example: 高中自然科教師導入 PhET 模擬實驗的 TPACK 評估

## Scenario

**背景**：台中市立光明高中自然科學領域召集人林老師，向課程發展委員會提案：2026 學年起，高一「波動與光」單元改用 PhET Interactive Simulations（科羅拉多大學免費物理模擬平台）取代原有實體光學實驗箱。理由是學校今年新增 35 台 Chromebook，且光學實驗箱耗材費用每年達 8 萬元。

教務主任想了解：這個提案是否具備足夠的教學理據？現有教師準備度如何？需要哪些專業發展支持？

**問題**：「我們有設備，老師也願意嘗試，但怎麼確定這不是『有科技就用科技』？要怎麼評估老師的準備度，並幫他們設計好的整合方式？」

---

## Analysis

### Step 1：知識領域現況盤點

對自然科三位將執行此單元的教師（林老師、陳老師、周老師）進行半結構訪談與教案審閱。

**TK — 技術知識**
- 林老師：熟悉 PhET，曾在研習中使用過，可自行調整模擬參數；陳老師：有基本 Chromebook 操作能力，未用過 PhET；周老師：不熟悉 PhET，對 Chromebook 管理感到焦慮。

**PK — 教學知識**
- 三位皆有探究式教學經驗（108 課綱素養導向），林老師擅長引導討論，陳老師偏向講述後練習，周老師採傳統講述。

**CK — 內容知識**
- 三位皆有物理學位，對波動與光的概念知識紮實；林、陳老師曾指導學生參加物理奧林匹亞。

### Step 2：識別真正的整合機會

**關鍵提問**：「用 PhET 模擬，學生能做到哪些用實體光學箱做不到的事？」

| 能力面向 | 實體光學箱 | PhET 模擬 |
|---------|-----------|-----------|
| 調整波長範圍 | 限於可見光 3 種濾光片 | 可連續調整 380–780 nm |
| 重複實驗速度 | 每組重設約 8 分鐘 | 即時重設，< 5 秒 |
| 觀察干涉條紋 | 受教室光源干擾，難以清晰 | 視覺化清晰，可放大 |
| 即時改變介質折射率 | 不可能 | 可滑桿調整 |
| 安全性 | 雷射安全疑慮 | 無風險 |

**整合機會確認**：PhET 在「波長與顏色關係探究」、「干涉條紋預測與驗證」確有實體器材無法提供的功能，符合 TPACK 的正當性基礎。

**但同時發現潛在風險**：若只是讓學生「看動畫」而非「設計實驗、收數據、解釋結果」，則退化為 TK without TPK — 科技取代教學，而非強化教學。

### Step 3：TPACK 交叉點設計

**目標學習成果**：學生能自行設計變因控制實驗，解釋波長如何決定顏色，並預測干涉條紋間距。

| 元素 | 設計決策 |
|------|---------|
| **CK 驅動**：內容目標 | 理解波長–顏色–折射率三者關係；能計算條紋間距（λ = d·sinθ） |
| **PK 驅動**：教學策略 | 探究式學習（5E 模式）：Engage → Explore → Explain → Elaborate → Evaluate |
| **TK 驅動**：科技角色 | PhET「Wave Interference」模組：學生主動操控變因，非被動觀看 |
| **TPACK 交叉**：整合理據 | PhET 讓學生在 1 節課內完成實體實驗需 3 節才能跑完的參數空間探索，直接服務「變因控制」這個探究能力目標 |

**具體教案片段（40 分鐘）**：
1. Engage（5 min）：教師用 PhET 展示不同波長的彩色條紋，問「哪個變因決定顏色？」
2. Explore（20 min）：學生以 PhET 設計實驗，每組提交「我改變了\_\_，觀察到\_\_」紀錄表
3. Explain（10 min）：班級分享，教師引導公式推導
4. Evaluate（5 min）：Google Forms 即測（預測新波長的條紋間距）

### Step 4：評估與迭代規劃

**教師準備度差距識別**：
- 周老師的 TK 不足需先解決（1 天工作坊即可上手 PhET 基本操作）
- 陳老師需強化 TPK：如何從講述風格轉換到讓學生自主操作；建議配對觀課
- 林老師可作為內部種子教師，帶領同儕備課

---

## Result

# TPACK Analysis: 光明高中「波動與光」PhET 整合評估

## Knowledge Domain Assessment

| Domain | Current State | Evidence | Gap |
|--------|-------------|----------|-----|
| TK | 林：強；陳：中；周：弱 | 訪談 + PhET 操作測試 | 周老師需基礎培訓（1 天） |
| PK | 林：強（探究）；陳：中（講述為主）；周：弱（純講述） | 教案審閱 | 陳、周需 5E 教學設計工作坊 |
| CK | 三位皆強 | 學科背景 + 教學年資 | 無顯著差距 |
| TPK | 林：中（知道可用但設計未成熟）；陳、周：弱 | 教案中無學生操作設計 | 最大差距：PhET 如何服務探究而非取代探究 |
| TCK | 林：中（知道 PhET 能模擬物理現象）；陳、周：弱 | 訪談 | 需展示 PhET 哪些功能對應哪些物理概念 |
| PCK | 林：強；陳：中；周：弱 | 教學年資與教案品質 | 周老師的 PCK 弱是 TPACK 發展的根本限制 |
| TPACK | 林：初步（有意圖但缺結構）；陳、周：無 | 無現成 TPACK 教案 | 整體需設計支持 |

## Technology Integration Design

- **Content goal**：學生能自行操控波長/介質變因，解釋顏色成因並預測條紋間距（對應 108 課綱「探究與實作」能力）
- **Pedagogical strategy**：5E 探究式教學，學生主動設計實驗紀錄，而非觀看示範
- **Technology role**：PhET「Wave Interference」提供實體器材不可能達成的連續參數空間與即時視覺化，直接服務「多變因系統性探索」目標
- **TPACK alignment**：科技選擇由「學生要能獨立控制變因」的 PK 目標驅動，PhET 的即時反饋功能恰好填補實體實驗的時間與安全限制（TCK），三者交叉成立

## Recommendations

**立即行動（2026 學年前）**：
1. 周老師：1 天 PhET 操作培訓（解決 TK 缺口），搭配 PCK 強化課（每月一次跨校共備）
2. 陳老師：配對觀課 — 觀摩林老師一節課，聚焦「如何設計學生操作任務」
3. 全體：共同設計一份標準 PhET 探究學習單，明確要求「學生記錄自己的變因設計」

**結構性警告**：
> 若周老師的 PCK 未改善，TK 培訓無法轉化為有效的 TPACK。建議教務主任在第一學期由林老師主責此單元，周老師擔任觀察員，確認 PCK 底層準備度後再獨立執行。

**評估計畫（2026 學年末）**：
- 使用 Schmidt et al.（2009）TPACK 量表 pre/post 測量教師自評
- 加入一節觀課（評分規準：學生主動操控時間佔比 > 60%）
- 學生學習成果：條紋間距計算正確率目標 > 75%（現行實體實驗歷年平均 58%）
