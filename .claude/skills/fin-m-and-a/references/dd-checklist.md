# Due Diligence 三大類完整清單與紅旗辨識

## DD 的本質

**目的**：驗證賣方陳述、發現隱藏風險、協助估值調整、支援 SPA 條款設計。

**時間窗**：通常 4–8 週（含資料室存取、訪談、報告撰寫）

**團隊**：
- 財稅 DD：會計師事務所
- 法律 DD：律師事務所
- 商業 DD：策略顧問或內部團隊
- 技術／IT DD：專業顧問
- 環安衛：獨立顧問

## 一、財務與稅務 DD

### 1.1 財務報表分析

**核對項目**
- [ ] 過去 3 年經審計財報
- [ ] 會計政策與其一致性
- [ ] 與同業會計政策比較
- [ ] 未經審計期中報表（最近 12 月）
- [ ] 合併範圍與關係人交易

**紅旗**
- 頻繁更換會計師
- 會計政策非一般性變更
- 合併範圍頻繁變動
- 關係人交易未公允揭露

### 1.2 收入品質（Quality of Revenue）

**核對項目**
- [ ] 收入組成（一次性 vs. 經常性）
- [ ] 客戶集中度（前 5、前 10）
- [ ] 合約期限分布
- [ ] 認列政策（時點 vs. 時段）
- [ ] 退款、折讓、保固提列

**紅旗**
- 一次性收入 > 10% 且未充分揭露
- 前三大客戶 > 50%
- 合約期末集中認列
- 退貨率異常下降（可能未提）

### 1.3 EBITDA 正常化

**典型調整項**
- (+) 一次性費用：訴訟、重組、關廠、併購費用
- (+) 非經常性損失：天災、意外
- (−) 非經常性收入：補助、處分利益
- (+/−) 關係人交易差距
- (+/−) 股東個人費用（家族企業常見）
- (+/−) 未市場化的租金或薪酬
- (+/−) 會計政策差異（LIFO vs. FIFO）

**實務比例**
- 正常化 EBITDA 通常比帳上 EBITDA 低 5–15%
- 調整超過 20% 需特別解釋
- 家族企業調整項通常較多

### 1.4 營運資金（Working Capital）

**核對項目**
- [ ] 應收帳款帳齡分析
- [ ] 應付帳款天數
- [ ] 存貨週轉、呆滯分析
- [ ] 季節性變動
- [ ] 正常化營運資金水位

**紅旗**
- AR 天數 > 產業中位數 + 30%
- AP 天數 < 產業中位數 − 30%（供應商壓力）
- 存貨天數持續上升
- 期末衝業績造成營運資金異常

**交割日營運資金調整（Working Capital Adjustment）**
```
Target WC = 過去 12 月平均
交割日實際 WC vs. Target → 差額調整交易對價
```

### 1.5 稅務 DD

**核對項目**
- [ ] 過去 5 年所得稅申報
- [ ] 其他稅（營業稅、關稅、印花稅）
- [ ] 稅捐機關稽查記錄
- [ ] 移轉定價文件（跨境）
- [ ] 關稅分類適正性
- [ ] 遞延所得稅資產評估

**紅旗**
- 有效稅率大幅低於法定稅率
- 境外所得比例高但未揭露
- 移轉定價文件缺漏
- 過去有稅務爭議紀錄
- 跨境集團架構涉稅天堂

### 1.6 負債與或有負債

**核對項目**
- [ ] 全部借款合約（含契約限制）
- [ ] 租賃負債（IFRS 16）
- [ ] 退休金精算報告
- [ ] 擔保、背書、保證
- [ ] 訴訟或有負債評估
- [ ] 環安衛清償義務

**紅旗**
- 關係人擔保高
- 退休金提撥不足
- 重大訴訟未依 IFRS 判斷揭露
- 環安衛潛在清償未反映財報

## 二、法律 DD

### 2.1 公司組織

**核對項目**
- [ ] 公司章程、登記資料
- [ ] 股東協議、特別股條款
- [ ] 董事會運作、議事錄
- [ ] 子公司架構、持股關係
- [ ] 家族控股結構（家族企業）
- [ ] 員工持股計畫（ESOP）

**紅旗**
- 股東協議有否決權條款（交易需特別同意）
- 章程有股份轉讓限制
- 子公司持股比例低於 100% 造成整合障礙

### 2.2 重大合約

**核對項目（分類）**
- 客戶合約：前 20 大
- 供應商合約：前 10 大
- 授權合約（技術、商標、版權）
- 通路合約（經銷、代理）
- 合資、策略聯盟合約
- 融資合約
- 不動產租賃／買賣
- 重大資產交易

**Change of Control（CoC）條款辨識**
- 是否自動終止？
- 是否需對方同意？
- 是否觸發加速條款（如債務加速到期）？

**紅旗**
- 大客戶合約含 CoC 條款
- 關鍵供應商合約含 CoC 條款
- 融資合約加速到期條款
- 授權合約不得轉讓

### 2.3 訴訟與爭議

**核對項目**
- [ ] 已發生訴訟清單（原告、被告）
- [ ] 金額、階段、律師意見
- [ ] 可能未來訴訟（客戶、員工、政府）
- [ ] 仲裁、調解進行中案件
- [ ] 行政處分紀錄

**或有負債量化**
```
期望損失 = 索賠金額 × 敗訴機率
```

通常以律師意見為參考：
- 極可能敗訴：機率 > 70%
- 可能敗訴：30–70%
- 不太可能：< 30%

### 2.4 智慧財產權

**核對項目**
- [ ] 專利（申請中、核准、即將到期）
- [ ] 商標（各地域註冊）
- [ ] 著作權
- [ ] 營業秘密與保密措施
- [ ] 授權合約（授出與取得）
- [ ] 侵權警告信或訴訟

**紅旗**
- 核心專利即將到期
- 核心技術未申請專利（營業秘密脆弱）
- 前員工可能帶走技術
- 商標在關鍵市場未註冊

### 2.5 勞動關係

**核對項目**
- [ ] 員工名冊（含關鍵人員）
- [ ] 僱傭合約、競業禁止
- [ ] 退休金計畫（DB vs. DC）
- [ ] 工會狀況、團體協約
- [ ] 勞動訴訟、爭議
- [ ] ESOP 行權狀態

**紅旗**
- 工會團體協約近期到期
- 未提撥退休金
- 關鍵人員無競業禁止
- 集體勞動爭議歷史

### 2.6 法規合規

**核對項目**
- [ ] 營業登記、執照有效性
- [ ] 環安衛合規紀錄
- [ ] 資安事件揭露
- [ ] 反貪腐、反壟斷合規
- [ ] 個資法合規（GDPR、PDPA）
- [ ] 出口管制合規
- [ ] 產業特定法規（金融、醫療、食品）

**紅旗**
- 近 3 年有重大行政處分
- 環安衛違規紀錄
- 資安事件未揭露
- 跨境交易觸及出口管制

## 三、商業與營運 DD

### 3.1 市場地位

**核對項目**
- [ ] 市場規模與成長率
- [ ] 市占率（目標 + 競爭者）
- [ ] 市場區隔與目標
- [ ] 產業生命週期位置
- [ ] 進入障礙

**紅旗**
- 市占率持續下滑
- 產業衰退期
- 進入障礙快速降低

### 3.2 客戶分析

**核對項目**
- [ ] 客戶集中度（前 5、前 10）
- [ ] 客戶關係年資
- [ ] 客戶流失率（Churn）
- [ ] 客戶獲利性分層
- [ ] NPS、滿意度

**紅旗**
- 前 3 大客戶 > 50%
- 關鍵客戶合約即將到期
- Churn rate 高
- NPS 下降趨勢

### 3.3 競爭分析

**核對項目**
- [ ] 主要競爭者 profile
- [ ] 競爭優勢來源
- [ ] 替代品威脅
- [ ] 新進者威脅
- [ ] 供應商議價力

### 3.4 產品／服務

**核對項目**
- [ ] 產品組合與生命週期
- [ ] 研發管線
- [ ] 品質控制
- [ ] 認證與驗證
- [ ] 客戶申訴分析

### 3.5 供應鏈

**核對項目**
- [ ] 關鍵供應商依賴度
- [ ] 原料供應穩定性
- [ ] 物流安排
- [ ] 庫存政策
- [ ] 備用供應源

**紅旗**
- 單一供應商佔 > 40%
- 關鍵原料無備案
- 運輸依賴單一路徑

### 3.6 IT 系統與資安

**核對項目**
- [ ] 核心系統清單（ERP、CRM、MES）
- [ ] 技術債務評估
- [ ] 資安政策與歷史事件
- [ ] 個資保護措施
- [ ] 災備計畫

**紅旗**
- 系統老舊需大規模更新
- 資安歷史事件多
- 個資保護不合規
- 無災備

### 3.7 管理團隊

**核對項目**
- [ ] 關鍵人員名單與經歷
- [ ] 接班規劃
- [ ] 薪酬結構
- [ ] 留任風險評估
- [ ] 內部人際關係

### 3.8 組織文化

**核對項目**
- [ ] 文化調查結果
- [ ] 流動率與投入度
- [ ] 創新指標
- [ ] 跨部門協作
- [ ] 與買方文化差距評估

## 紅旗總表與處理

### 紅旗等級

| 等級 | 處理 |
|---|---|
| 🔴 Deal Breaker | 終止交易或重大條件 |
| 🟠 重大調整 | 估值下修、特別保證 |
| 🟡 條款處理 | SPA 特殊條款 |
| 🟢 注意觀察 | 整合期注意 |

### 常見 Deal Breaker 清單

1. 財務詐欺或重大不實陳述
2. 違反反貪腐（FCPA、UK Bribery Act）
3. 重大環境污染未揭露
4. 核心技術為盜用
5. 關鍵資產無法移轉
6. 反壟斷審查通不過

### 從 DD 到 SPA 的轉化

```
DD 發現 → 估值調整（反映在對價）
       → R&W（陳述與保證）
       → 特別保證（Specific Indemnity）
       → 先決條件（Condition Precedent）
       → 整合計畫（PMI 輸入）
```

**原則**：DD 發現必須有處理路徑，不能發現後不處理。

## DD 報告結構建議

```
1. 執行摘要（Executive Summary）
   - 主要發現 TOP 5
   - Deal Breaker 判定
   - 對估值與條款的影響

2. 各類別詳細發現
   - 財稅（含 EBITDA 正常化表）
   - 法律（含重大合約與訴訟清單）
   - 商業營運（含市場、客戶、供應鏈）

3. 整合風險評估
   - 關鍵人才
   - 文化相容性
   - IT 與流程整合

4. 建議
   - 估值調整
   - SPA 條款建議
   - 整合關鍵議題

5. 附件
   - 詳細資料
   - 訪談紀錄
   - 外部資料引用
```
