# Example: 計程車業者遊說封鎖網約車平台立法

## Scenario

2023 年初，台灣某縣市議會正在審議一項「網路叫車平台管理自治條例草案」，草案核心條款要求：

- 平台須持有縣市核發的「平台業者許可證」（費用 NT$500 萬，每年更新）
- 每輛加入平台的車輛須額外繳交「安全管理基金」NT$12,000／年
- 每日尖峰時段（7–9am、5–8pm）平台可派遣車輛數不得超過傳統計程車總量的 30%

縣計程車商業同業公會（代表約 8,000 輛持牌計程車）積極支持此草案，並提供議員競選獻金共 NT$3,200 萬。反對方（Uber Taiwan、台灣大車隊旗下叫車 app）主要透過媒體公關，未直接遊說個別議員。政策研究者想用公共選擇理論評估：這套草案通過後的社會成本，以及為何理性議員會支持它？

---

## Analysis

### Step 1 — 識別政治市場

| 角色 | 最大化目標 | 關鍵約束 |
|------|-----------|---------|
| 縣市議員（27 席） | 選票極大化（連任機率） | 四年任期；需要競選資金 |
| 計程車公會 | 保護既有租值（牌照溢價） | 組織成本由 8,000 名會員分攤 |
| 網約車平台 | 市場份額與利潤 | 分散的使用者難以集體行動 |
| 乘客（縣市人口約 120 萬） | 低價、方便的運輸 | 理性無知：每票影響力趨近於零 |
| 縣府監理機關 | 預算規模、裁量權擴張（Niskanen） | 中央法規框架 |

**關鍵不對稱**：公會 8,000 人，集中利益；乘客 120 萬人，利益分散。Olson 的集體行動邏輯預測：即使草案讓乘客每人每年多付 NT$800（約 NT$9.6 億總損失），單一乘客的動員誘因接近零，組織阻力幾乎不存在。

---

### Step 2 — 套用模型

#### 模型 A：中位數選民定理的侷限

單一政策維度（「管制強度」）下，若選民偏好單峰，中位數選民應偏好溫和管制。然而本案涉及多維度（安全、就業保護、消費者福利），偏好排序無法線性化，產生 Arrow 循環：  
- 多數支持「安全 > 方便」  
- 多數支持「就業保護 > 安全成本」  
- 多數支持「低票價 > 就業保護」  

→ 無穩定多數均衡，議員有裁量空間選擇符合自身利益的任一聯合。

#### 模型 B：租值追尋（Tullock, 1967）

計程車牌照在管制保護下的溢價估算：

```
傳統計程車牌照市場價格（2022 年均值）：NT$150,000／張
無溢價的合理車輛殘值（無牌照保護）：NT$30,000／張
牌照租值（每張）：NT$120,000
全縣總租值：NT$120,000 × 8,000 張 = NT$9.6 億
```

公會願意花費以保護此租值：

```
遊說支出（競選獻金 + 公關）：NT$3,200 萬
占總租值比率：3,200 萬 / 96,000 萬 ≈ 3.3%
```

> Tullock 矩形：NT$9.6 億的租值被競爭性遊說部分耗散。若對手平台也投入同等資源回擊，社會浪費的資源總和趨近租值本身（此處因平台遊說不足，耗散仍偏低，但消費者剩餘損失已發生）。

#### 模型 C：票倉交換（Logrolling）

草案在議會表決前，公會議員以支持某都市更新容積獎勵案（主要惠及不動產開發商支持的議員）換取後者對網約車草案的支持票。每項議案單獨看都無法過半，票倉交換使兩者均通過——典型的少數互利立法，多數乘客（對兩項政策均輕微反對）被排除在外。

---

### Step 3 — 估算政府失靈成本

| 成本類別 | 估算說明 | 金額（NT$） |
|---------|---------|------------|
| 租值追尋支出（公會） | 獻金 + 公關活動 | 3,200 萬 |
| 租值追尋支出（平台） | 媒體公關、法律費用 | 800 萬（估） |
| 消費者剩餘損失（票價提高） | 派遣上限 30% → 尖峰供給不足 → 票價漲估 15% → 120 萬人均多付 NT$800/年 | 9.6 億／年 |
| X 低效率（許可證審查機關） | 新增業務編制 8 人、行政成本 | 480 萬／年 |
| 總靜態社會成本（首年） | | **≈ NT$10.1 億** |

**對照市場失靈**：若網約車確實有安全外部性（事故增加），估算年事故成本增加約 NT$4,000 萬（縣府交通局自身報告）。  

→ 政府介入矯正 NT$4,000 萬的市場失靈，卻引入 NT$10.1 億的政府失靈，**淨社會損失約 NT$9.7 億**。

---

### Step 4 — 制度改革建議

現行草案設計的核心問題是監管權力集中於單一機關，且無競爭壓力。建議替代方案：

1. **以績效替代配額**：取消派遣上限，改以事故率、乘客評分設定最低標準，不達標者吊照。消除供給扭曲，保留安全誘因。  
2. **許可費改精算制**：以實際監理成本為上限，公開會計，防止機關預算極大化。  
3. **落日條款**：自治條例 3 年後自動失效，需重新舉證市場失靈方可展延，逆轉「默認管制存在」的政治慣性。  
4. **Tiebout 競爭壓力**：鼓勵鄰近縣市開放競爭，使消費者可跨縣使用服務，削弱單一地方政府的租值保護能力。

## Result

```markdown
## Public Choice Analysis: 網約車平台管理自治條例草案

### Political Actors
| Actor           | Objective             | Key Constraint         |
|----------------|-----------------------|------------------------|
| 縣市議員        | 連任票數極大化         | 競選資金依賴公會等集中利益方 |
| 計程車公會      | 保護 NT$9.6 億牌照租值 | 組織成本由 8,000 會員平攤 |
| 網約車平台      | 市場份額與利潤         | 使用者分散，難以轉化為政治壓力 |
| 縣市乘客        | 便宜、便利的運輸       | 理性無知：單票影響力 ≈ 0 |
| 縣府監理機關    | 預算與裁量權擴張        | 中央法規框架 |

### Model Applied
- **Framework**: 租值追尋（主）+ Logrolling（次）+ Niskanen 機關預算極大化
- **Prediction**: 公會以 NT$3,200 萬遊說保護 NT$9.6 億租值，理性議員接受獻金並藉票倉交換通過草案；乘客因理性無知不組織反對
- **Observed outcome**: 草案以 17:10 通過一讀，符合公共選擇預測；反對票來自都市選區（網約車使用率較高）議員

### Government Failure Costs
| Cost Category          | Estimate / Description |
|-----------------------|----------------------|
| 租值追尋支出（雙方合計） | NT$4,000 萬（首年）|
| 消費者剩餘損失（供給扭曲） | NT$9.6 億／年 |
| 行政 X 低效率 | NT$480 萬／年 |

### Market Failure vs. Government Failure
- **Market failure being addressed**: 網約車安全外部性（縣府估 NT$4,000 萬／年事故成本）
- **Government failure introduced**: 租值追尋耗散 + 消費者剩餘損失共 NT$10.1 億／年
- **Net assessment**: 介入**惡化**社會福利；淨損失約 NT$9.7 億／年。政策表面為「安全管理」，實質為牌照保護法案。

### Institutional Recommendations
1. **以績效標準取代數量配額**：刪除 30% 派遣上限，改設事故率門檻（如每百萬趟次事故數不得高於計程車業均值的 1.2 倍），超標停牌。
2. **許可費精算公開化**：費用以實際監理成本為上限並每年公告，防止機關預算極大化。
3. **3 年落日條款**：要求重新舉證市場失靈才能展延，打破「默認管制存在」的政治均衡。
4. **跨縣服務開放**：允許持鄰近縣市許可的平台跨境接單，引入 Tiebout 競爭，削弱地方保護主義誘因。
```
