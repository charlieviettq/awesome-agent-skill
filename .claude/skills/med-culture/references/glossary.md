# Culture & Arts Glossary — Key Terms & Distinctions

## 博物館與美術館

| 術語 | 定義 | 報導使用場景 |
|------|------|------------|
| **典藏 vs 收藏** | 典藏:機構永久保存且編目的文物 / 收藏:個人或私人機構的物品集合 | 報導古蹟/文物時,「國立故宮典藏」(機構正式) vs 「私人收藏家的乾隆瓷」(非正式) |
| **借展 vs 駐館** | 借展:短期交流(通常 3-6 月) / 駐館:藝術家長期進駐(駐村、客座,通常 1-2 年) | 特展新聞區分期限與性質 |
| **策展人 vs 館員** | 策展人:策劃展覽內容與敘事 / 館員:日常運營 | 採訪時區分角色,策展人負責敘述框架,館員負責執行 |
| **開放典藏 vs 常設展** | 開放典藏:機構開放讀者查閱但不常態展示的物件 / 常設展:永久展廳(通常一年以上) | 學術研究 vs 大眾參觀的管道不同 |

## 藝術與設計

| 術語 | 定義 | 報導使用場景 |
|------|------|------------|
| **現代主義 vs 後現代 vs 當代** | 現代(~1970s):邁向抽象、純粹性 / 後現代(1970s-1990s):解構、消費文化反思 / 當代(1990s~):身份、議題導向,多元美學 | 評論時需位置藝術家與作品的時代思潮;避免混淆 |
| **裝置 vs 雕塑 vs 新媒體** | 裝置:與環境互動,可能包含聲音/光影/觀眾參與 / 雕塑:獨立物件,強調形式 / 新媒體:數位、影像、AI 等科技媒材 | 描述作品時精確使用,暗示藝術家的美學意圖 |
| **評論 vs 批評** | 評論:呈現與分析 / 批評:包含評價與立場(通常帶有批判性) | 「藝評」(評論) vs 「批評」(更強的規範意圖) |
| **大師 vs 新興藝術家** | 大師:有歷史地位、市場確認的藝術家 / 新興:初出茅廬或初獲機構重視者 | 避免濫用「大師」;市場價格≠大師地位 |

## 文化資產與遺產

| 術語 | 定義 | 報導使用場景 |
|------|------|------------|
| **古蹟 vs 遺址 vs 聚落 vs 紀念建築** | 古蹟:人為建造物,50 年以上且有文化價值 / 遺址:史前或歷史遺留物 / 聚落:有集體文化意義的村落/街區 / 紀念:為紀念人物/事件而指定 | 法律分類影響保存措施;報導時需精確區分 |
| **文化資產 vs 文化遺產 vs 無形資產** | 資產:廣義文化財產 / 遺產:世代傳承 / 無形:傳統工藝、民俗、口傳知識等非物質文化 | 「世界文化遺產」通常指聯合國認定的文化資產 |
| **保護 vs 復振 vs 傳承** | 保護:維持現狀(不改變) / 復振:復原失傳/衰微的文化 / 傳承:持續活態傳授(如民俗藝能) | 報導原民文化時區分:是保存古蹟?還是支持活態傳承? |
| **口傳文化 vs 書面文化** | 口傳:知識通過言語代代相傳,非書寫記錄 / 書面:文字記錄與典籍 | 原住民族知識多為口傳;報導須尊重來源形式 |

## 表演藝術

| 術語 | 定義 | 報導使用場景 |
|------|------|------------|
| **編舞 vs 編劇 vs 導演** | 編舞:舞蹈編排 / 編劇:劇本創作 / 導演:整體舞台演出指揮 | 採訪時區分創意領導與執行職位 |
| **劇場 vs 戲劇 vs 表演** | 劇場:物理空間+製作系統 / 戲劇:敘事內容 / 表演:藝術呈現方式(含舞蹈/音樂/裝置演出) | 「去劇場看演出」(場景) vs 「看戲劇」(內容) |
| **駐村 vs 客座 vs 駐地創作** | 駐村:藝術家駐進機構/社區創作 / 客座:客席演出或教學 / 駐地:在特定地點進行長期創作計畫 | 機構宣傳常混用;報導時需查證實際合約與期限 |
| **技術劇場 vs 劇場設計** | 技術:舞台技術與燈光/音響執行 / 設計:視覺概念與美學呈現 | 評論時分別涉及技術執行能力 vs 美學創意 |

## 文化政策與補助

| 術語 | 定義 | 報導使用場景 |
|------|------|------------|
| **補助 vs 獎助 vs 投資** | 補助:政府/基金會支持特定計畫(不計報酬) / 獎助:表揚優秀(通常含獎金) / 投資:預期經濟報酬(商業模式) | 報導文化經費時區分性質,暗示政府承諾與期待 |
| **評等 vs 分級** | 評等:相對排序(高/中/低或 1-5 分) / 分級:絕對分類(A/B/C 等) | 補助審查與文化資產分級的結果表述不同 |
| **文化部門 vs 文創產業** | 文化部門:政府文化預算與政策 / 文創產業:商業化文化產品(設計、動畫、遊戲等) | 避免混淆「文化推廣」與「文創商品化」 |
| **多元文化 vs 文化多樣性** | 多元文化:共存不同文化群體 / 文化多樣性:強調各文化的獨特價值與保護 | 「保護少數族群文化多樣性」(尊重) vs 「多元文化融合」(可能淡化差異) |

## 原住民族相關

| 術語 | 定義 | 報導使用場景 |
|------|------|------------|
| **原住民族 vs 少數民族 vs 原民** | 原住民族:台灣官方用詞(16 族) / 少數民族:國際用詞(可含移民) / 原民:簡稱 | 台灣報導使用「原住民族」為正式;「原民」可用於通俗語脈 |
| **部落 vs 鄉鎮 vs 聚落** | 部落:原住民族社群與文化組織單位 / 鄉鎮:行政單位 / 聚落:地理單位 | 「回到部落」(文化單位) vs 「返回鄉鎮」(行政單位) — 意義不同 |
| **傳統領域 vs 現住地** | 傳統領域:部落歷史上使用的地區 / 現住地:當代實際住所 | 土地與文化權議題常涉及兩者的差異 |
| **口傳歷史 vs 書寫歷史** | 口傳:部落世代相傳的知識與敘述 / 書寫:國家檔案與學術記錄 | 報導時須併列:「部落說法是...、學術研究則...」,平等呈現 |
| **文化回應 vs 文化挪用** | 回應:文化群體對外部詮釋的反應與協商 / 挪用:未經許可的使用與改寫(通常帶權力不對等) | 區分是「開放交流」還是「未得同意的使用」 |

## 跨文化與翻譯

| 術語 | 定義 | 報導使用場景 |
|------|------|------------|
| **翻譯 vs 改編 vs 詮釋** | 翻譯:將文本轉換至另一語言(力求等效) / 改編:改變形式或媒材 / 詮釋:重新解讀(可含創意變異) | 「這部作品是原著改編」(有改動) vs 「翻譯」(尋求原意) |
| **文化脈絡 vs 國際語言** | 脈絡:特定社會文化背景 / 國際語言:超越地方的通用美學(如「當代藝術」) | 評論時註明:是從地方文化脈絡理解,或套用國際美學標準? |
| **內部視角 vs 外部觀察** | 內部:社群成員的理解 / 外部:局外人觀察 | 「據部落內部人士說...」vs 「人類學家觀察...」—位置不同,權威性不同 |
