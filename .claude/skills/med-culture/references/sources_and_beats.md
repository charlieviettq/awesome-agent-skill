## 藝術與批評媒體

| 類型 | 代表性媒體 | 特色 |
|------|----------|------|
| **國際藝術評論** | ArtForum, Frieze, Hyperallergic, Artnet | 當代藝術國際視角,行業意見領袖 |
| **台灣藝術評論** | 《當代藝術評論季刊》、典藏藝術家庭、藝術很有事、藝術新聞 | 台灣藝術生態、評論與產業消息 |
| **美術館/畫廊新聞** | 各館官網新聞稿、策展自述、展覽評論合作出版物 | 機構官方立場、策展理論 |
| **策展人/評論家個人媒體** | 部落格、線上評論專欄、Matters、Medium | 獨立觀點、長形分析 |
| **表演藝術媒體** | 表演藝術評論台 PAREVIEWS、聯合報副刊、藝術新聞(劇場版) | 戲劇、舞蹈、音樂評論 |

## 機構與策展人來源 (Tier 2: Institutional Voice — 標記 PR 利益)

- **大型美術館策展人群**: 國立台灣美術館、國立現代美術館、高美館、台北美術館等策展團隊 — 有推廣展覽的利益,但策展論述值得參考
- **獨立策展人**: 無機構背景的策展人群(通常透過文化部補助計畫),其觀點較少受機構預算/PR 壓力影響
- **商業畫廊策展**: 藝廊代理藝術家,有銷售誘因;引用時標記「XX 藝廊代理藝術家」

## 獨立評論與專家 (Tier 3: Independent Analysis)

- **學者與美術史研究者**: 大學藝術系所、中研院、智庫
- **自由評論家**: 藝術雜誌、新聞媒體的專欄評論家(附帶出版物背景與評論立場)
- **藝術家與創作者**: 同行評價(需標註利益關係:競爭對手/友人/合作者)
- **民間藝術組織**: 非營利藝術倡議団體、藝術家自主聯誼會

## 特殊領域來源

### 文化資產與古蹟
- **文資審議委員會記錄** (公開)
- **古蹟保存再利用案例** (各地文資局案例庫)
- **UNESCO 世界遺產認定資料**
- **國際古蹟保存組織** (ICOMOS, ICCROM)

### 原住民族文化
- **部落文化組織與議會** (地方層級決策單位)
- **原住民族學術研究** (中研院民族所、大學原民研究中心)
- **原住民族委員會文化科**: 傳統知識保護、文化補助、語言推動
- **原民媒體與評論家** (獨立原民評論者、原民自媒體)

### 表演藝術
- **國家表演藝術中心** (國家劇院、音樂廳、兩廳院)
- **台灣戲劇協會、音樂學會等職業公會**
- **表演藝術評論家協會或聯誼組織**
- **各表演場館(劇院、音樂廳、實驗場地)的駐村/客座計畫** (通常有官方公告)

### 藝術市場
- **拍賣公司** (Sotheby's, Christie's Taiwan 分公司; 在地拍賣行) — 注意:估價與成交價有銷售利益
- **藝術基金與投資機構** — 標記利益衝突
- **國際藝博會** (ART TAIPEI, ART FAIR TOKYO 等) — 市場趨勢信號
- **美術館/畫廊銷售統計** (較少公開披露)

## 採訪 Beat 配置 (典型分工)

| Beat | 主要場域 | 必訪對象 |
|------|----------|---------|
| 展覽/美術館線 | 美術館、畫廊、策展人工作室 | 策展人、美術館館長/學藝組、參展藝術家、獨立評論家 |
| 表演藝術線 | 劇院、音樂廳、實驗場地、舞蹈工作坊 | 導演/編舞家、演員、場館藝術總監、表演藝術評論家 |
| 文化政策線 | 文化部、各縣市文化局、原委會、文資局 | 政策官員、補助計畫承辦人、政策研究者 |
| 文化資產線 | 古蹟現場、文資局、在地社群組織 | 文資保護工作者、居民、地方政府、建築師 |
| 藝術市場線 | 拍賣公司、畫廊、國際藝博會 | 藝術經紀人、收藏家、拍賣官、市場分析師(標記利益衝突) |

## 何時應該打給多少人

| 報導類型 | 最少訪問數 | 特殊需求 |
|---------|----------|--------|
| 即時展覽新聞 (< 500字) | 1 名(美術館 / 策展人)+ 公開資料 | 機構發言人聲音足夠 |
| 展覽報導 + 評論 (1000-1500字) | 策展人 + 獨立評論家 + 藝術家 (3+) | 機構 vs. 獨立聲音的平衡 |
| 文化政策追蹤 | 政策主管 + 受影響藝術家 / 機構 + 政策研究者 (3+) | 政策方 + 執行方 + 分析 |
| 文化資產報導 | 保護工作者 + 在地社群 + 文資官員 (3+) | **必須包含受影響社群的聲音** |
| 原住民族文化報導 | 部落文化代表 + 藝術家 + 學者 (3+) | **必須尋求部落同意與聲音,避免局外代言** |
| 藝術市場分析 | 3+ 市場參與者(經紀人/評論家/研究者) | **標記每位來源的經濟利益** |
