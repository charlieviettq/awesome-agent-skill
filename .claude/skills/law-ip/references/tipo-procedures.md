# 台灣 TIPO 智慧財產局申請程序

TIPO = 經濟部智慧財產局 (Taiwan Intellectual Property Office)  
官方網站：https://www.tipo.gov.tw

---

## 商標申請程序（最優先處理）

商標申請是大多數新創公司最應該先辦的 IP 行動：費用低、速度快、保護重要。

### 申請流程

```
1. 名稱/圖樣可行性搜尋（1–3天）
        ↓
2. 準備申請文件（1–3天）
        ↓
3. 線上提交申請 + 繳規費（1天）
        ↓
4. TIPO 形式審查（1–3個月）
        ↓
5. 實體審查：識別性、混淆近似性（3–12個月）
        ↓
6. 核准公告（3個月異議期）
        ↓
7. 領取商標證書 → 保護生效
```

總耗時：約 12–18 個月（無異議情況）

### 規費（2024年基準）

| 項目 | 金額 |
|------|------|
| 每件每類申請費 | NT$3,000 |
| 第2類起每件每類 | NT$2,500 |
| 加速審查（快速審查） | 另加 NT$4,000 |
| 每10年更新費 | NT$3,000/類 |

> **實務建議**：一個品牌名稱至少申請 2–3 類（主要業務類別）。費用通常 NT$6,000–9,000 即可取得初步保護。

### 尼斯分類（選類關鍵）

必須選對類別，否則其他類別無法主張權利。

常見錯誤：SaaS 新創只申請第 42 類（軟體服務），卻忘了第 35 類（商業管理服務）或第 38 類（通訊服務），造成保護缺口。

| 業務類型 | 建議申請類別 |
|---------|------------|
| SaaS / 軟體服務 | 第 9 類（軟體）、第 35 類（商業服務）、第 42 類（技術服務）|
| 電商平台 | 第 35 類、第 38 類、第 42 類 |
| 餐飲品牌 | 第 43 類（餐廳服務）、第 30 類（食品） |
| 製造業產品 | 視產品選對應商品類別 + 第 40 類（製造服務）|

### 識別性要求（核准關鍵）

TIPO 依識別性強弱審查，遵循以下層級：

```
強 ──→ 弱
臆造詞（KODAK）→ 任意詞（Apple 用於電腦）→ 暗示詞（Netflix）→ 描述性詞（COLD AND CREAMY）→ 通用詞（電腦）
           ✅ 容易核准                                              ❌ 通常駁回
```

**描述性商標補救**：若商標描述商品本身特性，TIPO 可能以「識別性不足」駁回。補救方式：
1. 修改圖樣增加圖形元素
2. 主張後天識別性（使用證明：廣告支出、銷售量、市場調查）
3. 限縮申請範圍

## 專利申請程序

### 三種專利類型比較

| 類型 | 保護對象 | 審查方式 | 年限 | 申請費（基本）|
|------|---------|---------|------|--------------|
| 發明專利 | 技術思想、方法、物品 | 實體審查（需申請） | 20年 | NT$9,000 |
| 新型專利 | 物品的形狀、構造 | 形式審查（快速） | 10年 | NT$3,500 |
| 設計專利 | 物品的外觀造型 | 實體審查 | 15年 | NT$8,000 |

### 發明專利申請流程

```
1. 先前技術搜尋（Patent Search）
        ↓
2. 撰寫說明書 + 申請專利範圍（Claims）
        ↓
3. 提交申請 → 取得申請日（filing date）← 最關鍵時間點
        ↓
4. 申請實體審查（自申請日起3年內，需另繳費）
        ↓
5. TIPO 審查委員審查（12–24個月）
        ↓
6. 核駁通知 or 核准
        ↓ （若核駁）
7. 答辯 / 修正申請專利範圍
        ↓
8. 核准 → 領證 → 繳年費維持
```

### 申請日的重要性（first-to-file）

台灣採**申請主義**，不是發明主義：

```
情境：A 在 2024/01 發明，未申請
      B 在 2024/06 獨立發明，2024/07 提出申請

結果：B 取得專利。A 即使能證明先發明，也無法取得專利。
```

**因此**：發明完成後，應在公開揭露之前提交申請。若必須先公開（如展覽、論文），需在公開後 **12 個月內**（台灣優惠期）提出申請，並主張優惠期。

### 申請專利範圍（Claims）撰寫原則

TIPO 的審查核心是 Claims，而非說明書本文。Claims 決定保護範圍。

**獨立項 vs 附屬項**：
```
獨立項（Independent Claim）：自成一體，定義最寬保護範圍
附屬項（Dependent Claim）：引用獨立項並進一步限縮，作為後備

Example：
Claim 1（獨立）：一種資料壓縮方法，包含步驟A、步驟B...
Claim 2（附屬）：如請求項1之方法，其中步驟A進一步包含...
```

若獨立項被無效，附屬項可能仍然有效——多層次保護。

### 實體審查費（依申請項數計算）

| 項數 | 費用 |
|------|------|
| 第1–10項 | 每項 NT$800 |
| 第11–20項 | 每項 NT$1,200 |
| 第21項以上 | 每項 NT$1,600 |
| 說明書超過 50頁（每頁）| NT$500 |

> 建議：一般發明申請 10–20 個 claims，總實體審查費約 NT$8,000–20,000，加上前期申請費，總成本約 NT$17,000–30,000（不含代理人費用）。

### 年費維持表（發明專利）

| 年度 | 年費 |
|------|------|
| 第1–3年 | 免繳（含在申請費內）|
| 第4–6年 | 每年 NT$2,500 |
| 第7–9年 | 每年 NT$5,000 |
| 第10年以上 | 每年 NT$8,000 |

逾期未繳 → 6個月內可補繳（加收倍數年費）→ 超過則專利消滅。

## 新型專利（快速保護路徑）

若保護對象是**具體物品的結構**（非純方法或純軟體），新型專利是更快的選擇：

- 只做形式審查，**不做實體審查**
- 約 **4–6 個月**可取得證書（vs 發明專利 2–3 年）
- 代價：權利不穩定，競爭對手可申請技術報告挑戰有效性

**實務策略（雙軌申請）**：
```
同一發明同時申請 發明專利 + 新型專利
→ 新型專利先核准 → 立即可主張權利、嚇阻競爭對手
→ 發明專利核准後 → 主動拋棄新型專利，改主張發明專利（保護期更長）
```
注意：兩者不可同時存在，發明專利核准時必須擇一。

## 著作權登記（非必要，但有用）

台灣著作權**自動產生**，無需登記。但 TIPO 提供自願登記服務，用途：

| 用途 | 是否需要登記 |
|------|------------|
| 主張著作權存在 | 不需要（自動） |
| 授權或轉讓 | 不需要（契約即可） |
| 訴訟中作為初步證明 | 登記有助舉證 |
| 海關查扣（邊境保護）| 需要登記 |

### 著作權登記費用

| 項目 | 費用 |
|------|------|
| 電腦程式著作登記 | NT$1,000 |
| 其他著作登記 | NT$1,000 |
| 著作權授權登記 | NT$500 |

## 營業秘密（不透過 TIPO 申請）

營業秘密**沒有申請程序**——保護完全靠內部管理。TIPO 不受理登記。

合法保護要件（須同時具備）：

```
1. 秘密性：非公眾所知
2. 經濟價值：因保密而具有商業價值
3. 合理保護措施：已採取保密行動
```

**「合理保護措施」的具體作法**（法院認可的標準）：

| 措施 | 具體做法 |
|------|---------|
| 契約保密 | 員工 NDA、廠商保密合約 |
| 存取控制 | 系統權限分級、需要知道原則 |
| 實體安全 | 伺服器機房門禁、文件上鎖 |
| 標記保密 | 文件標示「機密」或「Confidential」|
| 離職管理 | 離職程序包含提醒保密義務、設備回收 |

> 若未採取合理保護措施，即使資訊被竊取，也可能無法主張營業秘密侵害。

## 優先權申請（國際佈局）

### 巴黎公約優先權

若已在台灣（或其他 WTO 成員國）申請，可在 **12 個月內**（商標為 6 個月）在其他成員國主張優先權：

```
台灣申請日：2024/01/15（優先權基礎日）
美國申請截止：2025/01/15（+12個月）
主張優先權效果：視為 2024/01/15 在美申請
```

**重要**：即使在此期間競爭對手在美國提出申請，你的優先權日仍早於他。

### PCT 途徑（發明專利國際申請）

| 階段 | 時間 | 說明 |
|------|------|------|
| 提交 PCT 申請 | 優先權日起 12 個月內 | 向 TIPO 或 WIPO 提交 |
| 國際公布 | 申請後 18 個月 | 全球公開 |
| 進入各國國家階段 | 優先權日起 30 個月內 | 選擇目標國家 |

PCT 不直接產生「國際專利」——它只是延後在各國分別申請的時間和成本，讓你有更多時間評估市場。

## 線上申請系統

TIPO 電子申請系統（eApply）支援全程線上申請：

- 商標：https://tmonline3.tipo.gov.tw
- 專利：https://portal.tipo.gov.tw
- 著作權：https://cras.tipo.gov.tw

**前置作業**：需有自然人憑證或工商憑證（經濟部核發）才能使用線上申請系統。一般公司建議用工商憑證，個人用自然人憑證。

## 常見申請錯誤與代價

| 錯誤 | 後果 |
|------|------|
| 公開發明後超過12個月才申請 | 優惠期過，自己的公開成為先前技術，無法取得專利 |
| 商標只申請一類 | 其他類別無法主張，競爭對手可搶注 |
| 未在申請前先做近似商標搜尋 | 申請被駁回或核准後遭異議，浪費費用與時間 |
| Claims 寫得太窄 | 競爭對手只需稍作修改即可規避，專利形同虛設 |
| 忘記繳年費 | 專利消滅，且通常無法恢復 |
| 員工合約未明確 IP 歸屬 | 爭議時需靠法院認定，著作權法 Art.11 雖有預設規則，但合約更明確 |
