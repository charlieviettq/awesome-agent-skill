# 三道防線 2020 新版與台灣實務

## 版本演進

### 原版 Three Lines of Defense（2013 IIA）
- 核心：業務、風險、稽核三道「防線」
- 比喻：軍事防禦體系
- 問題：過度對抗性、忽略協作、漏掉治理

### 新版 Three Lines Model（2020 IIA 修訂）
- 核心：三道「功能」協作 + 治理機構
- 比喻：系統化協作
- 改變：
  1. 名稱去「Defense」改為「Model」
  2. 明確納入治理機構（governing body）
  3. 強調三道之間的協作與資訊流
  4. 新增外部保證提供者（external assurance）

## 2020 新版結構

```
┌─────────────────────────────────────────────────┐

│          (董事會 / 審計委員會)                    │

└─────────────────────────────────────────────────┘
   ↑ 報告與保證                            ↑ 報告
┌──────────────────────────────────────┐ ┌─────────────┐

│  │ 第一道       │  │ 第二道       │    │ │             │
│  │ 風險擁有／   │  │ 專業風險／   │    │ │ 獨立保證    │

      外部提供者：會計師、主管機關、評等、獨立專家
```

## 三道詳解

### 第一道：業務／營運（First Line）

**角色**
- 擁有（Own）並管理（Manage）風險
- 提供第一手風險資訊
- 執行控制措施

**典型單位**
- 業務部門
- 生產／製造
- 行銷
- IT 運維
- 客服

**常見誤解**
- ❌ 第一道只是「執行」，風險管理是第二道的事
- ✅ 第一道才是風險的真正主人；第二道只是協助方法論

**台灣實務常見問題**
- 業務主管不認為風險管理是自己的事
- 業務 KPI 與風險管理衝突（追業績不管風險）
- 第一道人員缺乏風險工具訓練

**改善建議**
- 每個事業部設 Risk Champion（風險聯絡人）
- 業務 KPI 納入風險指標
- 年度風險訓練必修

### 第二道：風險管理／合規（Second Line）

**角色**
- 提供風險管理方法論
- 監督第一道的執行
- 協調跨部門
- 挑戰（challenge）第一道的判斷

**典型職能**
- 風險管理部（Risk Management）
- 合規部（Compliance）
- 財務控管（Financial Control）
- 資安（IT Security / Cybersecurity）
- 品質管理（Quality Assurance）
- ESG（近年新興）

**權力邊界**
- 有權要求資訊、報告
- 有權挑戰第一道的風險判斷
- 無權直接決策業務（除非法遵事項）
- 獨立於被監督單位（不向 CFO 或業務單位直接報告風險事項）

**常見誤解**
- ❌ 第二道可以取代第一道做風險管理（風險保母）
- ✅ 第二道是「方法論提供者 + 獨立監督者」，不代執行

**台灣實務常見問題**
- 第二道空缺（尤其中型製造業）
- 第二道由 CFO 兼任（獨立性疑慮）
- 風管部人員地位低、發言權弱

**改善建議**
- 設立專職 CRO，虛線向 CEO、實線向董事會風管會
- 風管部預算與人力獨立編列
- 薪酬與業務部門解鉤

### 第三道：內部稽核（Third Line）

**角色**
- 獨立於管理層
- 客觀驗證第一、二道的有效性
- 提供董事會保證
- 改善建議

**必備獨立性**
- 行政上可向 CEO 報告，但稽核事項必須向審計委員會（獨董）報告
- 稽核長（CAE）的任免由審計委員會決定
- 薪酬不受業務績效影響
- 稽核範圍不受管理層限制

**方法論**
- IIA International Professional Practices Framework（IPPF）
- 風險導向稽核（Risk-Based Audit Planning）
- 三年滾動稽核計畫

**常見誤解**
- ❌ 內稽只管財務
- ✅ 內稽涵蓋營運、合規、策略各面向

**台灣實務常見問題**
- 稽核人力不足（中小型常 1–3 人）
- 稽核長實際向 CFO 報告（失去獨立性）
- 稽核報告被管理層「修飾」後送董事會
- 業務單位報復稽核人員（職涯、薪酬影響）

**改善建議**
- 稽核長直接向審計委員會召集人（獨董）報告
- 審計委員會每年獨立與稽核長無管理層在場會談
- 稽核報告雙軌送（管理層 + 獨董）
- 稽核計畫由審計委員會核准，不受管理層干預

### 治理機構（Governing Body）— 新版明確位置

**角色**
- 問責：對三道負最終責任
- 監督：接收三道報告
- 方向：設定風險胃納
- 挑戰：挑戰管理層假設

**組成**
- 董事會（含獨董）
- 審計委員會（Audit Committee，通常 3 獨董）
- 風險管理委員會（Risk Committee，近年新增）
- 薪酬委員會（影響風險文化）
- 提名委員會（影響人才）

**關鍵報告關係**
- 審計委員會 ← 稽核長（第三道）
- 風險管理委員會 ← CRO（第二道）
- 業務委員會（或董事會） ← CEO（第一、二道執行）

### 外部保證提供者（External Assurance Providers）

**類型**
- 簽證會計師（財務報表）
- 主管機關檢查（金管會、勞動部、環保署）
- 產業評等（穆迪、標普）
- ESG 評等（MSCI、FTSE）
- ISO 認證機構
- 外部風險顧問

**新版特色**
- 2020 版明確納入
- 協助治理機構取得獨立觀點
- 補充內部保證不足處

## 三道之間的協作機制

### 資訊流
```
第一道 → 第二道：風險事件報告、KRI 數據、控制措施執行結果
第二道 → 第一道：方法論、訓練、挑戰、協助
第二道 → 第三道：風險地圖、主要關切、政策
第三道 → 第二道：稽核發現、改善建議
三道 → 治理機構：各自獨立報告
治理機構 → 三道：指示、胃納、挑戰
```

### 協作會議
- **風險治理聯席會議**：每季一次，三道代表參與
- **重大事件聯合小組**：事件發生時臨時組成
- **年度策略對話**：ERM 與策略規劃對齊

### 獨立性保障
- 第二道不能指揮第一道（除非法遵強制）
- 第三道不能參與第一、二道的設計（避免自己稽核自己）
- 第三道稽核計畫由審計委員會核准

## 台灣金融業實務（金管會要求）

**銀行業**
- 金融檢查局定期檢查
- CRO 獨立設置
- 三道防線明文於《銀行內部控制及稽核制度實施辦法》
- 薪酬遞延與風險連結

**保險業**
- 風險基礎資本（RBC）
- 精算獨立性
- ORSA（Own Risk and Solvency Assessment）

**證券業**
- 自律規範
- 三道防線架構要求

## 台灣非金融業實務演進

### 大型製造業
- 鴻海、台塑、台積電：完整三道防線
- 中型製造：第一、三道完整，第二道部分空白

### 家族企業
- 第一、三道容易（家族成員本身承擔一部分）
- 第二道最難（需引入外部人才）
- 文化上挑戰：家族成員不習慣被挑戰

### 上市櫃 vs. 未上市
- 上市櫃：法規要求明確
- 未上市：多數仍 ad-hoc
- IPO 前 2–3 年需完整建置

## 三道防線常見失衡模式

### 失衡 1：只有一道與三道
**症狀**：業務部門做事、內稽檢查，中間沒有風險或合規
**風險**：業務決策時無人挑戰、稽核後才發現問題
**處方**：建立第二道，即使只是一人團隊起步

### 失衡 2：第二道取代第一道
**症狀**：風險部幫業務部做風險評估、填報表
**風險**：業務單位推卸責任、第二道超負荷
**處方**：明確權責、訓練第一道自己做

### 失衡 3：第二、第三道合署
**症狀**：風險長兼稽核長、或同一部門涵蓋兩者
**風險**：失去獨立性、「自己稽核自己」
**處方**：必須分離，即使組織小也要不同主管

### 失衡 4：第三道無獨立性
**症狀**：稽核長向 CFO 報告、稽核計畫受 CEO 干預
**風險**：稽核報告被過濾、重大發現被壓下
**處方**：正式向審計委員會報告、獨立薪酬考核

### 失衡 5：治理機構不參與
**症狀**：董事會一年才討論風險一次、獨董不進入風險議題
**風險**：三道防線形同虛設、管理層獨大
**處方**：設立風險管理委員會、獨董積極主導

## 路線圖（三道防線建置）

**階段 1（0–6 個月）：盤點與治理**
- 現況評估
- 設立委員會
- 招募 CRO
- 政策文件框架

**階段 2（6–12 個月）：第二道成形**
- CRO + 團隊到位
- 方法論建置
- 訓練啟動
- 第一道 Risk Champion 指派

**階段 3（12–18 個月）：第三道獨立**
- 稽核長向審計委員會報告
- 風險導向稽核計畫
- IPPF 導入

**階段 4（18–24 個月）：協作機制**
- 聯席會議
- 資訊系統整合
- 外部評估
