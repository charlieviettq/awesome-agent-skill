# 範例：隱形冠軍家族企業的傳承規劃

## 情境設定

**公司**：中部某精密機械家族企業（化名「恆光機械」）
- 主力產品：歐系車廠精密零件（Tier-2 供應商）
- 營收 18 億、員工 140 人、營業毛利 32%、ROE 18%
- 創辦人王董 68 歲，創業 42 年，台大機械系退休教授之子
- 長女（38 歲）美國 MBA + 機械碩士，返台 5 年任副總（主管國際業務與財務）
- 次子（35 歲）國內本科機械系，任生產部主管 12 年（實際操盤工廠）
- 股權結構：王董 65%、夫人 10%、長女 12%、次子 10%、王董胞弟 3%

**議題**
- 王董心臟血管曾做支架，家族開始擔心接班
- 長女主張轉型「機械 + 服務」（售後、預測性維護、數據分析）
- 次子主張「深耕精密製造」（提升良率、擴廠、續做 Tier-2）
- 家族聚餐時多次發生口角，老夫人（母親）介入調解
- 二位元老（副總、廠長）將於 2 年內退休

**課堂要求**（成大 EMBA 中小企業管理期末作業）
> 以中小企業／家族企業管理框架，分析恆光機械的傳承議題，並提出未來 5 年傳承藍圖。字數 5,000–6,000。

---

## 一、基本盤

**規模**：營收 18 億、員工 140 人 — 典型「隱形冠軍」型中型企業
**家族代數**：第一代仍在位、第二代進入企業、第三代尚未出生

**家族結構圖**
```
創辦人王董 (65%) ─── 夫人 (10%)
   │                      │
   ├─── 長女 (12%, 副總)
   └─── 次子 (10%, 生產主管)

胞弟 (3%, 不任職)
```

**事業階段**：成熟期偏後，面臨電動車轉型 + 傳承雙重轉折

**資源盤點**
- 資金：現金流穩定、低負債（負債比 28%）
- 人才：技術深度強、管理層薄（一人多角）
- 品牌：B2B 領域內知名，消費端無
- 通路：依賴 3 家德系車廠（集中度高）

---

## 二、核心議題診斷

**主要張力（依優先序）**

1. **傳承規劃缺失**（Layer 1–4 均未系統啟動）
2. **家族成員戰略分歧**（長女 vs. 次子）
3. **產業轉型壓力**（EV 化 → 傳統車廠供應鏈洗牌）
4. **治理結構薄弱**（無家族憲章、董事會成員皆家族內部）
5. **元老接班空窗**（2 年內副總與廠長退休）

**優先序邏輯**
- 王董健康是觸發點，傳承不可再拖
- 戰略分歧若不解，接班後易分裂
- 產業轉型需要接班人自己做決定，反向要求接班機制儘快明確

---

## 三、框架分析

### 框架一：資源受限下的策略選擇

**資源現況**
| 維度 | 狀態 | 風險 |
|---|---|---|
| 資金 | 穩定 | 充足但單一客戶集中 |
| 人才 | 技術強、管理弱 | 中階幹部斷層 |
| 品牌 | B2B 強 | 轉型服務需新品牌能力 |
| 通路 | 3 家德系車廠 | 高度依賴、談判地位弱化 |

**策略選項配比建議**
- 60% 資源聚焦現有利基（次子路線：精密製造極致化）
- 30% 投入新興領域（長女路線：機械 + 服務，做小做精）
- 10% 通路風險分散（開發一家亞系或美系車廠）

**關鍵是避免**
- 全面轉型（資源不足會兩頭空）
- 完全保守（將被電動化邊緣化）

### 框架二：三圓分析

**七位置盤點**
| 位置 | 成員 | 動機／視角 |
|---|---|---|
| 家族 + 所有權 + 員工 | 王董、長女、次子 | 三圓交集，最深涉入 |
| 家族 + 所有權（無員工） | 王董夫人、胞弟 | 情感與財產但未營運 |
| 家族（無所有權、無員工） | 未婚／未來配偶、未出生下一代 | 未來利害關係人 |
| 員工（無家族、無所有權） | 即將退休副總、廠長、140 位員工 | 專業經理人 + 基層 |

**張力來源**
- 長女（偏所有權 + 國際視角）vs. 次子（偏事業 + 營運深度）：兩人「三圓投射」不同
- 長女可能更重視「所有權價值最大化」（有服務轉型、IPO 潛力）
- 次子可能更重視「事業系統穩定」（生產效率、製造深耕）
- 兩人視角都合理，但未公開對話

### 框架三：四層傳承盤點

| 層 | 現況 | 缺口 |
|---|---|---|
| 所有權傳承 | 未正式啟動，股權仍集中於王董 | 需規劃閉鎖控股 + 信託 + 分年贈與 |
| 治理結構傳承 | 董事會皆家族內部、無獨董、無家族憲章 | 急需治理升級 |
| 經營權傳承 | 長女、次子各負責不同領域，但無明確接班路徑 | 需設計雙軌培育 + 評估機制 |
| 文化／無形傳承 | 王董人脈、德系車廠關係、技術知識未系統化移轉 | 需啟動關係盤點與知識文件化 |

### 框架四：家族憲章檢核

**現況**：尚未制定

**建議條款優先序**
1. 家族願景（為何續經營）— 避免分家變現的訊號
2. 家族就業政策（三代子女進入公司的條件）
3. 股權移轉機制（家族內 vs. 家族外）
4. 接班機制（遴選、過渡、備案）
5. 衝突解決機制（家族委員會 → 獨董 → 外部仲裁）

## 四、介入方案

### 短期（6 個月）

| 項目 | 行動 | 負責人 |
|---|---|---|
| 聘家族顧問 | 引入專業家族企業顧問公司 | 王董 |
| 家族憲章草擬 | 召開 3–4 次家族會議啟動 | 顧問主導 |
| 元老轉移計畫 | 副總、廠長退休前的知識移轉清單 | 次子對接生產、長女對接國際業務 |
| 治理體檢 | 委外做一次家族企業治理診斷 | 顧問 |

### 中期（1–3 年）

| 項目 | 行動 |
|---|---|
| 董事會改組 | 引進 2 位獨董（一位產業、一位財稅），家族席次 5 席（王董、夫人、長女、次子、胞弟） |
| 家族委員會成立 | 制度化家族對話機制，每季一次 |
| 雙軌試煉 | 長女主導「服務轉型專案」獨立事業部，次子主導「製造深耕」本業；各設 KPI 與 3 年檢視點 |
| 所有權規劃第一階段 | 王董分年贈與股權給下一代（8–10% per year），閉鎖控股架構建置 |
| 三年檢視 | 家族委員會 + 獨董聯合評估接班型態（長女 CEO、次子 CEO、雙 CEO、家族控股 + 外部 CEO） |

### 長期（5–10 年）

| 項目 | 行動 |
|---|---|
| 接班型態確立 | 基於雙軌試煉結果選擇 |
| 完整所有權移轉 | 5 年內王董股權降至 30%，治理權保留（家族信託設計） |
| 關鍵客戶關係移轉 | 德系車廠 VIP 會面由接班人主導 |
| 第三代教育 | 家族就業政策執行（外部歷練 3 年後方可入公司） |
| 家族辦公室（Family Office）評估 | 若家族資產結構複雜化則啟動 |

**專業經理人配比**
- 短期：家族成員 85% + 專業經理人 15%
- 中期：家族 60% + 專業 40%（補財務長、CMO、HR）
- 長期：家族 50% + 專業 50%（家族聚焦治理，專業聚焦營運）

## 六、風險與侷限

**家族情感風險**
- 長女、次子若未公開對話，暗流可能在王董離世後爆發
- 老夫人介入可能成為正向緩衝或反向激化因子
- 胞弟 3% 股權看似小，若未來增資或併購時可能成為關鍵少數

**外部顧問風險**
- 顧問若只按歐美模式推動 → 忽略台灣稅法與家族文化
- 過度文件化（動輒 50 頁家族憲章）反而讓家族抗拒執行
- 建議選擇「華人家族企業 + 台灣法稅」雙重專業的顧問

**法稅專業界面**
- 閉鎖型公司 vs. 控股公司設計需會計師 + 律師共同規劃
- 遺贈稅率雖降至 20%，但高額部分仍需規劃
- 家族信託在台灣法律環境下仍有限制，需謹慎評估

**本分析的資訊侷限**
- 未訪談長女、次子本人意願（僅以案文推論）
- 財務細節與客戶結構為假設性描繪
- 三代下一代尚未出生，無法納入規劃
