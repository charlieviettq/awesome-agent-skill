# 二代接班規劃實務手冊

## 定位

本手冊整理台灣家族企業傳承的實操流程，**不構成法律與稅務建議**。實際規劃必須與律師、會計師、家族顧問共同執行。

## 傳承的四層矩陣（執行細節）

### Layer 1：所有權傳承

**工具組合**
| 工具 | 適用情境 | 注意 |
|---|---|---|
| 分年贈與 | 股權價值集中、家族關係穩定 | 年度免稅額 244 萬（2023 年起），可多年分割 |
| 閉鎖型公司 | 想保留家族控制權、限制外人介入 | 2015 年後新工具，需公司組織變更 |
| 信託（家族信託） | 多代傳承、資產隔離、跨境 | 成立時仍課贈與稅；架構複雜，顧問費高 |
| 控股公司 | 結構清晰、可分離所有權與經營權 | 常搭配閉鎖型 + 複數表決權特別股 |
| 一次繼承（遺囑） | 來不及規劃、創辦人意外 | 遺產稅 10–20%、家族易爭產 |
| 股份回購 | 處理不任職家族成員 | 需充足現金；公司治理須經董事會 |

**時程建議**
- **60 歲前**：開始規劃，控股架構 + 信託
- **60–70 歲**：分年贈與 + 閉鎖型轉換
- **70 歲以後**：若尚未規劃，採取應急方案（大額贈與 + 基本信託）
- **80 歲以後**：重點轉為遺囑與意定監護

### Layer 2：治理結構傳承

**治理成熟度模型**

```
Level 1：Family Only（純家族）
  - 家族成員擔任所有董事、所有高管
  - 決策靠家族會議
  - 90% 小型家族企業停在這層

Level 2：Family + Trusted Advisors
  - 引入 1–2 位獨立董事（通常會計師、長年好友）
  - 仍是家族主導
  - 常見於 5–20 億營收階段

Level 3：Professional Board
  - 獨立董事佔董事會 1/3 或以上
  - 設獨立審計、薪酬委員會
  - IPO 前的必經階段

Level 4：Family Governance System
  - 設家族委員會（Family Council）
  - 家族憲章、家族會議、家族辦公室
  - 成熟代際傳承的標誌

Level 5：Institutional Governance
  - 家族控股 + 完全專業化 CEO
  - 家族以股東身分參與（董事會而非管理層）
  - 台灣少數（富邦金控、國泰金控接近此級）
```

**進階路徑**
- 每級需 2–4 年過渡期
- 跳級風險高（Level 1 → Level 4 常失敗）
- 外部投資人介入（PE/VC）可加速升級，但治理文化衝突

### Layer 3：經營權傳承

**接班人培育三階段**

```
階段 1：外部歷練（22–32 歲）
  目標：建立自信、累積人脈、失敗在不影響家業之處
  方式：
    - 海外留學 + 工作經驗
    - 投資銀行、顧問公司、跨國企業
    - 避免「太早回家」變成「公主／王子」

階段 2：內部輪調（32–40 歲）
  目標：了解全局、建立內部威信
  方式：
    - 從基層做起（故意派至艱困部門）
    - 依序歷練業務、生產、財務、策略
    - 設明確績效評估（避免「老闆兒子」特權）

階段 3：獨立戰場（40–50 歲）
  目標：證明能力、完成權力轉移
  方式：
    - 指派獨立事業部（子公司、新事業）
    - 3–5 年獨立績效檢視
    - 成功者正式繼任 CEO／總經理
```

**培育失敗模式**
- 直接從「公主／王子」跳到副總、總經理
- 輪調但無 KPI
- 沒有退出機制（失敗後無法降級）

### Layer 4：文化與無形資產傳承

**清單盤點**

| 類別 | 具體項目 | 移轉方式 |
|---|---|---|
| 關鍵客戶關係 | VIP 客戶、高層友誼 | 一代帶二代出席 2–3 年 |
| 供應商信任 | 主要供應商 CEO 關係 | 定期共同拜訪、年節聯繫 |
| 技術與知識 | 創業累積的 know-how | 系統化文件 + 師徒制 |
| 員工向心力 | 老員工對創辦人的忠誠 | 內部家族形象塑造 |
| 產業網絡 | 公會、協會地位 | 逐步由二代擔任幹部 |
| 品牌故事 | 企業精神、公開敘事 | 年度致辭、對外訪問 |

**時程**：無法壓縮，至少 5 年，通常 10 年以上

## 接班型態選擇矩陣

| 接班型態 | 適用情境 | 優點 | 風險 |
|---|---|---|---|
| **長子（女）直接接班** | 培育充分、能力服眾 | 家族認同高 | 能力不足時失敗率高 |
| **兄弟姊妹共治** | 多子有意願、互補性強 | 降低單一人風險 | 權力結構易崩 |
| **專業 CEO + 家族控股** | 家族無適任者或多代後 | 專業化徹底 | 需強董事會制衡 |
| **家族內外部 CEO 輪替** | 成熟治理、長期規劃 | 兼顧家族認同與專業 | 過渡期長 |
| **雙 CEO 過渡** | 接班人需時間成熟 | 分享風險 | 權責模糊、易衝突 |
| **家族控股基金會** | 代數多、資產龐大 | 永續性高 | 法稅結構複雜 |

## 常見傳承情境處方

### 情境 A：一代強勢、不願放權
**症狀**：二代回國 5 年仍未授權、會議仍由一代拍板
**處方**：
1. 外部顧問介入對一代做退休心理輔導
2. 設定「過渡年限」明確終點（如 70 歲卸任 CEO）
3. 二代先接獨立事業部證明能力
4. 引入獨董制衡，弱化一代單一決策

### 情境 B：二代不願接班
**症狀**：二代在國外發展，明確表示不想回來
**處方**：
1. 尊重意願，不強迫
2. 啟動「家族控股 + 專業 CEO」路徑
3. 二代擔任「家族代表董事」而非 CEO
4. 提前設計股權與分紅結構
5. 考慮第三代培育（若有）

### 情境 C：多子競爭
**症狀**：兄弟姊妹各有長處、皆想接班
**處方**：
1. 家族委員會召開多次對話
2. 「雙軌試煉」各負責一條事業線 3–5 年
3. 獨立評估（獨董 + 外部顧問）
4. 依結果設計分工（如：一位 CEO + 一位 COO + 一位董事）
5. 備案：若共治失敗，分家方案（不同事業分立）

### 情境 D：家族紛爭已訴訟
**症狀**：家族成員間已有法律糾紛、股權爭議
**處方**：
1. 立即凍結公司重大決策
2. 引入中立外部仲裁者（律師 + 法院調解）
3. 考慮股份回購解除衝突方
4. 公司治理緊急應變（董事會決議防弊）
5. 長期需重建家族憲章或分家

## 法稅實務要點（概念）

**台灣主要稅賦**
- **遺產稅**：1.333 億以下 10%、1.333–2.667 億 15%、2.667 億以上 20%（2023 年）
- **贈與稅**：同累進結構，年度免稅額 244 萬
- **土地增值稅**：家族企業擁有大量土地時需特別規劃
- **所得稅**：經營所得、股利所得、海外所得規則

**常用結構工具**
- 閉鎖型股份有限公司（限制股東轉讓）
- 控股公司（分離所有權與經營權）
- 家族信託（資產隔離與跨代傳承）
- 私募基金結構（近年大型家族採用）

**警告**：以上皆為概念描繪，實際稅務規劃必須有執業會計師、律師共同簽署意見，本手冊不取代專業建議。

## 接班規劃常見錯誤

1. **太晚規劃**：70 歲才開始，通常來不及做完整 Layer 1-4
2. **只做 Layer 1 忽略 2-4**：只處理股權，沒建治理與文化
3. **過度依賴文件**：家族憲章寫完就放檔案夾，沒執行機制
4. **忽略配偶**：一代夫人、二代配偶的角色未明確
5. **把「公平」當「平等」**：一律等比分產 ≠ 公平，可能導致經營權碎片化
6. **不面對情緒**：所有規劃都是理性化設計，忽略家族情感
7. **單一顧問依賴**：只請一位律師或一位會計師，視角受限
8. **跨國家族忽略他國法律**：海外資產、他國籍成員的法律複雜度
