## 理論三支柱

### 支柱一：Habitual Domain（習慣領域）
- 個人想法／觀念／判斷隨時間穩定化的集合
- 包含四個子領域（Potential, Actual, Reachable, Activated）
- 強調「擴展性」而非「優劣」

### 支柱二：Competence Set Analysis（能力集合分析）
- 執行決策所需的技能、資訊、資源的集合
- Truly Needed Set（實需）vs. Acquired Set（已有）的差距 = 需補強項
- 常用於創業、新事業進入、人才盤點

### 支柱三：Decision Making in Changing Environments（動態決策）
- 七層決策結構
- 決策參數、偏好、自我狀態會隨情境變動
- 結合博弈論、系統動力學、認知心理

## 與其他相近理論的區別

| 理論 | 提出者 | 核心關注 | 與 HD 關係 |
|---|---|---|---|
| Bounded Rationality | Simon (1955) | 理性有上限 | HD 是 BR 的認知結構化版本 |
| Mental Models | Johnson-Laird (1983) | 思考是建構模型 | HD 含 mental model 但更廣 |
| Dual Process Theory | Kahneman (2011) | 系統 1／系統 2 | HD 描述「為何 System 1 如此」的結構 |
| Cognitive Schemas | Piaget / Bartlett | 認知基模 | HD 的部分元素近 schema |
| Reflexivity | Bourdieu / Giddens | 習性 habitus | 社會學視角，HD 偏個體決策 |

**結論**：HD 的獨特貢獻在「把認知結構拆成可操作的四領域 + 擴展工具」，其他理論或偏哲學、或偏心理實驗、或偏社會結構，均未將認知結構化到可直接用於決策介入的程度。
