# 陽明交大 EMBA 習慣領域課程授課脈絡筆記

> 本檔整理陽交系統（原交大管理科學系 → 陽明交大管理學院）對 HD 理論的教學傳統與授課偏好，協助學員在作業與口試時對齊老師期待。

## 陽交系統 HD 教學脈絡

### 歷史源流
- 交大管理科學系 1980 年代起，因游伯龍多次返台客座而引入 HD 理論
- 1995 後由曾國雄（決策科學）、黃家齊（組織理論）等教授延續
- 2020 年陽明大學與交通大學合併後，管院 EMBA 維持 HD 作為決策類選修
- 清大服科所、中山大學企管系亦有獨立開設（但陽交仍為中心）

### 當前課程定位
- EMBA 決策分析組必修或高度推薦選修
- 通常搭配「多準則決策」、「策略思考」、「創新決策」共同開課
- 結合個案分析與自我 HD 反思作業（期末「個人 HD 擴展計畫」）

## 典型授課者偏好

### 老師最在意的（加分項）
1. **能對應通性編號與名稱**：不是泛指「思維僵化」，而要明確說「通性 7 固化」
2. **四領域內容具體**：每一層列 3–5 項，不是一句帶過
3. **擴展工具有具體操作**：不是「建議擴展 HD」，而是「用工具 X 做 Y 動作」
4. **連結產業脈絡**：用台灣或自己公司的案例印證
5. **承認分析侷限**：明白 HD 不是萬能解，需搭配其他工具

### 老師不喜歡的（扣分項）
1. **堆砌八條通性**：全部列出卻無主次
2. **把 HD 神秘化**：「東方智慧、西方思維」等本質主義論述
3. **用 HD 解釋一切**：連財務估值都要用 HD 硬套
4. **抄教科書定義**：直接貼游伯龍 1990 年原文翻譯
5. **自傳式反思**：寫成心靈雞湯而非分析文本

## 常見作業題型

### 題型 A：個案分析（4,500–6,000 字）
- 提供 5–10 頁個案，要求用 HD 分析決策者盲點
- 需涵蓋：四領域 + 主導通性 2–3 條 + 七層卡點 + 擴展介入
- 評分重點：診斷深度、工具運用、介入可行性

### 題型 B：個人 HD 擴展計畫（3,000–4,000 字）
- 分析自己過去三年一個關鍵決策（創業、接班、重大投資）
- 指出當時 HD 的限制，設計未來 12 個月擴展計畫
- **學術誠信注意**：這類作業要求第一人稱真實反思，AI 協助僅能限於「結構梳理」，不可代寫反思內容

### 題型 C：組織 HD 診斷（小組報告）
- 選一家公司（通常是學員任職公司或熟悉產業）
- 分析該組織的「共通 HD」特徵與盲點
- 結合通性 2（共通法則）與組織文化分析

## 口試常見問題

| 問題 | 背後考點 |
|---|---|
| 「請指出你分析中的主導通性與證據」 | 是否能對應案文 |
| 「若通性 7 固化這麼嚴重，你提的介入真的能撼動嗎？」 | 介入可行性自省 |
| 「HD 和 Kahneman 的 System 1／2 有什麼不同？」 | 理論比較深度 |
| 「如果決策者本人拒絕擴展，你的方案還有效嗎？」 | 擴展工具的前提 |
| 「四領域如何量測？」 | 操作化能力 |
| 「為什麼選這三種擴展工具而不是其他？」 | 工具選擇邏輯 |

## 引用規範（陽交偏好）

- 中文引用：游伯龍（年份）《書名》。出版社。
- 英文期刊：APA 格式
- 陽交老師普遍接受 HD 中文論述，但鼓勵同時引用 1–2 篇英文原典以示嚴謹
- 避免只引 1998 年中文版 — 至少補一本英文 Springer 版

## 與其他課程的銜接建議

- **修過「決策分析」後選本課**：可與 Simon IDC、AHP、多準則決策串接
- **修過「組織理論」後選本課**：用 HD 補強個體層診斷，與組織文化配套
- **準備寫質性論文**：HD 四領域可作為訪談架構（Actual / Reachable / Activated 等）
- **準備創業／創新類論文**：結合 Competence Set Analysis 與創業機會辨識

## 推薦閱讀順序（陽交課前預習）

1. 游伯龍（1998, 1 版）《HD 習慣領域》前四章 — 取得整體概念
2. Yu (1990) *Forming Winning Strategies* Ch. 3–5 — 學術化理解
3. 選讀游伯龍〈Behavioral Bases and Habitual Domains〉1980 — 原始論文
4. 陽交 EMBA 學長姐論文 2–3 篇（管院圖書館可調） — 了解應用範圍
5. 上課前嘗試對自己最近一次重大決策做四領域盤點 — 熱身

## 授課者個別偏好（概略）

> 以下為通則傾向，實際以當屆授課者公告為準。

- **決策科學背景老師**：偏好七層結構與多準則決策的結合
- **組織行為背景老師**：偏好通性分析與組織文化的對應
- **創業創新背景老師**：偏好 Competence Set 與 HD 擴展工具
- **經驗取向老師（業界資深）**：偏好案例豐富、介入具體，理論深度次之
- **學術取向老師**：要求完整文獻回顧、英文原典引用

**建議**：開課第一週務必釐清授課者偏好（觀察 syllabus 引用的核心文獻），作業方向相應調整。
