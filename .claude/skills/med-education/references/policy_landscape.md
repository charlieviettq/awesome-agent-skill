# Taiwan Education Policy Landscape — Essential Context

## 教育制度概覽

### 十二年國民基本教育結構(2014 年至今)

```
國小 1-6 年級 (6 年,義務教育)
  ↓
國中 7-9 年級 (3 年,義務教育)
  ↓
高中 10-12 年級 (3 年) / 高職 (3-4 年)
  ├─ 進路選擇:升學、技職、就業
  ↓
大學 (4 年) / 科技大學 (2-4 年) / 專科 (2 年)
```

**升學通道**:
- **普通教育路線**: 國中會考 → 高中 → 學測 → 大學
- **技職教育路線**: 國中會考 → 高職 → 統測 → 科技大學 / 四年制技術學院
- **實驗教育路線**: 自學、非學校型態實驗教育、學習園區

### 兒童人口與結構變化

| 年份 | 國小在學人數 | 國中在學人數 | 重點 |
|------|------------|------------|------|
| 2010 | ~230 萬 | ~75 萬 | 峰值 |
| 2015 | ~217 萬 | ~68 萬 | 開始下滑 |
| 2020 | ~181 萬 | ~61 萬 | 持續下降 |
| 2025(預估) | ~160 萬 | ~58 萬 | 預計續跌 |
| 2035(預估) | ~130 萬 | ~50 萬 | 20 年減 40% |

**衍生議題**: 少子化 → 教室空置 → 教師員額 → 私校退場壓力 → 偏遠校資源問題

## 近年重大教育改革

### 1. 108 課綱(2019 年實施)

**背景**: 自 87 課綱(民國 2000 年)以來的第二次大改革;107 課綱只微調一年,108 重新設計

**核心變化**:

| 面向 | 舊(99/101 課綱) | 新(108 課綱) |
|------|-----------------|-------------|
| **教學哲學** | 內容導向 | 素養導向 |
| **課程結構** | 各科獨立 | 跨領域整合 |
| **學習重點** | 知識記憶 | 應用、思考、轉移 |
| **課程時數** | 國小: 1,960 節 | 國小: 1,900 節(減 3%) |
| **彈性課程** | 22% | 增至 30-40% |

**教師反應**: 
- 肯定: 減少教學負擔、更貼近學生興趣
- 疑慮: 實施不均勻、弱勢校資源不足、準備時間倉促

**評估進度**: 2019-2023 年為推行期;2024 年後開始累積效果研究(尚無大規模長期評估報告)

### 2. 雙語政策(2018 年宣布,持續推廣)

**政策目標**: 2030 年成為雙語國家;提升學生英文能力與國際競爭力

**實施措施**:
- 國小英文課程: 2001 年起納入(現行 3-6 年級,每週 1-2 堂)
- 國小試點「雙語班」: 特定地區、自願參加、英文授課比例 50% 左右
- 大學雙語教學: 部分課程改為英文授課
- 師資培訓與教材開發: 教育部投入預算

**成效疑慮**:
- 補習業蓬勃(家長擔心英文落後)
- 校內師資不足(英文教師缺乏)
- 城鄉差異(偏遠校難以實施雙語班)
- 本土語言被邊緣化(日語、越南語等族群語言投入相對少)

**相關新聞議題**: 「雙語政策會不會造成教學品質下降?」「英文教學應納入本土語言嗎?」「低社經家庭小孩補習壓力加重?」

### 3. 高等教育招生制度改革

**進行中的改變**:

| 舊系統 | 新系統 | 年度 |
|------|--------|------|
| 指考(筆試 + 分數) | 分科測驗(2021 年起;選擇性,難度較指考低) | 2021 |
| 繁星推薦(單一廉價途徑) | 繁星推薦 + 申請入學(多元評估) | 持續優化 |
| 學測考試科目固定 | 學測科目與次數調整(2024 年起分測次) | 2024 |

**社會反應**:
- 肯定: 減少升學壓力、多元入學管道
- 疑慮: 申請入學資料複雜、家庭資源不均(有補習、顧問資源的學生優勢大)

### 4. 私立學校退場與轉型

**背景**: 少子化導致招生困難;私立大學與高中數量下降

**政策工具**:
- 董事會改革(加強監督)
- 退場基金(協助教職員安置)
- 併校或轉型(e.g. 改為實驗教育、技能專班)

**新聞風險**: 學生權益保護、教職員權益爭議、社區衝擊

## 當前爭議議題(2024-2026 年)

1. **教育預算與資源不均**: 少子化下,教育總預算相對穩定,但偏遠校資源更加吃緊
2. **課綱實施效果評估缺位**: 108 課綱至今無大規模全國評估報告
3. **雙語教育的公平性**: 偏遠校、低社經學生無法參與,加劇差異
4. **高中招生制度複雜化**: 繁星、申請、分發並行,考生與家長困惑
5. **教師流失與士氣**: 教師待遇未追上生活成本;部分縣市師資荒
6. **私校學費與退場**: 家庭負擔加重;退場風險不穩定
7. **性別議題進入課綱**: 同志教育、性平教育遭部分家長反對
8. **AI 與教育**: 如何融入課程?會不會加劇不平等?

## 數據速查(常見報導背景)

### 升學統計(2023 年)

- 國中畢業生進入高中: 約 65%
- 進入高職: 約 35%
- 高中升大學: 約 71%
- 高職升科技大學: 約 65%

### 教育支出(2023 年)

- 國教經費: 約佔政府總預算 9-10%
- 高等教育經費: 約佔 2-3%
- 私立學校學生比: 小學 27%、國中 29%、高中 43%、大學 62%

### 國際比較(PISA 2022)

- 台灣閱讀素養: OECD 平均 500,台灣約 497(略低)
- 數學: OECD 平均 492,台灣約 528(高於平均)
- 科學: OECD 平均 500,台灣約 516(高於平均)

## 新聞寫作提示

- **引用統計時**: 標明年份與來源(教育部 / 主計總處 / 國際機構)
- **報導改革時**: 區分「政策目標」與「實際執行」與「評估結果」
- **引述利益相關者**: 清楚標明立場(工會 / 家長團體 / 官方)
- **少子化議題**: 不可忽視對教育資源、教師就業、校園運營的長期衝擊
- **國際比較**: PISA / TIMSS 年度波動常在誤差內;強調趨勢而非排名
