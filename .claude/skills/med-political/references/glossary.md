# 政治新聞 — 術語對照與常見誤用

> 用詞精確度直接關係到報導可信度。以下是最常被混用、誤用、或翻譯不一致的詞彙。

## 立法相關

| 術語 | 定義 | 常見誤用 |
|------|------|---------|
| 一讀 | 程序委員會排入議程,送交相關委員會 | 媒體常稱「初審通過」(✗),應為「一讀通過」 |
| 二讀 | 院會逐條討論、表決條文 | 「實質討論」階段,通常此階段才有實質修正 |
| 三讀 | 院會表決法案全案,完成立法程序 | 「立法院通過」實際上指三讀通過 |
| 黨團協商 | 各黨團就法案進行非公開討論 | 「藍綠協商」常為媒體簡化說法,實際包括各黨團 |
| 復議 | 對已決議案件提請重新討論 | 「翻案」是口語化用法,正式應為「復議」 |
| 散會動議 | 提議結束會議 | 常用於議事杯葛 |
| 不予審議 | 委員會議決不續審 | ≠「退回提案」 |
| 朝野協商 | 通常指多數黨與少數黨之間 | 在多黨情境下更精確說法為「黨團協商」 |
| 提案連署 | 法案提出需立委連署 | 各院規則對連署人數有明定 |
| 附帶決議 | 法案三讀附帶通過的政策要求 | 附帶決議無強制力,但具政治壓力 |

## 預算相關

| 術語 | 定義 | 常見誤用 |
|------|------|---------|
| 經常門 | 維持機關正常運作所需 (人事、業務) | ≠ 資本門 |
| 資本門 | 用於建設、設備、土地等 | ≠ 經常門 |
| 法定預算 | 立法院三讀通過之預算 | ≠ 中央政府總預算案(行政院提) |
| 追加預算 | 預算年度進行中追加 | 較罕見,需經立院審議 |
| 凍結 | 預算經立院凍結,需報告解凍 | 不等於「砍掉」 |
| 刪除 | 立院實質刪減預算數 | 對機關預算有實質影響 |
| 主決議 | 三讀主案附加之決議 | 與附帶決議不同,主決議具拘束力 |

## 選舉相關

| 術語 | 定義 | 常見誤用 |
|------|------|---------|
| 區域立委 | 單一選區、一席當選 | 與不分區立委制度不同 |
| 不分區立委 | 政黨票按比例分配,該黨提名名單 | 又稱「政黨比例代表」 |
| 政黨補助款 | 依政黨得票率核發,每票補助一定金額 | 屬公費,不應稱「政黨拿到的錢」 |
| 連署 | 公民提案需達連署門檻 | 適用於罷免、創制、複決 |
| 罷免 | 對公職人員任期內提出罷免 | 通過門檻有具體規定 |
| 複決 | 對立法院通過之法案進行複決 | 罕用 |
| 創制 | 由公民提出立法案 | 罕用,憲法保留條款 |
| 政見發表會 | 法定候選人公開政見 | 與一般競選活動不同 |

## 民調與民意

| 術語 | 定義 | 常見誤用 |
|------|------|---------|
| 樣本數 (n) | 受訪人數 | 樣本數越大,誤差越小 |
| 信賴水準 | 通常為 95% | 一般默認值,應標明 |
| 抽樣誤差 (MOE) | 在 95% 信賴水準下的最大誤差 | 「誤差範圍」是常用同義詞 |
| 加權 | 依母體結構調整樣本權重 | 加權方法 (年齡/地區/性別) 影響結果 |
| 訪問方法 | CATI (電話)、CAWI (網路)、面訪 | 各方法的代表性不同 |
| 委託方 | 出資進行調查的單位 | 必揭露,影響中立性判讀 |
| 執行方 | 實際執行調查的單位 | 與委託方可能不同,應分別揭露 |

## 政府組織

| 術語 | 定義 |
|------|------|
| 一條鞭體系 | 中央 → 地方分支機關直接指揮 |
| 法人組織 | 行政法人(如國表藝中心)、公益法人、私法人 |
| 黨政分際 | 政黨人士不得介入行政體系運作 |
| 行政中立 | 公務員不得參與政黨活動 |
| 政務官 vs 事務官 | 政務官隨政黨進退、事務官常任 |
| 內閣改組 | 通常指部長級異動 |

## 易混淆的兩岸 / 國際用語

| 不應混用 | 原因 |
|---------|------|
| 「中國」vs「中華人民共和國」 | 政治脈絡敏感,根據用途選擇 |
| 「我國」vs「台灣」vs「中華民國」 | 同樣敏感,媒體應有自家風格指南 |
| 「對岸」「中國大陸」「中共」 | 各有意涵,讀者解讀不同 |
| 「九二共識」 | 各方對其內涵定義不同,引用時應交代脈絡 |
| 「兩岸關係」vs「兩國關係」 | 法律與政治意涵差異極大 |

## 容易引發框架爭議的字詞

| 偏綠常用 | 偏藍常用 | 中立替代 |
|---------|---------|---------|
| 「親中」 | 「友中」「兩岸交流派」 | 「主張兩岸對話 / 經貿合作」 |
| 「黑箱」 | 「依程序」「依規定」 | 描述具體程序步驟,讓讀者判斷 |
| 「綠色執政」 | 「綠色濫權」 | 「現任政府的 XX 政策」 |
| 「轉型正義」 | 「政治追殺」 | 「不當黨產處理 / 促轉條例執行」 |
| 「大內宣」 | 「正面宣傳」 | 「政府宣傳支出 / 政府廣告投放」 |

**處理原則**:讓事實自己說話,避免採用任一陣營的標籤化用語。
