# 即時新聞／一般新聞稿範本

## 適用情境

- 突發事件（車禍、火災、災害、犯罪、示威）
- 官方發布（政府公告、記者會、財報、判決、人事案）
- 會議／活動報導（新品發表、運動賽事結果、選舉結果）
- 任何需要「快」「準」「齊」的資訊傳遞

## 核心原則：倒金字塔（Inverted Pyramid）

最重要資訊放最前面；讀者讀完第一段就能掌握 80% 核心；即使被截斷，剩下內容仍可獨立閱讀。

```
╔═══════════════╗
║   最重要      ║  ← 導言：What + Who + When + Where
╠═══════════════╣
║   次重要      ║  ← How + Why
╠═══════════════╣
║   細節        ║  ← 細節、背景、多方反應
╠═══════════════╣
║   補充        ║  ← 後續、影響、歷史脈絡
╚═══════════════╝
```

## 結構模板

### 標題（15-25 字）

- 必含：主體 + 動作 + 對象／結果
- 避免：懸疑式（「竟發生這種事」）、情緒式（「震驚！」）、主觀形容（「可怕的」）

**範例**：
- ✅ 台積電公布 Q1 財報 營收年增 41%
- ✅ 北市信義區大樓火警 造成 3 傷 1 失蹤
- ❌ 你絕對不敢相信！這家公司竟然...
- ❌ 恐怖車禍！北市大塞車

### 副標（選用，20-30 字）

補充導言未涵蓋的次重要資訊，如原因、影響、時間背景。

### 導言（30-50 字，最重要段落）

必含 5W（What、Who、When、Where、Why/How 擇要）。

**公式一（事件型）**：
> [時間]，[地點]，[誰] [做了什麼]，[造成什麼結果]。

**範例**：
> 4 月 22 日下午 3 時許，北市信義區松仁路一棟 20 層商辦大樓 8 樓發生火警，火勢延燒約 1 小時，造成 3 人輕傷送醫、1 人失聯。

**公式二（發布型）**：
> [主體] [今日／某時] [動作]，[核心數字或結果]，[意義／影響]。

**範例**：
> 台積電 22 日公布今年第一季財報，合併營收新台幣 8,392 億元，年增 41.6%，創同期新高，主要受 AI 晶片需求帶動。

### 第二段 — 核心展開

回答 How / Why，或補齊導言未及的細節。一段一重點。

### 第三段起 — 細節、背景、反應

- **現場細節**：具體觀察、數字、引言
- **多方反應**：事件涉及多方時，各方說法按順序列出
- **背景脈絡**：本事件與過去事件的關聯、制度背景
- **專家意見**：必要時引用專家觀點（需標明職稱、機構）

### 結尾

- 後續進展（警方調查中、預計何時開庭、下波影響）
- **不要**用「總之」「綜上所述」或個人感想收尾

## 引言使用規範

### 直接引語（有雙引號）

```
「今天的火勢主要在 8 樓，目前已初步控制，」信義分隊隊長李明說，
「起火原因還需進一步鑑識。」
```

- 人物首次出現時必須完整介紹：職稱 + 姓名
- 之後可只用姓氏或代稱

### 間接引語（無引號）

```
李明表示，目前火勢已控制，起火原因仍在調查中。
```

### 動詞選擇

| 中性 | 強度較高 | 慎用 |
|------|---------|-----|
| 說、表示、指出、說明 | 強調、批評、質疑、反駁、駁斥 | 承認、坦承、辯稱、宣稱 |

「承認」「坦承」暗示對方有錯；「辯稱」「宣稱」暗示不可信。未經當事人確認認錯前，**不要**用這些動詞描述其發言。

## 數字處理

- 大數字用千位逗號：`1,234,567`
- 百分比並列絕對值：`裁員 30%，約 1,200 人`
- 金額統一單位：`新台幣 8,392 億元` 或 `美元 263 億元`
- 避免誤導性比較：「年增」「季增」明確標示；基期要合理

## 完整範例

```
# 台積電 Q1 營收創同期新高 年增 41.6%

**副標**：AI 晶片需求強勁帶動 毛利率 58.8% 略高於財測上緣

台積電 22 日公布 2026 年第一季財務報告，合併營收新台幣 8,392 億元，
較去年同期成長 41.6%，創歷年同期新高，毛利率 58.8%，略高於財測區間
57%-59% 上緣。

台積電財務長黃仁昭在法人說明會上表示，本季營收強勁主要來自 3 奈米與 5
奈米製程的 AI 相關晶片需求；其中 HPC（高效能運算）平台占總營收 59%，
較前季提升 3 個百分點。

「AI 需求目前看起來至少會延續到 2027 年，」黃仁昭說，「我們正在加速
2 奈米產能擴充。」

展望第二季，台積電預估美元計價營收介於 285 億至 295 億美元，季增約
8-12%，毛利率可望維持在 57%-59% 區間。

台積電股價 22 日收盤 1,025 元，小漲 0.5%。

---

**稿件類型**：即時新聞
**字數**：約 280 字
**消息來源層級**：一手（法說會發言、財報）
**待查證事項**：無
**倫理／識讀檢核**：✅ 無誹謗、無匿名來源、數字來源清楚
```

## 常見錯誤

1. **導言塞太多**：超過 60 字、寫出 2 個以上新聞點 → 拆成多段
2. **時序混亂**：突發事件請用「事件發生順序」而非「記者獲知順序」
3. **漏反方**：有爭議時僅報一方 → 補上另方回應或標明「未獲回應」
4. **形容詞氾濫**：「慘重的災情」「驚人的數字」→ 用具體數字取代
5. **結尾畫蛇添足**：「相信未來一定會更好」類型 → 刪除
