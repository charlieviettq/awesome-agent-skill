# 開放式創新、CVC 與併購後整合筆記

## 開放式創新（Open Innovation）

### Chesbrough 原始定義（2003）

「將有用的外部知識融入、並將未用的內部知識授出，以創造更多價值」。

- 起源著作：*Open Innovation: The New Imperative*
- 2006：*Open Business Models*
- 2011：*Open Services Innovation*

### 從封閉到開放的轉變

**封閉式創新的假設**：
- 最優秀人才在我這
- 內部研發決定競爭優勢
- 我們發明、我們商業化
- 贏者需先發明
- IP 保護是競爭關鍵

**開放式創新的假設**：
- 不可能全部最優秀人才都在我這
- 外部研發可帶來顯著價值
- 不用自己發明也能商業化
- 建立更好商業模式贏於先發明
- 與他人合作、授權可獲得價值

### 開放式創新的類型

**Inbound（引入型）**
- 授權外部技術
- 專利收購
- 大學合作
- 供應商創新
- 眾包（Crowdsourcing）
- 收購新創

**Outbound（輸出型）**
- 授權內部技術
- 出售閒置專利
- 衍生新創（Spin-out）
- 技術移轉

**Coupled（合資 / 聯盟型）**
- 研發聯盟
- 合資公司
- 產業聯盟
- 標準化組織

### 開放式創新的模式選擇

| 創新階段 | 適合模式 |
|---|---|
| 探索（早期） | 大學合作、眾包 |
| 驗證（中期） | 策略合作、JV、CVC |
| 商業化（後期） | 授權、併購 |
| 優化（成熟） | 供應商創新、顧客創新 |

### 開放式創新的陷阱

**1. NIH 症候群（Not Invented Here）**
- 內部拒絕外部技術
- 修正：高層壓制、KPI 掛勾

**2. NSH 症候群（Not Sold Here）**
- 內部拒絕授權自己技術出去
- 修正：Outbound 收入 KPI

**3. IP 糾紛**
- 共創成果歸屬爭議
- 修正：事前協議、IP 律師

**4. 文化衝突**
- 大公司慢、新創快
- 修正：獨立窗口、快速通道

**5. 策略目標模糊**
- 做了但不知道要什麼
- 修正：明確目標（技術、市場、人才、生態）

### 實務操作

**建立開放式創新辦公室**：
- 專職團隊 3–10 人
- 負責：外部掃描、合作洽談、IP 管理、整合協調

**建立「創新門戶」**：
- 對外：新創可提交合作提案
- 每季 Review
- 明確回應時程

**設「內部大使」**：
- 各事業部指派創新大使
- 協調內外對接

### 典型案例（公開資訊）

**P&G「Connect + Develop」**
- 目標：50% 創新來自外部
- 建立網路、全球掃描

**IBM Open Innovation**
- Linux、Open Source 貢獻
- 轉型服務與軟體

**Samsung NEXT**
- CVC + 加速器 + 策略夥伴

**統一企業、遠傳電信**
- 台灣企業開放式創新案例

## 企業創投（Corporate Venture Capital, CVC）

### CVC 與傳統 VC 的差異

| 面向 | 傳統 VC | CVC |
|---|---|---|
| 主要目標 | 財務報酬 | 策略價值 + 財務 |
| 基金期 | 10 年 | 可彈性（常為 evergreen） |
| LP | 機構投資人 | 母集團 |
| 投後管理 | 治理為主 | 策略整合 |
| 退場 | IPO / 併購 | 可含母集團整合 |

### CVC 類型（Birkinshaw et al., 2002）

**Strategic Driver 型（策略 - 高度整合）**
- 主要目標：接觸新技術 / 市場
- 例：Intel Capital

**Emergent 型（策略 - 低度整合）**
- 探索未知領域、保留可能性
- 例：Salesforce Ventures

**Enabling 型（財務 - 高度整合）**
- 幫助本業服務 / 客戶
- 例：某金融業 CVC

**Passive 型（財務 - 低度整合）**
- 純財務投資
- 例：少數消費品業 CVC

### CVC 的治理設計

**投資委員會**
- 母集團代表 + 獨立委員
- 快速決策（4–8 週）

**投資策略**
- 主題（Thesis）清晰
- 投資階段（天使 / 種子 / A 輪 / B 輪 +）
- 地理範圍
- 資金規模（單案、基金）

**與本業整合機制**
- 策略訊號回饋
- 本業的「業務開發」支持新創
- 共同專案

**退出策略**
- IPO
- 二級市場賣出
- 母集團收購
- 其他收購方

### CVC 的常見問題

**1. 策略與財務搖擺**
- 不同時期強調不同
- 造成新創合作困惑

**2. 母集團文化壓迫**
- 要求被投新創走流程
- 扼殺敏捷性

**3. CVC 團隊不被信任**
- 薪資 / 獎金低於傳統 VC
- 人才流失

**4. 集團經營更迭影響**
- 高層換人後策略轉向
- 基金中斷

**5. 接續輪不跟投**
- A 輪投、B 輪不跟 → 稀釋
- 信號誤傳（其他 VC 看壞）

### CVC 的成功要素

- 獨立、穩定的 LP（母集團承諾 5–10 年）
- 專業團隊（薪酬激勵）
- 清晰策略主題
- 與本業整合但不干擾
- 退出紀律

## 併購（M&A）與創新

### 創新併購的三大目的

**1. Acqui-hire（收購人才）**
- 目標：取得團隊與 know-how
- 特徵：小額、快速、保留團隊

**2. 技術收購**
- 目標：取得特定技術 / 產品
- 特徵：中等金額、整合技術

**3. 業務收購**
- 目標：取得市場、客戶、營收
- 特徵：大額、整合業務

### 併購後整合（PMI）

**整合程度**
- **完全整合**：原事業併入本業
- **部分整合**：保留部分獨立
- **保留**：完全獨立運作（常見於「人才 / 文化保留型」）

**常見失敗原因**
- 急於整合 → 扼殺新創文化
- 核心人才離職（Acqui-hire 失敗）
- 產品線重疊衝突
- 溝通不足
- 客戶流失

**成功要素**
- 明確整合計畫（Day 1、100 Day）
- 被併購方領導者留任機制
- 文化保留設計
- 員工留任方案（股權、分紅）

### 新創併購的特殊挑戰

- **文化保留**：2–3 年獨立過渡期
- **產品路線圖**：承諾不大幅改變
- **人才 earn-out**：2–4 年 vesting
- **客戶公告時機**：溝通策略
- **品牌是否保留**：視情況

### 案例原則

**「買了不動」（保留）**：
- 被併新創仍獨立運作
- 大公司提供「大平台」不干預
- 適合：業務模式本業無法複製

**「逐步整合」**：
- 前 2 年獨立、後 2 年逐步融合
- 適合：技術與本業協同性高

**「快速整合」**：
- 6–12 個月完成
- 適合：資產 / 客戶型收購、少量人才保留

## 併購 vs. CVC vs. 合作的決策

### 決策矩陣

| 情境 | 建議 |
|---|---|
| 技術已驗證、可立即用 | 授權 / 採購 |
| 新領域、需長期建能力 | CVC 投資、學習 |
| 新事業、想完全取得 | 併購 |
| 互補但不易取代 | 策略夥伴、JV |
| 本業有強協同性 | 併購 + 整合 |
| 新創文化敏感 | 併購 + 保留獨立 |

### 決策考量

- 資金成本
- 時間（建自己 vs. 買）
- 策略優先順序
- 組織吸收能力
- IP 與控制需求
- 文化相容性

## 平台化創新

### 生態系（Ecosystem）思維

- 從「公司競爭」到「生態系競爭」
- 例：Apple 生態、Android 生態、Salesforce 生態

### 開放 API 策略

- 開放給開發者與合作夥伴
- 網路效應
- 資料 / 流量變現

### 平台創新治理

- 誰是核心？誰是邊緣？
- 開放程度
- 價值分配機制
- 治理規則（Gatekeeping）
