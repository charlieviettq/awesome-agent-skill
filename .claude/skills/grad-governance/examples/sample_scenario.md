# Example: 台灣離岸風電開發治理分析

## Scenario

一位公共行政碩士生正在撰寫論文，研究台灣離岸風電第三期區塊開發（2022–2030）的治理結構。她的問題是：

> 台灣離岸風電從第一期躉購到第三期競標，治理模式發生了什麼轉變？現行的多方利害關係人安排是否有效？問題在哪裡？

她手上有：經濟部能源局公告、開發商財報、漁民團體陳情書、環評委員會議紀錄。

---

## Analysis

### Step 1: Map the Governance Arrangement

| Actor | Sector | Role | Resources | Authority |
|-------|--------|------|-----------|-----------|
| 經濟部能源局 | 國家 | 主管機關：訂定遴選規則、核發許可 | 法律權威、補貼預算 | 最高 |
| 台電 | 國家/準市場 | 強制購電、電網接續 | 電網基礎設施、購電合約 | 高 |
| Ørsted、沃旭 | 私部門（外資） | 第一、二期主要開發商 | 技術、融資 | 中（受許可制約） |
| 中鋼、台船等本土廠商 | 私部門（國內） | 第三期國產化供應鏈 | 製造能量（仍在建立中） | 中 |
| 漁業署 / 縣市政府 | 國家（次級） | 海域使用諮商、地方許可 | 海域管轄、地方民意 | 中 |
| 沿海漁民協會 | 公民社會 | 陳情、協商補償 | 道德正當性、媒體能見度 | 低–中 |
| 環保署環評委員會 | 國家（半獨立） | 環評審查 | 審查否決權 | 中（功能性高） |
| 金融機構（開發銀行、輸出入銀行） | 私/準國家 | 專案融資 | 資金 | 低（間接） |

**資源依賴關係**：開發商依賴能源局許可 + 台電購電合約；能源局依賴開發商技術與投資；漁民依賴諮商補償；本土廠商依賴國產化要求。

---

### Step 2: Classify the Governance Mode

**主導模式：階層（Hierarchy）＋市場混合，網絡成分薄弱**

| 維度 | 判斷 |
|------|------|
| 遴選機制 | 競標（市場信號），但價格上限由政府設定 → 管制市場 |
| 執行機制 | 能源局逐案核可、環評一票否決 → 強階層 |
| 協調機制 | 無正式多方協商平台；漁民透過陳情而非談判桌參與 → 網絡治理缺位 |
| 國產化要求 | 政府以行政命令強制本土供應鏈，繞過市場邏輯 → 產業政策疊加 |

**形式規則**：《電業法》、《再生能源發展條例》、離岸風電區塊開發遴選規範  
**非正式規範**：「先談補償後施工」的默契；外資開發商理解政治敏感性而自我審查選址

---

### Step 3: Assess Performance

| 準則 | 評分 | 證據 |
|------|------|------|
| 有效性（Effectiveness） | ★★★☆☆ | 2025年裝置量達3.2 GW，低於原訂5.6 GW目標；國產化延誤是主因 |
| 效率（Efficiency） | ★★☆☆☆ | 第三期得標電價（NT$2.49–2.75/度）高於韓國同期競標價，國產化溢價明顯 |
| 正當性（Legitimacy） | ★★★☆☆ | 競標程序透明，但漁民諮商流於形式；多起行政訴訟懸而未決 |
| 課責性（Accountability） | ★★☆☆☆ | 能源局對開發商有合約制裁，但無機制使開發商對漁民負責；環評委員個案負責但無整體成效追蹤 |
| 公平性（Equity） | ★★☆☆☆ | 補償金分配集中於有組織漁民協會；散戶漁民獲益有限 |
| 透明度（Transparency） | ★★★☆☆ | 遴選結果公開，但供應鏈國產化進度查核報告未對外公布 |

---

### Step 4: Diagnose Failures

**階層失靈（Hierarchy Failure）**
- **官僚剛性**：環評程序平均耗時14個月，未依開發期程彈性調整，造成融資窗口關閉
- **協調碎片化**：能源局、漁業署、縣市政府各自審查，無單一窗口；同一開發商須向5個機關遞交重疊文件

**市場失靈（Market Failure）**
- **資訊不對稱**：本土供應鏈廠商缺乏水下基礎工程技術資訊，無法有效競標；外資掌握技術但缺乏轉移誘因
- **外部性內化不足**：風場對底棲漁業的長期生態衝擊未納入開發成本

**網絡失靈（Network Failure）**
- **排除效應**：無正式多方協商機制，漁民、環團以抗爭替代協作，增加交易成本
- **搭便車**：部分本土廠商宣稱符合國產化要求，實際以進口零件充數，損害先行廠商投資誘因

## Result

```markdown
# Governance Analysis: 台灣離岸風電第三期區塊開發（2022–2030）

## Actors and Roles
[見上表]

## Governance Mode
- 主導模式：管制市場（regulated market）中的階層監督
- 混合組成：階層 60% / 市場 35% / 網絡 5%
- 正式規則：《再生能源發展條例》、遴選規範、環評制度
- 非正式規範：補償先行默契、外資政治敏感度自我管理

## Performance Assessment
[見上表，整體評分：有效性與公平性為最弱項]

## Governance Failures
1. 官僚碎片化 → 審查耗時，融資風險上升
2. 技術資訊不對稱 → 國產化目標空轉
3. 漁民諮商網絡缺位 → 衝突成本外部化給開發商

## Recommendations

**短期（1–2年）**
- 設立「離岸風電跨機關協調辦公室」，統一受理許可申請（解決碎片化）
- 強制公開國產化進度查核報告，季度更新（解決透明度不足）

**中期（3–5年）**
- 建立法制化的漁業諮商平台（參考荷蘭 Hollandse Kust Noord 模式），
  給予漁民代表正式否決/修改提案權，而非事後陳情
- 導入「生態系統服務補償」機制，將底棲衝擊貨幣化納入開發成本

**長期（5年以上）**
- 開放本土廠商參股風場股權（而非純供應商），使供應鏈廠商成為
  共同治理者，解決國產化搭便車問題
```

**核心診斷**：台灣離岸風電治理目前是「強階層、弱網絡」的失衡配置。階層有效動員投資，但缺乏網絡機制整合漁民與本土產業，導致衝突成本與效率損耗。改革方向不是去除階層，而是在維持政府最終權威（hierarchy's shadow）的前提下，補上協商網絡的缺口。
