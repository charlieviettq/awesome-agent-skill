# DataAgent Kit — Offer Sheet

**Product:** DataAgent Kit (SpecFlow AI P2)  
**Built on:** [awesome-agent-skill](https://github.com/charlieviettq/awesome-agent-skill)

## One-liner

Curated Cursor/Claude skill bundles so data teams stop reinventing analysis workflows every sprint.

## Tiers

### Free (public GitHub)

- Bundle: `dataagent-free`
- 5 core skills: explore, query, analyze, spec discipline, verify
- Install: `skillhub install-bundle dataagent-free . --format cursor`
- Purpose: lead gen, community credibility

### Pro (commercial)

- Bundle: `dataagent-pro`
- Full data workflow: EDA, SQL, stats, viz, dashboards, validation, specs
- **Pricing:** $29–99/month/team OR $199 one-time (includes 12mo updates)
- Delivery: install command + changelog + email support (48h SLA)

### Enterprise (productized consulting)

- Custom skill pack from your warehouse patterns, metrics defs, notebook conventions
- **Pricing:** 20–50M VND (discovery + build + workshop)
- Includes: 1-day install workshop, 3 custom skills, validation against your repo
